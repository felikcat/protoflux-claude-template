# Basic Controls

## Desktop Controls

| Key | Action |
|-----|--------|
| **Escape** | Open/Close Dash |
| **Left Mouse Button** or **Y** | Primary action - Interact with UI / use equipped tool |
| **Right Mouse Button** | Grab - Move objects |
| **T** or **Middle Mouse Button** | Context Menu |
| **R** or **Mouse Button 4** | Secondary action |
| **WASD** | Move (forward/left/backward/right) |
| **Space** | Jump (walk/run), Move up (fly/noclip) |

## Index Controllers

| Button | Action |
|--------|--------|
| **A button** (Left) | Open/Close Dash |
| **Trigger** | Primary (also toggles laser) |
| **Grip** | Grab |
| **B button** | Context menu |
| **Touchpad** | Secondary |
| **Thumbstick** | Move/rotate (Press to jump) |

## Quest / Rift S / CV1 / Cosmos / WMR Type 2

| Button | Action |
|--------|--------|
| **X button** (Left) | Open/Close Dash |
| **Trigger** | Primary (also toggles laser) |
| **Grip** | Grab |
| **Y button** | Context menu |
| **Thumbstick** | Move/rotate (Press to jump/secondary) |

## Vive Wand Controllers

| Button | Action |
|--------|--------|
| **Menu button** | Context menu (Hold on left to open/close dash) |
| **Trigger** | Primary (also toggles laser) |
| **Grip** | Grab |
| **Touchpad** | Secondary (Press direction to move/rotate, center to jump) |

## WMR Type 1 Controllers

| Button | Action |
|--------|--------|
| **Menu button** | Context menu (Hold on left to open/close dash) |
| **Trigger** | Primary (also toggles laser) |
| **Grip** | Grab |
| **Touchpad** | Secondary/jump |
| **Thumbstick** | Move/rotate |
