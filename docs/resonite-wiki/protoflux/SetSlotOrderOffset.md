# ProtoFlux:SetSlotOrderOffset

> Source: https://wiki.resonite.com/ProtoFlux:SetSlotOrderOffset

Set Slot Order Offset

\*

Next

Instance

OrderOffset

Info

Set's the Slot Order Offset field of the provided Instance [slot](https://wiki.resonite.com/Slot "Slot") upon calling \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")).

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Call this to set the Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s order offset to OrderOffset ( [long](https://wiki.resonite.com/Type:Long "Type:Long")).

### Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))

[Slot](https://wiki.resonite.com/Slot "Slot") to set the property for.

### OrderOffset ( [long](https://wiki.resonite.com/Type:Long "Type:Long"))

The value to set Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s Order Offset to when \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) is called.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Sends an impulse when Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s Order Offset is set to OrderOffset ( [long](https://wiki.resonite.com/Type:Long "Type:Long")) when \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) is called.
