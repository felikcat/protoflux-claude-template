# ProtoFlux:CompareStrings

> Source: https://wiki.resonite.com/ProtoFlux:CompareStrings

Compare

A

\*

B

ComparisonType

Strings

Comapre Strings is a ProtoFlux node that allows you to tell the difference between two different strings as a whole number.
The comparison used is based on [Comparing Strings Microsoft 4.6.2](https://learn.microsoft.com/en-us/dotnet/api/system.string.compare?view=netframework-4.6.2)

## Inputs

### A ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to compare from.

### B ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to compare to.

### ComparisonType ( [StringComparison](https://wiki.resonite.com/Type:StringComparison "Type:StringComparison"))

The string comparison to use.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The difference between the strings. can be positive or negative.
