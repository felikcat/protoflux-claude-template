# ProtoFlux:DateTimeMonth

Month

DateTime

\*

DateTime

The `DateTime Month` node takes in a [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime") and returns the month of that DateTime.

## Inputs

### DateTime ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The DateTime we are getting info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The month portion of the DateTime.
