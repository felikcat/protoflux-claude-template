# ProtoFlux:ColorLuminance

> Source: https://wiki.resonite.com/ProtoFlux:ColorLuminance

Luminance

Color

\*

Colors

Color Luminance calculates the [relative luminance](https://en.wikipedia.org/wiki/Relative_luminance) of an input color. For correct result, the input must be in the [Linear color profile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile"), which may be a bug[\[1\]](https://wiki.resonite.com/ProtoFlux:ColorLuminance#cite_note-1).

## Inputs

### Color ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to calculate the relative luminance of.

## Outputs

### \\* ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The relative luminance of the input color.
