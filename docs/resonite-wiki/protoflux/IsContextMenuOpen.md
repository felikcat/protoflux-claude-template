# ProtoFlux:IsContextMenuOpen

> Source: https://wiki.resonite.com/ProtoFlux:IsContextMenuOpen

Is Context Menu Open

User

\*

Interaction

**Is Context Menu Open** returns a [bool](https://wiki.resonite.com/Type:Bool "Type:Bool") of if the inputted user's [context menu](https://wiki.resonite.com/Context_menu "Context menu") is currently opened.

## Inputs

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

User to track Context Menu open state of.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

If the inputted user's context menu is currently open.
