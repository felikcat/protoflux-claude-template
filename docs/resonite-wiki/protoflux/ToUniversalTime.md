# ProtoFlux:ToUniversalTime

> Source: https://wiki.resonite.com/ProtoFlux:ToUniversalTime

To Universal Time

DateTime

\*

Time

The `To Universal Time` node takes in a DateTime and converts it to universal time using your local time offset. This is in accordance to [Universal Time](https://en.wikipedia.org/wiki/Coordinated_Universal_Time).

## Inputs

### DateTime ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The DateTime we want to convert into universal time.

## Outputs

### \\* ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

Returns the universal DateTime.
