# ProtoFlux:IsLetterOrDigit

> Source: https://wiki.resonite.com/ProtoFlux:IsLetterOrDigit

Is Letter Or Digit

Character

\*

Characters

The **Is Letter Or Digit** node takes in a character literal and returns if that character is a letter or a digit. It is a shorthand of doing an [OR Operation](https://wiki.resonite.com/ProtoFlux:OR "ProtoFlux:OR") on the results of the [Is Letter](https://wiki.resonite.com/ProtoFlux:Is_Letter "ProtoFlux:Is Letter") and [Is Digit](https://wiki.resonite.com/ProtoFlux:Is_Digit "ProtoFlux:Is Digit") nodes.

## Inputs

### Character ( [char](https://wiki.resonite.com/Type:Char "Type:Char"))

The character literal to check.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns `true` if `Character` is a letter or digit, `false` otherwise.
