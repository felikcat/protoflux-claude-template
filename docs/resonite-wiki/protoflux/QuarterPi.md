# ProtoFlux:QuarterPi

> Source: https://wiki.resonite.com/ProtoFlux:QuarterPi

π/4

Constants

The **QuarterPi** or **π/4** node outputs the mathematical constant [π](https://en.wikipedia.org/wiki/Pi) divided by 4.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value of π ÷ 4, approximately `0.7853982`.
