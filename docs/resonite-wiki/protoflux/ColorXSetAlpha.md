# ProtoFlux:ColorXSetAlpha

> Source: https://wiki.resonite.com/ProtoFlux:ColorXSetAlpha

ColorXSet Alpha

colorX

\*

Value

Channels

Set Alpha is a ProtoFlux node that does a Set operation to a colorX on the Alpha channel.

## Inputs

### colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

the colorX to do a Set operation to the Alpha channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

the value to do a Set operation to the Alpha channel on the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

returns the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")) with a Set operation done to the Alpha channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
