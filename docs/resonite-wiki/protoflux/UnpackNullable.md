# ProtoFlux:UnpackNullable

> Source: https://wiki.resonite.com/ProtoFlux:UnpackNullable

Packing

The **Unpack Nullable** node takes in a [nullable](https://wiki.resonite.com/Type:Nullable%601 "Type:Nullable`1"), and returns the [value](https://wiki.resonite.com/Value_Type "Value Type") and if this nullable has a value.

## Inputs

### Nullable (Pseudo-Generic)

The nullable value.

## Outputs

### Value (Pseudo-Generic)

Returns the value itself.

### HasValue ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if this has a value.
