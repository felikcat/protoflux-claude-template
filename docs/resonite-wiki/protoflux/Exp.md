Math
The `Exp` node takes in a power and raises the base of [e](https://wiki.resonite.com/ProtoFlux:E "ProtoFlux:E") using that power, then returns the result. Similar to the [Pow](https://wiki.resonite.com/ProtoFlux:Pow "ProtoFlux:Pow") node but using e instead for the base. 
## Inputs
### N (Pseudo-generic)
The power for this node. 
## Outputs
### * (Pseudo-generic)
Returns the result of the base ([e](https://wiki.resonite.com/ProtoFlux:E "ProtoFlux:E")) raised to the provided power. 
## Examples
Let's consider the expression e². 
e ≈ 2.718282 
Using a calculator, we find: e² ≈ 2.71828 × 2.71828 ≈ 7.38906 
Explanation 
```
   Base: The base in this example is the constant e.
   Exponent: The exponent is 2.
   Power: e² is the power, which means e is raised to the second power.

```
