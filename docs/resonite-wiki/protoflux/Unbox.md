# ProtoFlux:Unbox

> Source: https://wiki.resonite.com/ProtoFlux:Unbox

Unbox

Input

\*

Core

The **Unbox<T>** node converts a boxed [object](https://wiki.resonite.com/Type:Object "Type:Object") back into its underlying [value type](https://wiki.resonite.com/Value_type "Value type").

## Inputs

### Input ( [Object](https://wiki.resonite.com/Type:Object "Type:Object"))

The object to unbox.

## Outputs

### \\* (Generic)

The unboxed value of the object.
