# ProtoFlux:DateTimeMillisecond

> Source: https://wiki.resonite.com/ProtoFlux:DateTimeMillisecond

Millisecond

DateTime

\*

DateTime

The `DateTime Millisecond` node takes in a [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime") and returns the millisecond of that DateTime.

## Inputs

### DateTime ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The DateTime we are getting info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The millisecond portion of the DateTime.
