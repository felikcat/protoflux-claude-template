The `Compose Bits Sbyte` node takes in 8 bits of [boolean](https://wiki.resonite.com/Type:Bool "Type:Bool") data and returns a [sbyte](https://wiki.resonite.com/Type:Sbyte "Type:Sbyte") result. Used for binary math and data transmission.

## Inputs

### Bit\# ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The bit for this sbyte value.

## Outputs

### \\* ( [sbyte](https://wiki.resonite.com/Type:Sbyte "Type:Sbyte"))

Returns the combined bits of this sbyte.
