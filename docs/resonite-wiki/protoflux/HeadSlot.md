# ProtoFlux:HeadSlot

> Source: https://wiki.resonite.com/ProtoFlux:HeadSlot

Head Slot

UserRoot

\*

User Root

The **Head Slot** node takes in a reference of a [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"), and if that UserRoot is active or being used, it returns the head slot of the [User](https://wiki.resonite.com/Type:User "Type:User").

## Inputs

### UserRoot ( [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"))

The user's user root.

## Outputs

### \\* ( [Slot](https://wiki.resonite.com/Slot "Slot"))

Returns the head slot.
