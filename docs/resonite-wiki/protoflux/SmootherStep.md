# ProtoFlux:SmootherStep

> Source: https://wiki.resonite.com/ProtoFlux:SmootherStep

Smoother Step

Value

\*

Shaping

The **Smoother Step** node takes in a value, then returns the smoothed shape value (similar to the [Smooth Step](https://wiki.resonite.com/ProtoFlux:Smooth_Step "ProtoFlux:Smooth Step") node, a curve that starts and ends fast and slows down in the middle, but more drastic on the ends).

## Inputs

### Value (Pseudo-Generic)

The value for the curve.

## Outputs

### \\* (Pseudo-Generic)

Returns the curve value.
