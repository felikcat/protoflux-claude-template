The `Add DateTime TimeSpan` node takes in a starting DateTime and a TimeSpan, then returns the new DateTime.

## Inputs

### A ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The starting DateTime.

### B ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The TimeSpan to add on to the DateTime.

## Outputs

### \\* ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The new DateTime result.
