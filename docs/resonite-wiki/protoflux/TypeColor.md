# ProtoFlux:TypeColor

> Source: https://wiki.resonite.com/ProtoFlux:TypeColor

Type Color

Type

\*

Utility

The `Type Color` node takes in a [Type](https://wiki.resonite.com/Type:Type "Type:Type"), and returns the color associated with that type's class inside the [FrooxEngine](https://wiki.resonite.com/FrooxEngine "FrooxEngine").

## Inputs

### Type ( [Type](https://wiki.resonite.com/Type:Type "Type:Type"))

The type to get the color from.

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

The type's specific class color.
