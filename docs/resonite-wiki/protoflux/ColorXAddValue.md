# ProtoFlux:ColorXAddValue

> Source: https://wiki.resonite.com/ProtoFlux:ColorXAddValue

colorX

\*

Value

Channels

Add Value is a ProtoFlux node that does a Add operation to a colorX on the Value channel.

## Inputs

### colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

the colorX to do a Add operation to the Value channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

the value to do a Add operation to the Value channel on the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

returns the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")) with a Add operation done to the Value channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
