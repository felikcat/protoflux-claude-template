New Line

Constants
The `New Line` node is a character literal constant that represents a new line in text. Note that this node comes in two versions: A [char](https://wiki.resonite.com/Type:Char "Type:Char") version and a [String](https://wiki.resonite.com/Type:String "Type:String") version. 
## Outputs
### * ([char](https://wiki.resonite.com/Type:Char "Type:Char") or [String](https://wiki.resonite.com/Type:String "Type:String"))
The character literal constant of the New Line character. 
For the char node, this will output `U+000A`, or a line feed character (`\n`). 
For the string node, the output will be a Carriage Return (`U+000D`, `\r`) followed by a line feed (`\r\n`) on Windows (and other non-Unix) platforms, or a singular line feed (`\n`) on Unix platforms (Linux, MacOS, ...). This is local to each user.
