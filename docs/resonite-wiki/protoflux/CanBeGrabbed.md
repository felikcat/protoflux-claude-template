# ProtoFlux:CanBeGrabbed

> Source: https://wiki.resonite.com/ProtoFlux:CanBeGrabbed

Can Be Grabbed

Slot

\*

Grabber

Grabbable

The `Can Be Grabbed` node takes in a [slot](https://wiki.resonite.com/Slot "Slot") and a [grabber](https://wiki.resonite.com/Type:Grabber "Type:Grabber") from a user, and checks if this slot can be grabbed using that provided grabber. It will not check if it is being grabbed, only if it could be grabbed.

## Inputs

### Slot ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The Slot to check if it can be grabbed.

### Grabber ( [Grabber](https://wiki.resonite.com/Type:Grabber "Type:Grabber"))

The grabber on a user (`LeftHand` or `RightHand`).

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if this slot can be grabbed with the provided grabber from a user.
