# ProtoFlux:WorldTimeHalfFloat

> Source: https://wiki.resonite.com/ProtoFlux:WorldTimeHalfFloat

T/2

\*

Time

The `World Time Half Float` node returns the total world time that this world has been open for divided by 2.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you want to use the normal world time value, use [World Time Float](https://wiki.resonite.com/ProtoFlux:World_Time_Float "ProtoFlux:World Time Float") instead.


## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Returns the total world time as a float divided by 2.
