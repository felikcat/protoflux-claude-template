# ProtoFlux:TypeDelta

> Source: https://wiki.resonite.com/ProtoFlux:TypeDelta

Type Delta

\*

Keyboard

The `Type Delta` node returns the new key press this frame. This happens very quickly (within a frame), if you plan on using this data, consider writing this to a variable.

## Outputs

### \\* ( [String](https://wiki.resonite.com/String "String"))

The key change for this frame.
