# ProtoFlux:EarsRotation

> Source: https://wiki.resonite.com/ProtoFlux:EarsRotation

Ears Rotation

\*

Local Output

The **Ears Rotation** node returns the local user's audio output (hearing audio) rotation.

The [Camera](https://wiki.resonite.com/Camera "Camera") has the ability to override this rotation by using the `Audio from camera viewpoint` setting, and if true, will return this new rotation value, otherwise it will use the user's head proxy's rotation.

## Outputs

### \\* ( [floatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ"))

Returns the local user's audio output (hearing audio) rotation.
