# ProtoFlux:ObjectFieldDrive

> Source: https://wiki.resonite.com/ProtoFlux:ObjectFieldDrive

_due to wiki limitations, this node does not currently have an HTML rendering of the node._

The **Object Field Drive<T>** node creates a [drive](https://wiki.resonite.com/Drive "Drive") on a field with the value of its input. The field type allows [object types](https://wiki.resonite.com/Type:Object "Type:Object").

See [ProtoFlux Tool Usage](https://wiki.resonite.com/ProtoFlux#Interfacing_With_Components_/_Slots "ProtoFlux") for how to make this node.

The field that is driven is chosen on the \`Proxy\` sub-class of the component.

## Inputs

### \\* (Generic)

A value to drive the internal field with.
