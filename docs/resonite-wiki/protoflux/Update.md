Flow

The **Update** node sends a pulse at every [engine update](https://wiki.resonite.com/Engine_update "Engine update") from the `UpdatingUser`.

## Outputs

### OnUpdate ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Sends a pulse every engine update from the `UpdatingUser`.

## Globals

### UpdatingUser ( [User](https://wiki.resonite.com/Type:User "Type:User"))

The user that the `OnUpdate` pulse will be generated from and owned by.

This input can be [driven](https://wiki.resonite.com/Drives "Drives") by a [Local User](https://wiki.resonite.com/ProtoFlux:Local_User "ProtoFlux:Local User") node to generate impulses for every user. However, it would be better to use [Local Update](https://wiki.resonite.com/ProtoFlux:Local_Update "ProtoFlux:Local Update") instead of this node for this purpose.

### SkipIfNull ( [Nullable](https://wiki.resonite.com/Type:Nullable%601 "Type:Nullable`1") < [bool](https://wiki.resonite.com/Type:Bool "Type:Bool") >)

If `True`, no pulses will be generated if `UpdatingUser` is null. Otherwise, impulses will be generated by the [host user](https://wiki.resonite.com/ProtoFlux:Host_User "ProtoFlux:Host User") if `UpdatingUser` is null.
