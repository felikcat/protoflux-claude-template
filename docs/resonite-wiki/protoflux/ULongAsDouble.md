# ProtoFlux:ULongAsDouble

> Source: https://wiki.resonite.com/ProtoFlux:ULongAsDouble

ULong As Double

Value

\*

Binary

The `ULong As Double` node takes in a [ulong](https://wiki.resonite.com/Type:Ulong "Type:Ulong") and converts it into a [double](https://wiki.resonite.com/Type:Double "Type:Double").

## Inputs

### Value ( [ulong](https://wiki.resonite.com/Type:Ulong "Type:Ulong"))

The ulong to convert.

## Outputs

### \\* ( [double](https://wiki.resonite.com/Type:Double "Type:Double"))

Returns a double value.
