# ProtoFlux:DateTimeKind

> Source: https://wiki.resonite.com/ProtoFlux:DateTimeKind

Kind

DateTime

\*

DateTime

The `DateTime Kind` node takes in a [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime") and returns the [enum](https://wiki.resonite.com/Category:Enums "Category:Enums")`Kind` of that DateTime.

## Inputs

### DateTime ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The DateTime we are getting info from.

## Outputs

### \\* ( [DateTimeKind](https://wiki.resonite.com/Type:DateTimeKind "Type:DateTimeKind"))

The kind portion of the DateTime (`Unspecified`, `Utc`, or `Local`).
