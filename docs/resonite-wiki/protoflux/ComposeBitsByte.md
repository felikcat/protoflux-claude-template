The `Compose Bits Byte` node takes in 8 bits of [boolean](https://wiki.resonite.com/Type:Bool "Type:Bool") data and returns a [byte](https://wiki.resonite.com/Type:Byte "Type:Byte") result. Used for binary math and data transmission.

## Inputs

### Bit\# ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The bit for this byte value.

## Outputs

### \\* ( [byte](https://wiki.resonite.com/Type:Byte "Type:Byte"))

Returns the combined bits of this byte.
