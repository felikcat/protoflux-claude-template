# ProtoFlux:ControllerSlot

> Source: https://wiki.resonite.com/ProtoFlux:ControllerSlot

Controller Slot

UserRoot

\*

Side

User Root

The **Controller Slot** node takes in a reference of a [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot") and a side, and if that UserRoot is active or being used, it returns a [Slot](https://wiki.resonite.com/Slot "Slot") reference of the user's controller.

## Inputs

### UserRoot ( [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"))

The user's user root.

### Side ( [Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality"))

The side to use.

## Outputs

### \\* ( [Slot](https://wiki.resonite.com/Type:Slot "Type:Slot"))

Returns the controller slot under the user.
