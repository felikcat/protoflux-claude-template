The `Tool Events` node takes in a referenced [ITool](https://wiki.resonite.com/Type:ITool "Type:ITool"), and fires events if the [tool](https://wiki.resonite.com/Tools "Tools") is equipped or dequipped from the user.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you are looking for tool events that are more custom, try the [Raw Data Tool Events](https://wiki.resonite.com/ProtoFlux:RawDataTool_Events "ProtoFlux:RawDataTool Events") node instead.


## Outputs

### Equipped ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Fires when the tool is equipped.

### Dequipped ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Fires when the tool is dequipped.

## Globals

### Tool ( [ITool](https://wiki.resonite.com/Type:ITool "Type:ITool"))

The tool component reference.
