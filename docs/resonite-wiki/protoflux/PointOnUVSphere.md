# ProtoFlux:PointOnUVSphere

> Source: https://wiki.resonite.com/ProtoFlux:PointOnUVSphere

Point On UV Sphere

UV

\*

Radius

Geometry 3D

The **Point On UV Sphere** node takes a sphere's UV coordinates and the radius of the sphere, then returns the point on that sphere.

## Inputs

### UV ( [float2](https://wiki.resonite.com/Type:Float2 "Type:Float2"))

The sphere UV.

### Radius ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The sphere radius.

## Outputs

### \\* ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the point on the sphere from the provided UV.
