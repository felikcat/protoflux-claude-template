Get Slot Active Self
Instance
*
Info
Gets the value of the provided Instance [Slot's](https://wiki.resonite.com/Slot "Slot") Active field. 
This Protoflux node does not update by itself on the Active field change. If you need to react to a Slot's activity dynamically, consider sourcing the Slot's Active field instead. 
## Inputs
### Instance ([Slot](https://wiki.resonite.com/Slot "Slot"))
[Slot](https://wiki.resonite.com/Slot "Slot") to check. 
## Outputs
### * ([bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))
Instance ([Slot](https://wiki.resonite.com/Slot "Slot"))'s Active field.
