# ProtoFlux:IsStringEmpty

> Source: https://wiki.resonite.com/ProtoFlux:IsStringEmpty

Is String Empty

A

\*

Strings

Is String Empty is a ProtoFlux node that checks if A ( [String](https://wiki.resonite.com/Type:String "Type:String")) is empty, but not null or white space.

## Inputs

### A ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to check if the string is null.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

If A ( [String](https://wiki.resonite.com/Type:String "Type:String")) is empty, but not null or white space.
