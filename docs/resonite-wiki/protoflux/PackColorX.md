The **Pack ColorX** node combines the specified red, green, blue and alpha channels into a [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX") object after applying the specified [Color Profile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile"). This is not to be confused with [Pack Color](https://wiki.resonite.com/ProtoFlux:Pack_Color "ProtoFlux:Pack Color"), which does not use color profiles, and results in a [Color](https://wiki.resonite.com/Type:Color "Type:Color") object.

## Inputs

### R ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Red Channel of resulting [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")

### G ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Blue Channel of resulting [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")

### B ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Green Channel of resulting [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")

### A ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Alpha (Aka Transparency) Channel of resulting [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")

### Profile ( [ColorProfile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile"))

[ColorProfile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile") of resulting [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"). Can be either Linear, sRGB, sRGBA.

## Outputs

### \\* ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

[ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX") output which is a compilation of all this node's inputs.
