The `Unpack ProtoFlux` node takes in the [Slot](https://wiki.resonite.com/Slot "Slot") root (where we want to unpack from), then unpacks them from the provided Slot. Many [community-based tools](https://wiki.resonite.com/Community_Tools "Community Tools") (such as [Redprint](https://wiki.resonite.com/Redprint "Redprint")) use this to unpack nodes into the world.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Calls an impulse to unpack our nodes from a slot.

### Root ( [Slot](https://wiki.resonite.com/Type:Slot "Type:Slot"))

The Slot we want to unpack our nodes from.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Continues the code from here.
