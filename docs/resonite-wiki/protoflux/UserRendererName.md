# ProtoFlux:UserRendererName

> Source: https://wiki.resonite.com/ProtoFlux:UserRendererName

User Renderer Name

User

\*

Info

The `User Renderer Name` node returns a string describing the renderer being used by a user.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The renderer in use by the user.
