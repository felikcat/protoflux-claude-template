# ProtoFlux:OrientationOnUVSphere

> Source: https://wiki.resonite.com/ProtoFlux:OrientationOnUVSphere

Orientation On UV Sphere

UV

\*

Geometry 3D

The **Orientation On UV Sphere** node takes a sphere's UV coordinates and returns the orientation of that sphere.

## Inputs

### UV ( [float2](https://wiki.resonite.com/Type:Float2 "Type:Float2"))

The sphere UV.

## Outputs

### \\* ( [floatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ"))

Returns the sphere rotation from the provided UV.
