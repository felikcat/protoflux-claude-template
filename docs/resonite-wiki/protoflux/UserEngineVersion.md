# ProtoFlux:UserEngineVersion

> Source: https://wiki.resonite.com/ProtoFlux:UserEngineVersion

User Engine Version

User

\*

Info

The `User Engine Version` node returns the version of [FrooxEngine](https://wiki.resonite.com/FrooxEngine "FrooxEngine") being used by a user.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The user's running version of [FrooxEngine](https://wiki.resonite.com/FrooxEngine "FrooxEngine").
