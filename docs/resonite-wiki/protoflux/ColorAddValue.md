# ProtoFlux:ColorAddValue

> Source: https://wiki.resonite.com/ProtoFlux:ColorAddValue

Add Value

color

\*

Value

Channels

Add Value is a ProtoFlux node that does a Add operation to a color on the Value channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Add operation to the Value channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Add operation to the Value channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Add operation done to the Value channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
