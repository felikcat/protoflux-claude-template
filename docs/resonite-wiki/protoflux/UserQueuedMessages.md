# ProtoFlux:UserQueuedMessages

> Source: https://wiki.resonite.com/ProtoFlux:UserQueuedMessages

User Queued Messages

User

\*

Info

The `User Queued Messages` node returns the user's queued messages (also called packets), and shows how behind a user is and how much they need to catch up by.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The user's queued packets.
