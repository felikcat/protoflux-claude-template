The `Compose Bits Ushort` node takes in 16 bits of [boolean](https://wiki.resonite.com/Type:Bool "Type:Bool") data and returns a [ushort](https://wiki.resonite.com/Type:Ushort "Type:Ushort") result. Used for binary math and data transmission.

## Inputs

### Bit\# ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The bit for this ushort value.

## Outputs

### \\* ( [ushort](https://wiki.resonite.com/Type:Ushort "Type:Ushort"))

Returns the combined bits of this ushort.
