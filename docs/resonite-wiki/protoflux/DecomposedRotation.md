# ProtoFlux:DecomposedRotation

> Source: https://wiki.resonite.com/ProtoFlux:DecomposedRotation

DecomposedRotation

A

\*

Matrix

The **Decomposed Rotation** node takes in a matrix value, and returns either a [floatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ") or a [doubleQ](https://wiki.resonite.com/Type:DoubleQ "Type:DoubleQ") by decomposing it.

## Inputs

### A (Pseudo-Generic)

The matrix value.

## Outputs

### \\* (Pseudo-Generic)

Returns the decomposed rotation.
