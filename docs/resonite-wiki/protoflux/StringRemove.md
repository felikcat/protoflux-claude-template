# ProtoFlux:StringRemove

> Source: https://wiki.resonite.com/ProtoFlux:StringRemove

String Length

Str

\*

StartIndex

Length

Strings

String Remove is a ProtoFlux node that allows you to remove part of a provided Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) of a given Length ( [int](https://wiki.resonite.com/Type:Int "Type:Int")) starting at StartIndex ( [int](https://wiki.resonite.com/Type:Int "Type:Int")).

## Inputs

### Str ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to remove a segment from.

### StartIndex ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The starting point of removal from Str ( [String](https://wiki.resonite.com/Type:String "Type:String")).

### Length ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

How long of a segment to remove from Str ( [String](https://wiki.resonite.com/Type:String "Type:String")).

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) with a segment starting at StartIndex ( [int](https://wiki.resonite.com/Type:Int "Type:Int")) to Length ( [int](https://wiki.resonite.com/Type:Int "Type:Int")) \+ StartIndex ( [int](https://wiki.resonite.com/Type:Int "Type:Int")) removed.
