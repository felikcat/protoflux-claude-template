# ProtoFlux:InvertedQuarterPi

> Source: https://wiki.resonite.com/ProtoFlux:InvertedQuarterPi

1/(π/4)

Constants

The **InvertedQuarterPi** or **1/(π/4)** node outputs the value of 1 ÷ ( [π](https://en.wikipedia.org/wiki/Pi) ÷ 4).

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value of 1 ÷ (π ÷ 4), approximately `1.273239`.
