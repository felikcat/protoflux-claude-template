From Euler
Angles
*
Rotation
The **From Euler** node takes in [Euler Angles](https://en.wikipedia.org/wiki/Euler_angles) (a [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3") or [double3](https://wiki.resonite.com/Type:Double3 "Type:Double3")), and returns the converted value as a [Quaternion](https://en.wikipedia.org/wiki/Quaternion). 
## Inputs
### Angles (Pseudo-Generic)
The euler angles we want to convert. 
## Outputs
### * (Pseudo-Generic)
Returns the quaternion equivalent.
