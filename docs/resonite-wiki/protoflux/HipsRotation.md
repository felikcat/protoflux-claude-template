# ProtoFlux:HipsRotation

> Source: https://wiki.resonite.com/ProtoFlux:HipsRotation

Hips Rotation

UserRoot

\*

User Root

The **Hips Rotation** node takes in a reference of a [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"), and if that UserRoot is active or being used, it returns the hip rotation value of the [User](https://wiki.resonite.com/Type:User "Type:User").

## Inputs

### UserRoot ( [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"))

The user's user root.

## Outputs

### \\* ( [floatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ"))

Returns the hip rotation value.
