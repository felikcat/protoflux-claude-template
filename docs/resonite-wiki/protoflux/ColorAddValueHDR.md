# ProtoFlux:ColorAddValueHDR

> Source: https://wiki.resonite.com/ProtoFlux:ColorAddValueHDR

Add Value HDR

color

\*

Value

Channels

Add Value HDR is a ProtoFlux node that does a Add operation to a color on the Value HDR channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Add operation to the Value HDR channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Add operation to the Value HDR channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Add operation done to the Value HDR channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
