Color To HSL converts a [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX") to [HSL](https://en.wikipedia.org/wiki/HSL_and_HSV). The result is represented as 3 separate [Float](https://wiki.resonite.com/Type:Float "Type:Float") values for Hue, Saturation, and Lightness.

## Inputs

### Color ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

The color to calculate the HSL equivalent of.

## Outputs

### H ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The hue of the input color.

### S ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The saturation of the input color.

### L ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The lightness of the input color.
