# ProtoFlux:TimeSpanFromDays

> Source: https://wiki.resonite.com/ProtoFlux:TimeSpanFromDays

Time Span From Days

Value

\*

Timespan

The `TimeSpan From Days` node takes in the amount of days and converts it into a [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan") that shows the span of time in days. When using a [display](https://wiki.resonite.com/ProtoFlux:Value_Display "ProtoFlux:Value Display") node, the TimeSpan will show the extra information.

## Inputs

### Value ( [double](https://wiki.resonite.com/Type:Double "Type:Double"))

The amount of days.

## Outputs

### \\* ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

Returns the TimeSpan with the amount of days.
