Anchors

Anchor events is a protoflux node that gives changes in an anchor as events without having to use a [Fire On Change](https://wiki.resonite.com/ProtoFlux:Fire_On_Change "ProtoFlux:Fire On Change"). It will also tell you the user that fired those events.

## Outputs

### OnAnchored ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Fires when a user is anchored to Anchor ( [AvatarAnchor](https://wiki.resonite.com/Type:AvatarAnchor "Type:AvatarAnchor")).

### OnReleased ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Fires when a user is unanchored from Anchor ( [AvatarAnchor](https://wiki.resonite.com/Type:AvatarAnchor "Type:AvatarAnchor")).

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

## Globals

### Anchor ( [AvatarAnchor](https://wiki.resonite.com/Type:AvatarAnchor "Type:AvatarAnchor"))

The anchor to monitor for user anchoring events.
