# ProtoFlux:ColorXLuminance

> Source: https://wiki.resonite.com/ProtoFlux:ColorXLuminance

Color XLuminance

Color

\*

Colors

Color Luminance calculates the [relative luminance](https://en.wikipedia.org/wiki/Relative_luminance) of an input color.

This node is currently bugged and only provides correct results for the [Linear color profile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile")[\[1\]](https://wiki.resonite.com/ProtoFlux:ColorXLuminance#cite_note-1).

## Inputs

### Color ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

The color to calculate the relative luminance of.

## Outputs

### \\* ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The relative luminance of the input color.
