# ProtoFlux:ComposeScaleRotation

> Source: https://wiki.resonite.com/ProtoFlux:ComposeScaleRotation

ComposeScaleRotation

Scale

\*

Rotation

Matrix

The **Compose Scale Rotation** node takes in a scale value and a rotation value, and composes it into a matrix table value.

## Inputs

### Scale (Pseudo-Generic)

The scale value.

### Rotation (Pseudo-Generic)

The rotation value.

## Outputs

### \\* (Pseudo-Generic)

Returns the composed matrix.
