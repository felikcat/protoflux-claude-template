Get Left
Instance
*
Direction
The **Get Left** node returns the left [direction](https://wiki.resonite.com/Slot_transforms#Directions "Slot transforms") of the provided `Instance` slot in [global space](https://wiki.resonite.com/Global_space "Global space"). 
## Inputs
### Instance ([Slot](https://wiki.resonite.com/Slot "Slot"))
The slot to find the left direction of. 
## Outputs
### * ([float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))
The left drection of the provided `Instance` in global space.
