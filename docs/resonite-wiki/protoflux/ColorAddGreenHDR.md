# ProtoFlux:ColorAddGreenHDR

> Source: https://wiki.resonite.com/ProtoFlux:ColorAddGreenHDR

Add Green HDR

color

\*

Value

Channels

Add Green HDR is a ProtoFlux node that does a Add operation to a color on the Green HDR channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Add operation to the Green HDR channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Add operation to the Green HDR channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Add operation done to the Green HDR channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
