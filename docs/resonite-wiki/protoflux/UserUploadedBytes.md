# ProtoFlux:UserUploadedBytes

> Source: https://wiki.resonite.com/ProtoFlux:UserUploadedBytes

User Uploaded Bytes

User

\*

Info

The `User Uploaded Bytes` node returns a user's total uploaded bytes.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [ulong](https://wiki.resonite.com/Type:Ulong "Type:Ulong"))

The total uploaded bytes of the user.
