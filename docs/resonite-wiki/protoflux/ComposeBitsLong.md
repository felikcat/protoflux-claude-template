The `Compose Bits Long` node takes in 64 bits of [boolean](https://wiki.resonite.com/Type:Bool "Type:Bool") data and returns a [long](https://wiki.resonite.com/Type:Long "Type:Long") result. Used for binary math and data transmission.

![](https://wiki.resonite.com/images/d/dd/InformationIcon.svg)

This node is commonly used for holding a lot of data for a [Cloud Variable](https://wiki.resonite.com/Cloud_Variables "Cloud Variables").


## Inputs

### Bit\# ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The bit for this long value.

## Outputs

### \\* ( [ulong](https://wiki.resonite.com/Type:Ulong "Type:Ulong"))

Returns the combined bits of this long.
