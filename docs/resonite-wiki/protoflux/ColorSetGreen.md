# ProtoFlux:ColorSetGreen

> Source: https://wiki.resonite.com/ProtoFlux:ColorSetGreen

Set Green

color

\*

Value

Channels

Set Green is a ProtoFlux node that does a Set operation to a color on the Green channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Set operation to the Green channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Set operation to the Green channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Set operation done to the Green channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
