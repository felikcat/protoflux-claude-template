# ProtoFlux:ClipLength

> Source: https://wiki.resonite.com/ProtoFlux:ClipLength

Clip Length

Source

\*

Media

Clip Length is a ProtoFlux node that returns an [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable")'s length in seconds.

This node has the following variants, with their respective outputs:

| Variant | Node Type | Output Example |
| --- | --- | --- |
| Clip Length ( [float](https://wiki.resonite.com/Types:Float "Types:Float")) | `Clip Length Float` | `10.000087` (less precision) |
| Clip Length ( [double](https://wiki.resonite.com/Types:Double "Types:Double")) | `Clip Length Double` | `10.0008374623287624` |

## Inputs

### Source ( [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable"))

The clip to check if it is playing

## Outputs

### \\* (Pseudo-generic)

The length of the provided Source ( [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable")) in seconds.
