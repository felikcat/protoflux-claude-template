# ProtoFlux:PreviousValue

> Source: https://wiki.resonite.com/ProtoFlux:PreviousValue

Previous Value < T >

Value

\*

Enums

See [Enums](https://wiki.resonite.com/Category:Enums "Category:Enums") for a more in depth explaination on what Enums are.

Previous Value is a [ProtoFlux](https://wiki.resonite.com/ProtoFlux "ProtoFlux") node that when given an [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") will give the previous enum numerically according to the [Enums](https://wiki.resonite.com/Category:Enums "Category:Enums") page. "\*" and "Value" will never be different enum types.

## Inputs

### Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic)

The Enum to start at.

## Outputs

### \\* ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic)

The previous Enum before Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic) numerically.
This node will wrap around forwards and backwards for the output.
