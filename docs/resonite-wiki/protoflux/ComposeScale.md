# ProtoFlux:ComposeScale

> Source: https://wiki.resonite.com/ProtoFlux:ComposeScale

ComposeScale

Scale

\*

Matrix

The **Compose Scale** node takes in a scale value and composes it into a matrix table value. This just enters the values diagonally on the matrix (top-left to bottom-right).

## Inputs

### Scale (Pseudo-Generic)

The scale value.

## Outputs

### \\* (Pseudo-Generic)

Returns the composed matrix.
