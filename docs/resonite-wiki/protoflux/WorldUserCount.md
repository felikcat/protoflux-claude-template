# ProtoFlux:WorldUserCount

> Source: https://wiki.resonite.com/ProtoFlux:WorldUserCount

World User Count

\*

Info

The `World User Count` node returns this world's current user count in the world now. This will include users that are AFK/focused away from the world.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The world's current user count.
