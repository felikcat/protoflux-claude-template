# ProtoFlux:UserRuntimeVersion

> Source: https://wiki.resonite.com/ProtoFlux:UserRuntimeVersion

User Runtime Version

User

\*

Info

The `User Runtime Version` node returns the runtime version in use, which is typically the version of .NET.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The version of .NET being used, such as "10.0.0".
