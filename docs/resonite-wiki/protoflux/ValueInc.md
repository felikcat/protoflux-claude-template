# ProtoFlux:ValueInc

> Source: https://wiki.resonite.com/ProtoFlux:ValueInc

+1

N

\*

Operators

The **Value Inc** node takes in a value and returns that value +1.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you need to increment a value from an [impulse](https://wiki.resonite.com/Impulses "Impulses"), use the [Increment](https://wiki.resonite.com/ProtoFlux:Increment "ProtoFlux:Increment") node instead.

This node is similar to the [Add](https://wiki.resonite.com/ProtoFlux:Add "ProtoFlux:Add") node, but setting the add node to only add 1.

## Inputs

### \\* (Pseudo-Generic)

The value to increment by 1.

## Outputs

### \\* (Pseudo-Generic)

Returns the incremented value.
