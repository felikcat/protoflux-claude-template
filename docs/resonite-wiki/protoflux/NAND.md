B
Boolean
The **NAND** node takes in two value inputs (either a [bool](https://wiki.resonite.com/Type:Bool "Type:Bool") or a [number](https://wiki.resonite.com/Value_Type "Value Type") value), and functions as both a [logical NAND](https://en.wikipedia.org/wiki/Sheffer_stroke) or bitwise NAND depending on the type used. 
```
- If any or none of the inputs are true, then the result will be true.
- If both inputs are true, then the result is false.

```

## Inputs
### A (Pseudo-generic)
The first input value. 
### B (Pseudo-generic)
The second input value. 
## Outputs
### * (Pseudo-generic)
Returns the result of a logical NAND of `A` and `B` if the node is a [bool](https://wiki.resonite.com/Type:Bool "Type:Bool") type, bitwise NAND otherwise. 
## See Also
  * Wikipedia's definition of [logical NAND](https://en.wikipedia.org/wiki/Sheffer_stroke) & [nand gate](https://en.wikipedia.org/wiki/NAND_gate).
