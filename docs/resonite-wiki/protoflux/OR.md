The **OR** node takes in two value inputs (either a [bool](https://wiki.resonite.com/Type:Bool "Type:Bool") or a [number](https://wiki.resonite.com/Value_Type "Value Type") value), and functions as both a [logical OR](https://en.wikipedia.org/wiki/Logical_disjunction) or [bitwise OR](https://en.wikipedia.org/wiki/Bitwise_operation#OR) depending on the type used.

```
If at least one of the inputs are true, then the result will be true.
```

## Inputs

### A (Pseudo-generic)

The first input value.

### B (Pseudo-generic)

The second input value.

## Outputs

### \\* (Pseudo-generic)

Returns the result of a logical OR of `A` and `B` if the node is a [bool](https://wiki.resonite.com/Type:Bool "Type:Bool") type, bitwise OR otherwise.
