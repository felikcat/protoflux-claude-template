# ProtoFlux:UserRecordingVoiceMessage

> Source: https://wiki.resonite.com/ProtoFlux:UserRecordingVoiceMessage

User Recording Voice Message

User

\*

Info

The **User Recording Voice Message** node returns if a user is currently recording a voice message in the [Contacts Menu](https://wiki.resonite.com/Contacts_Menu "Contacts Menu").

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user to check.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Outputs `true` if the user is recording a voice message, otherwise outputs `false`.
