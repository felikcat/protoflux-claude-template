# ProtoFlux:IsHostUser

> Source: https://wiki.resonite.com/ProtoFlux:IsHostUser

Is User Host

User

\*

Status

The `Is User Host` node returns if the user is the host user or not.

![](https://wiki.resonite.com/images/f/fe/WarningIcon.svg)

Beware, if the host is the headless, this will respect it and your code may work differently depending how the code is written using this node.


## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The user being the host user or not.
