# ProtoFlux:OtherSide

> Source: https://wiki.resonite.com/ProtoFlux:OtherSide

Other Side

Node

\*

Body Nodes

Other Side is a ProtoFlux node that returns the opposite side equivalent of the provided Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) on an anthro body.

## Inputs

### Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))

The node to flip to the opposite side.

## Outputs

### \\* ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))

Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) but on the opposite side of an anthro body. This value will be "NONE" if no such conversion exists (Like spine or head).
