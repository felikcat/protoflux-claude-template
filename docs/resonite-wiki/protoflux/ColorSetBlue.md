# ProtoFlux:ColorSetBlue

> Source: https://wiki.resonite.com/ProtoFlux:ColorSetBlue

Set Blue

color

\*

Value

Channels

Set Blue is a ProtoFlux node that does a Set operation to a color on the Blue channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Set operation to the Blue channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Set operation to the Blue channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Set operation done to the Blue channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
