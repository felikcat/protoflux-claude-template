Multiply String is a ProtoFlux node that takes an input string A ([String](https://wiki.resonite.com/Type:String "Type:String")) and makes it repeat B ([int](https://wiki.resonite.com/Types:Int "Types:Int")) times for the output * ([String](https://wiki.resonite.com/Types:String "Types:String")). 
## Inputs
### A ([String](https://wiki.resonite.com/Type:String "Type:String"))
The string to repeat B ([int](https://wiki.resonite.com/Types:Int "Types:Int")) times. 
### B ([int](https://wiki.resonite.com/Types:Int "Types:Int"))
How many times to repeat A ([String](https://wiki.resonite.com/Type:String "Type:String")). 
## Outputs
### * ([String](https://wiki.resonite.com/Types:String "Types:String"))
The string A ([String](https://wiki.resonite.com/Type:String "Type:String")) repeated B ([int](https://wiki.resonite.com/Types:Int "Types:Int")) times.
