# ProtoFlux:ViewScale

> Source: https://wiki.resonite.com/ProtoFlux:ViewScale

View Scale

\*

Local Output

The **View Scale** node returns the [local user](https://wiki.resonite.com/ProtoFlux:Local_User "ProtoFlux:Local User")'s view output scale in \[\[global space\].\
\
When in [desktop mode](https://wiki.resonite.com/Desktop_mode "Desktop mode") or [VR mode](https://wiki.resonite.com/VR_mode "VR mode"), this node will return the value of where your head proxy is scaled. When an item or object that can override your view is used (such as using a different head proxy, templated head proxy, or something else that moves your view point), this node will return a new scale value.\
\
## Outputs\
\
### \\* ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))\
\
Returns the local user's view output scale.\
