# ProtoFlux:PulseRandom

Pulse Random

\*

Impulses

+

-

Flow

**Pulse Random** will take an [impulse](https://wiki.resonite.com/Impulses "Impulses") as an input and pulse a random output each time.

You can manually add as many outputs as you wish to this node.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

The entry pulse.

## Outputs

### Impulses (List of [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

The random pulse out.
