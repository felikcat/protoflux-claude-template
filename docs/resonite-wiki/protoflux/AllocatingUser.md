# ProtoFlux:AllocatingUser

> Source: https://wiki.resonite.com/ProtoFlux:AllocatingUser

Allocating User

Element

\*

References

The `Allocating User` node takes in an [IWorldElement](https://wiki.resonite.com/Type:IWorldElement "Type:IWorldElement") and returns with the user that allocated (spawned/created) it.

## Inputs

### Element ( [IWorldElement](https://wiki.resonite.com/Type:IWorldElement "Type:IWorldElement"))

The world element that was spawned/created.

## Outputs

### \\* ( [User](https://wiki.resonite.com/Type:User "Type:User"))

Returns the user that spawned out the world element.
