Get Side
Node
*
Side
Body Nodes
Get Side is a ProtoFlux node that converts the provided Node ([BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) from its current side to the Side ([Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality")) provided. If given a node that is in the center of the body, then it will return a "None" value. If the Side ([Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality")) provided is the current side of the provided Node ([BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")), no change will occur between the Node ([BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) and the * ([BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) output. 
## Inputs
### Node ([BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))
The node to convert to the provided Side ([Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality")). 
### Side ([Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality"))
the side to convert Node ([BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) to. 
## Outputs
### * ([BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))
The provided Node ([BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) converted to the provided Side ([Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality")).
