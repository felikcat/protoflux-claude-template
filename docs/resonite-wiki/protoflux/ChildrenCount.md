# ProtoFlux:ChildrenCount

> Source: https://wiki.resonite.com/ProtoFlux:ChildrenCount

Children Count

Instance

\*

Slots

Returns the number of children of a [slot](https://wiki.resonite.com/Slot "Slot").

## Inputs

### Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The slot for read children count for.

## Outputs

### \\* ( [Int](https://wiki.resonite.com/Types:Int "Types:Int"))

The amount of children under the provided Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))
