# ProtoFlux:WavelengthColor

> Source: https://wiki.resonite.com/ProtoFlux:WavelengthColor

Wavelength Color

Wavelength

\*

Colors

The **Wavelength Color** node takes in a [float](https://wiki.resonite.com/Type:Float "Type:Float") number value (in nanometers) and converts that to a visible light spectrum color.

The table here shows the wavelength values that correspond to the output color:

| Color | Wavelength |
| --- | --- |
| **Violet** | 380–450 nm (688–789 THz frequency) |
| **Blue** | 450–495 nm. |
| **Green** | 495–570 nm. |
| **Yellow** | 570–590 nm. |
| **Orange** | 590–620 nm. |
| **Red** | 620–750 nm (400–484 THz frequency). |

## Inputs

### Wavelength ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The wavelength value in nanometers.

## Outputs

### \\* ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color value of the wavelength.
