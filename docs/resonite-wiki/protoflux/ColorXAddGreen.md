# ProtoFlux:ColorXAddGreen

> Source: https://wiki.resonite.com/ProtoFlux:ColorXAddGreen

ColorXAdd Green

colorX

\*

Value

Channels

Add Green is a ProtoFlux node that does a Add operation to a colorX on the Green channel.

## Inputs

### colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

the colorX to do a Add operation to the Green channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

the value to do a Add operation to the Green channel on the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

returns the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")) with a Add operation done to the Green channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
