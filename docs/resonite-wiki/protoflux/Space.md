# ProtoFlux:Space

> Source: https://wiki.resonite.com/ProtoFlux:Space

' ' (ˢᵖᵃᶜᵉ)

\*

Constants

The `Space` node is a character literal constant for the "Space" character (Unicode: U+0020).

This node is about the text character. Its usage should not be confused with that of [keyboard keys](https://wiki.resonite.com/Type:Key "Type:Key") used for handling input.

## Outputs

### \\* ( [char](https://wiki.resonite.com/Type:Char "Type:Char"))

The character literal constant of the Space character.
