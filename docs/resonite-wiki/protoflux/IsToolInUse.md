# ProtoFlux:IsToolInUse

> Source: https://wiki.resonite.com/ProtoFlux:IsToolInUse

Is Tool In Use

Tool

\*

Tools

The `Is Tool In Use` node takes in a [tool](https://wiki.resonite.com/Tools "Tools") reference and checks if a user is currently using it ("Using it" counts as the user pressing the primary or secondary buttons while the tool is held).

## Inputs

### Tool ( [ITool](https://wiki.resonite.com/Type:ITool "Type:ITool"))

The tool reference.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The result of the user using the tool or not.
