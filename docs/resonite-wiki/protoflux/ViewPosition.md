# ProtoFlux:ViewPosition

> Source: https://wiki.resonite.com/ProtoFlux:ViewPosition

View Position

\*

Local Output

The **View Position** node returns the local user's view output position.

When in [Desktop Mode](https://wiki.resonite.com/Desktop_Mode "Desktop Mode") or [VR Mode](https://wiki.resonite.com/VR_Mode "VR Mode"), this node will return the value of where your head proxy is positioned. When an item or object that can override your view is used (such as using a different head proxy, templated head proxy, or something else that moves your view point (such as the [AvatarUserRootOverrideAssigner](https://wiki.resonite.com/Component:AvatarUserRootOverrideAssigner "Component:AvatarUserRootOverrideAssigner") or [AvatarObjectSlot](https://wiki.resonite.com/Component:AvatarObjectSlot "Component:AvatarObjectSlot") components)), this node will return a new position value.

## Outputs

### \\* ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the local user's view output position.
