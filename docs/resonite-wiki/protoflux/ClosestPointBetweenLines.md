The **Closest Points Between Lines** node takes in 2 lines as inputs (position and direction). Then it returns the intersecting point(s) from those lines.

## Inputs

### LinePoint0 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The first line's point.

### LineDir0 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The first line's direction.

### LinePoint1 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The second line's point.

### LineDir0 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The second line's direction.

## Outputs

### Point0 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the first point.

### Point1 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the second point.
