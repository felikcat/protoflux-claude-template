# ProtoFlux:InverseRotation

> Source: https://wiki.resonite.com/ProtoFlux:InverseRotation

Inverse Rotation

Q

\*

Rotation

The **Inverse Rotation** node takes in a [Quaternion](https://en.wikipedia.org/wiki/Quaternion) value (a [floatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ") or [doubleQ](https://wiki.resonite.com/Type:DoubleQ "Type:DoubleQ")), and returns the inverted rotational value of the same type.

## Inputs

### Q (Pseudo-Generic)

The quaternion we want to invert.

## Outputs

### \\* (Pseudo-Generic)

Returns the inverted quaternion.
