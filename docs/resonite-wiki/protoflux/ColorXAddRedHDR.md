# ProtoFlux:ColorXAddRedHDR

> Source: https://wiki.resonite.com/ProtoFlux:ColorXAddRedHDR

ColorXAdd Red HDR

colorX

\*

Value

Channels

Add Red HDR is a ProtoFlux node that does a Add operation to a colorX on the Red HDR channel.

## Inputs

### colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

the colorX to do a Add operation to the Red HDR channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

the value to do a Add operation to the Red HDR channel on the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

returns the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")) with a Add operation done to the Red HDR channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
