The `Trigger Haptics In Hierarchy` node takes in a [Slot](https://wiki.resonite.com/Slot "Slot") hierarchy and will send out haptic pulses through it.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Calls an impulse to send haptic data to the given hierarchy.

### TargetHierarchy ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The slot hierarchy that specifically looks for your haptic components.

### RelativeIntensity ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

How much this will notify the user using haptics.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Fires after sending the haptic pulse.
