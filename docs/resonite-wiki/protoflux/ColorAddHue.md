# ProtoFlux:ColorAddHue

> Source: https://wiki.resonite.com/ProtoFlux:ColorAddHue

Add Hue

color

\*

Value

Channels

Add Hue is a ProtoFlux node that does a Add operation to a color on the Hue channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Add operation to the Hue channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Add operation to the Hue channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Add operation done to the Hue channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
