# ProtoFlux:Speed

> Source: https://wiki.resonite.com/ProtoFlux:Speed

Speed

Source

\*

Media

The Speed node tells the speed of the provided IPlayable.

## Inputs

### Source ( [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable"))

The [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable") to get the speed of.

## Outputs

### \\* ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

Output of the speed from the source.
