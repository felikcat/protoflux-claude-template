The **Simplex** set of nodes takes in a position (between 1 and 4 dimensions) and returns a [float](https://wiki.resonite.com/Type:Float "Type:Float") noise value. It uses the simplex noise algorithm (similar to Perlin noise), which means that changing the position smoothly will result in a smooth change in the output. Unlike other nodes in the /Math/Random category, Simplex nodes are deterministic, so each position will always result in the same value.

## Inputs

### Position ( [float](https://wiki.resonite.com/Type:Float "Type:Float") or [float2](https://wiki.resonite.com/Type:Float2 "Type:Float2") or [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3") or [float4](https://wiki.resonite.com/Type:Float4 "Type:Float4"))

The value used to generate a simplex noise.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The random noise result.
