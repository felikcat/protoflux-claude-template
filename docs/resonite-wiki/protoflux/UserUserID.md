# ProtoFlux:UserUserID

Users

This node returns a user's [user ID](https://wiki.resonite.com/User_ID "User ID") from their user object.

## Inputs

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

[User](https://wiki.resonite.com/Type:User "Type:User") you want the user ID of.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The user ID of the [user](https://wiki.resonite.com/Type:User "Type:User").
