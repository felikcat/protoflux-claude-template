This node detects collisions based on the [Collisions Table](https://wiki.resonite.com/Collider#Types_of_Collisions_Table "Collider").

## Output

### OnEvent ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

When the table's conditions including any exceptions to it are met for the Collider ( [ICollider](https://wiki.resonite.com/Type:ICollider "Type:ICollider")) and it's a collision start event this will send an impulse.

### Other ( [ICollider](https://wiki.resonite.com/Type:ICollider "Type:ICollider"))

The other collider that Collider ( [ICollider](https://wiki.resonite.com/Type:ICollider "Type:ICollider")) collided with for this node's event. This value only exists during OnEvent ( [impulse](https://wiki.resonite.com/Impulses "Impulses")).

## Globals

### Collider ( [ICollider](https://wiki.resonite.com/Type:ICollider "Type:ICollider"))

The [ICollider](https://wiki.resonite.com/Type:ICollider "Type:ICollider") to receive collision start events for.
