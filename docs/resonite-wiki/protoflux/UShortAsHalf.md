# ProtoFlux:UShortAsHalf

> Source: https://wiki.resonite.com/ProtoFlux:UShortAsHalf

UShort As Half

Value

\*

Binary

The `UShort As Half` node takes in a [ushort](https://wiki.resonite.com/Type:Ushort "Type:Ushort") and converts it into a [half](https://wiki.resonite.com/Type:Half "Type:Half").

## Inputs

### Value ( [ushort](https://wiki.resonite.com/Type:Ushort "Type:Ushort"))

The ushort to convert.

## Outputs

### \\* ( [half](https://wiki.resonite.com/Type:Half "Type:Half"))

Returns a half value.
