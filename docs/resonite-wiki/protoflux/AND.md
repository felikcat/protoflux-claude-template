# ProtoFlux:AND

> Source: https://wiki.resonite.com/ProtoFlux:AND

Boolean

The **AND** node takes in two value inputs (either a [bool](https://wiki.resonite.com/Type:Bool "Type:Bool") or a [number](https://wiki.resonite.com/Value_Type "Value Type") value), and functions as both a [logical AND](https://en.wikipedia.org/wiki/Logical_conjunction) or [bitwise AND](https://en.wikipedia.org/wiki/Bitwise_and#AND) depending on the type used.

```
If both inputs are true, then the result will be true.
```

![](https://wiki.resonite.com/images/d/dd/InformationIcon.svg)

This node's inputs default to true.


## Inputs

### A (Pseudo-generic)

The first input value.

### B (Pseudo-generic)

The second input value.

## Outputs

### \\* (Pseudo-generic)

Returns the result of a logical AND of `A` and `B` if the node is a [bool](https://wiki.resonite.com/Type:Bool "Type:Bool") type, bitwise AND otherwise.
