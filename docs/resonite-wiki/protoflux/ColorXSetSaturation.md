# ProtoFlux:ColorXSetSaturation

> Source: https://wiki.resonite.com/ProtoFlux:ColorXSetSaturation

ColorXSet Saturation

colorX

\*

Value

Channels

Set Saturation is a ProtoFlux node that does a Set operation to a colorX on the Saturation channel.

## Inputs

### colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

the colorX to do a Set operation to the Saturation channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

the value to do a Set operation to the Saturation channel on the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

returns the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")) with a Set operation done to the Saturation channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
