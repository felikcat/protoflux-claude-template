# ProtoFlux:SetTag

Info

The **Set Tag** node sets the [Tag](https://wiki.resonite.com/Tag "Tag") of the provided instance [slot](https://wiki.resonite.com/Slot "Slot") upon calling \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")).

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Call this to set the Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s tag to Tag ( [String](https://wiki.resonite.com/Type:String "Type:String")).

### Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))

[Slot](https://wiki.resonite.com/Slot "Slot") to set the property for.

### Tag ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The value to set Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s Tag field to when \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) is called.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Sends an impulse when Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s tag is set to Tag ( [String](https://wiki.resonite.com/Type:String "Type:String")) when \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) is called.
