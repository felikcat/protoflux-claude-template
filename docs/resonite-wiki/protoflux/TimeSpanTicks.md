# ProtoFlux:TimeSpanTicks

> Source: https://wiki.resonite.com/ProtoFlux:TimeSpanTicks

Ticks

TimeSpan

\*

Timespan

The `TimeSpan Ticks` node takes in a [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan") and returns with only the amount of ticks in that TimeSpan. Anything other than the ticks will be ignored.

## Inputs

### TimeSpan ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The TimeSpan we are getting info from.

## Outputs

### \\* ( [Long](https://wiki.resonite.com/Type:Long "Type:Long"))

Returns the amount of ticks in this TimeSpan as an long.
