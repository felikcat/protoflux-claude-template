# ProtoFlux:Reflect

> Source: https://wiki.resonite.com/ProtoFlux:Reflect

Reflect

A

\*

B

Vectors

The **Reflect** node takes in a vector value and a vector that acts as a "wall" to reflect from, and returns the reflected vector.

## Inputs

### A (Pseudo-Generic)

The vector that we want to reflect.

### B (Pseudo-Generic)

The "wall" we are reflecting from.

## Outputs

### \\* (Pseudo-Generic)

Returns a reflected vector.
