Concatenate Multi String is a protoflux node that combines multiple strings in order of the inputs into one combined string

## Inputs

### Inputs ( [String](https://wiki.resonite.com/Type:String "Type:String") (List))

List of string inputs to combine

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

All of Inputs ( [String](https://wiki.resonite.com/Type:String "Type:String") (List)) combined in order
