# ProtoFlux:NumberToEnum

> Source: https://wiki.resonite.com/ProtoFlux:NumberToEnum

NumberToEnum

Value

\*

Enums

See [Enums](https://wiki.resonite.com/Category:Enums "Category:Enums") for a more in depth explaination on what Enums are.
This node can be useful for turning numbers back into Enums, in cases like reading [Cloud Variables](https://wiki.resonite.com/Cloud_Variables "Cloud Variables") that can't store Enums normally.

For what to input for variants other than `IntToEnum`, please see [Complex Types in Components](https://wiki.resonite.com/Complex_Types_in_Components "Complex Types in Components")

This node has the following variants, with their respective outputs:

| Variant | Node Type | Output |
| --- | --- | --- |
| Number To Enum ( [Byte](https://wiki.resonite.com/Types:Byte "Types:Byte")) | `ByteToEnum` | provided Byte as any chosen ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) |
| Number To Enum ( [Int](https://wiki.resonite.com/Types:Int "Types:Int")) | `IntToEnum` | provided Int as any chosen ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) |
| Number To Enum ( [Long](https://wiki.resonite.com/Types:Long "Types:Long")) | `LongToEnum` | provided Long as any chosen ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) |
| Number To Enum ( [Sbyte](https://wiki.resonite.com/Types:Sbyte "Types:Sbyte")) | `SbyteToEnum` | provided Sbyte as any chosen ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) |
| Number To Enum ( [Short](https://wiki.resonite.com/Types:Short "Types:Short")) | `ShortToEnum` | provided Short as any chosen ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) |
| Number To Enum ( [Uint](https://wiki.resonite.com/Types:Uint "Types:Uint")) | `UintToEnum` | provided Uint as any chosen ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) |
| Number To Enum ( [Ulong](https://wiki.resonite.com/Types:Ulong "Types:Ulong")) | `UlongToEnum` | provided Ulong as any chosen ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) |
| Number To Enum ( [Ushort](https://wiki.resonite.com/Types:Ushort "Types:Ushort")) | `UshortToEnum` | provided Ushort as any chosen ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) |

## Inputs

### Value (Variant)

The whole number to convert to a Enum of the type chosen for \* ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic).

## Outputs

### \\* ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic)

The Enum with the chosen enum type that has a number matching Value (Variant) according to [Enums Page](https://wiki.resonite.com/Category:Enums "Category:Enums").
