String Join is a protoflux node that can take a Separator ( [String](https://wiki.resonite.com/Type:String "Type:String")), and a some Inputs (List of [String](https://wiki.resonite.com/Type:String "Type:String")) and combine them, separating each item with Separator ( [String](https://wiki.resonite.com/Type:String "Type:String")), or with nothing if Separator ( [String](https://wiki.resonite.com/Type:String "Type:String")) is not provided.

## Inputs

### Separator ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The item to separate the Inputs (List of [String](https://wiki.resonite.com/Type:String "Type:String")) when joined.

### Inputs (List of [String](https://wiki.resonite.com/Type:String "Type:String"))

The items to join together

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

Returns all Inputs (List of [String](https://wiki.resonite.com/Type:String "Type:String")) joined together in order while separated by Separator ( [String](https://wiki.resonite.com/Type:String "Type:String")).
