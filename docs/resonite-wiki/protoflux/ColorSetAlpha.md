# ProtoFlux:ColorSetAlpha

> Source: https://wiki.resonite.com/ProtoFlux:ColorSetAlpha

Set Alpha

color

\*

Value

Channels

Set Alpha is a ProtoFlux node that does a Set operation to a color on the Alpha channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Set operation to the Alpha channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Set operation to the Alpha channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Set operation done to the Alpha channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
