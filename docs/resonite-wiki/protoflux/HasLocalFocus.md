# ProtoFlux:HasLocalFocus

> Source: https://wiki.resonite.com/ProtoFlux:HasLocalFocus

Has Local Focus

Target

\*

UI

The **Has Local Focus** node takes in an [IFocusable](https://wiki.resonite.com/Type:IFocusable "Type:IFocusable") and returns if that focusable is currently focused by the local [user](https://wiki.resonite.com/User "User").

## Inputs

### Target ( [IFocusable](https://wiki.resonite.com/Type:IFocusable "Type:IFocusable"))

The target IFocusable reference to check.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Outputs true if the target IFocusable is currently focused by the local user.
