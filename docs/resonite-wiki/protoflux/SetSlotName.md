# ProtoFlux:SetSlotName

> Source: https://wiki.resonite.com/ProtoFlux:SetSlotName

Set Slot Name

\*

Next

Instance

Name

Info

Set's the name of the provided Instance [slot](https://wiki.resonite.com/Slot "Slot") upon calling \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")).

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Call this to set the Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s name to Name ( [String](https://wiki.resonite.com/Type:String "Type:String")).

### Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))

[Slot](https://wiki.resonite.com/Slot "Slot") to set the property for.

### Name ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The value to set Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s name field to when \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) is called.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

sends an impulse when Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s name field is set to Name ( [String](https://wiki.resonite.com/Type:String "Type:String")) when \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) is called.
