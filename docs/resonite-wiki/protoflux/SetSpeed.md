# ProtoFlux:SetSpeed

The Set Speed node tells a IPlayable to playback at a specific speed upon impulse.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

### Target ( [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable"))

The [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable") to set the position on.

### Speed ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

Set the speed of the playback on the target. 1 is equivalent to x1.0.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Sends an impulse after the initial call.
