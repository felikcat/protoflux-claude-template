# ProtoFlux:IsCharacterController

> Source: https://wiki.resonite.com/ProtoFlux:IsCharacterController

Is Character Controller

Collider

\*

Physics

The `Is Character Controller` node takes in a reference to an [ICollider](https://wiki.resonite.com/Type:ICollider "Type:ICollider") and returns if that ICollider is labeled as a [character controller](https://wiki.resonite.com/Component:CharacterController "Component:CharacterController") type and the character controller has a simulating user in the same [Slot](https://wiki.resonite.com/Slot "Slot").

## Inputs

### Collider ( [ICollider](https://wiki.resonite.com/Type:ICollider "Type:ICollider"))

The Collider we want to check if it is a character controller.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The result if it is a character controller or not.
