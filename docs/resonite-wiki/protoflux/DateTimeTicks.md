# ProtoFlux:DateTimeTicks

> Source: https://wiki.resonite.com/ProtoFlux:DateTimeTicks

Ticks

DateTime

\*

DateTime

The `DateTime Ticks` node takes in a [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime") and returns the amount of ticks of that DateTime. There is Microsoft documentation about [Ticks](https://learn.microsoft.com/en-us/dotnet/api/system.datetime.ticks?view=net-8.0).

## Inputs

### DateTime ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The DateTime we are getting info from.

## Outputs

### \\* ( [long](https://wiki.resonite.com/Type:Long "Type:Long"))

The amount of ticks of this DateTime.
