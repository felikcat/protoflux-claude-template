The `Adder` node takes in 3 boolean values (`A`, `B`, and `CarryIn`), and returns 2 boolean values (`Y` and `CarryOut`). In computing & electronics, this is called an [adder](https://en.wikipedia.org/wiki/Adder_(electronics)), and helps with boolean math.

## Inputs

### A ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The first boolean input value.

### B ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The second boolean input value.

### CarryIn ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The carry in boolean input value.

## Outputs

### Y ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The result boolean output value.

### CarryOut ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The carry out boolean output value.
