Get Type
Object
*
Utility
The `Get Type` node takes in an [Object](https://wiki.resonite.com/Type:Object "Type:Object"), and returns the [Type](https://wiki.resonite.com/Type:Type "Type:Type") of that object. For example, using a [float](https://wiki.resonite.com/Type:Float "Type:Float") and connecting it to this node will return `System.Single`. 
![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)
If you want something more familiar, you can also use the [Nice Type Name](https://wiki.resonite.com/ProtoFlux:Nice_Type_Name "ProtoFlux:Nice Type Name") node as well. 
## Inputs
### Type ([Object](https://wiki.resonite.com/Type:Object "Type:Object"))
The Object to get the type from. 
## Outputs
### * ([Type](https://wiki.resonite.com/Type:Type "Type:Type"))
Returns the long type name.
