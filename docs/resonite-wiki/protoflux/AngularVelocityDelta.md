# ProtoFlux:AngularVelocityDelta

> Source: https://wiki.resonite.com/ProtoFlux:AngularVelocityDelta

Angular Velocity Delta

AngularVelocity

\*

DeltaTime

Rotation

The **Angular Velocity Delta** node takes in the angular velocity and the change in time, then returns the rotation of that result.

## Inputs

### AngularVelocity (Pseudo-Generic)

The direction of the rotation.

### DeltaTime (Pseudo-Generic)

The change in time.

## Outputs

### \\* (Pseudo-Generic)

Returns the rotation of the delta of the angular velocity.
