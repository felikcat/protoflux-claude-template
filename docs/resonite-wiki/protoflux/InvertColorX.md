# ProtoFlux:InvertColorX

> Source: https://wiki.resonite.com/ProtoFlux:InvertColorX

Invert ColorX

Color

\*

Colors

The `Invert ColorX` node takes in a [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX") and mathematically inverts it, returning a new colorX.

## Inputs

### Color ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

The colorX we want to invert.

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

Returns the inverted colorX.
