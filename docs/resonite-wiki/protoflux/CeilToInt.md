# ProtoFlux:CeilToInt

> Source: https://wiki.resonite.com/ProtoFlux:CeilToInt

CeilToInt

N

\*

Math

The **Ceil To Int** node acts like [Ceil](https://wiki.resonite.com/ProtoFlux:Ceil "ProtoFlux:Ceil"), but will always return an [int](https://wiki.resonite.com/Type:Int "Type:Int").

## Inputs

### N (Generic)

Numeric value to ceil.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The ceiling of the input as an integer. For types with multiple values, all individual values will be rounded up to int.
