# ProtoFlux:RelativeBodyNode

> Source: https://wiki.resonite.com/ProtoFlux:RelativeBodyNode

Relative Body Node

Node

\*

Body Nodes

Relative Body Node is a ProtoFlux node that returns a hand if the Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) provided is a finger or a palm, returns Head if it is an eye, and returns "Root" otherwise. Unless it is "None" which returns "None".

## Inputs

### Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))

The node to find the relative node for.

## Outputs

### \\* ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))

Returns a hand if the Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) provided is a finger or a palm, returns Head if it is an eye, and returns "Root" otherwise. Unless it is "None" which returns "None".
