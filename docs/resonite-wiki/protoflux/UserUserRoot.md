# ProtoFlux:UserUserRoot

User User Root

User

Users

The `User User Root` node returns a user's [User Root](https://wiki.resonite.com/Component:UserRoot "Component:UserRoot").

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting the user root from.

## Outputs

### \\* ( [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"))

The user root of the user.
