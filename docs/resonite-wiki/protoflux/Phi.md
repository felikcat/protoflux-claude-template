# ProtoFlux:Phi

> Source: https://wiki.resonite.com/ProtoFlux:Phi

Ï†

Constants

The **Phi** node outputs the [Golden Ratio](https://en.wikipedia.org/wiki/Golden_ratio) mathematical constant.

## Inputs

None.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The golden ratio. Its [float](https://wiki.resonite.com/Type:Float "Type:Float") value is approximately `1.61803401`.
