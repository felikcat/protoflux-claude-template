Easing

The **Ease-In/Out Elastic** node takes in the length of time, amplitude, and period, and returns the curve values.

## Inputs

### Time (Pseudo-Generic)

The length of time for the curve.

### Amplitude (Pseudo-Generic)

The height of the curve.

### Period (Pseudo-Generic)

The length of the curve.

## Outputs

### \\* (Pseudo-Generic)

Returns the curve value.
