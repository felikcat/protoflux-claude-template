# ProtoFlux:IsToolEquipped

> Source: https://wiki.resonite.com/ProtoFlux:IsToolEquipped

Is Tool Equiped

Tool

\*

Tools

The **Is Tool Equiped** node checks to see if a [tool](https://wiki.resonite.com/Tools "Tools") reference is equipped on a user.

## Inputs

### \\* ( [ITool](https://wiki.resonite.com/Type:ITool "Type:ITool"))

The tool to check.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The result if this tool is equipped.
