The **Link** node is used to connect two different node groups into one. You can drag an output wire (must be [value](https://wiki.resonite.com/Value_Type "Value Type") or [object](https://wiki.resonite.com/Reference_Type "Reference Type"), not [call](https://wiki.resonite.com/Impulses "Impulses")) into the null input to connect it. This can be used to make it easier to pack separated nodes together, due to being part of the same group and selecting one node will select them all. 
![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)
This node can also be used to work around this bug (GitHub Issue: [#907](https://github.com/Yellow-Dog-Man/Resonite-Issues/issues/907)) by forcing the node groups to be rebuilt.
