# ProtoFlux:WorldMobileFriendly

> Source: https://wiki.resonite.com/ProtoFlux:WorldMobileFriendly

World Mobile Friendly

\*

Info

The `World Mobile Friendly` node returns this world's "Mobile Friendly" setting from the [Session](https://wiki.resonite.com/Session "Session") tab in the [Dash](https://wiki.resonite.com/Dash "Dash").

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The world's mobile friendly setting.
