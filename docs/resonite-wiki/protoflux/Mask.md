The **Mask** node takes in 2 values and is controlled by the `mask` input. This mask input can can be either a [bool2](https://wiki.resonite.com/Type:Bool2 "Type:Bool2"), [bool3](https://wiki.resonite.com/Type:Bool3 "Type:Bool3"), or [bool4](https://wiki.resonite.com/Type:Bool4 "Type:Bool4"). Depending what values are provided in the mask will then determine what values get used, which can be mixed and matched by the `mask`. 
![](https://wiki.resonite.com/images/d/dd/InformationIcon.svg)
This is similar to the [Conditional](https://wiki.resonite.com/ProtoFlux:Conditional "ProtoFlux:Conditional"), except this node allows for parts of the value to pass through and blocking others within a value. 
## Inputs
### OnTrue (Pseudo-Generic)
The true value provided to be masked. 
### OnFalse (Pseudo-Generic)
The false value provided to be masked. 
### Mask (Pseudo-Generic)
A [bool2](https://wiki.resonite.com/Type:Bool2 "Type:Bool2"), [bool3](https://wiki.resonite.com/Type:Bool3 "Type:Bool3"), and [bool4](https://wiki.resonite.com/Type:Bool4 "Type:Bool4") that can allow parts of either value to pass through into the output. 
## Outputs
### * (Pseudo-Generic)
The end result of the masked input.
