# ProtoFlux:UserMachineID

> Source: https://wiki.resonite.com/ProtoFlux:UserMachineID

User Machine ID

User

\*

Users

The `User Machine ID` node returns a machine ID from a user.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we want the machine ID from.

## Outputs

### \\* ( [String](https://wiki.resonite.com/String "String"))

The machine ID from the user.
