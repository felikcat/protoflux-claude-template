# ProtoFlux:EarsOverriden

> Source: https://wiki.resonite.com/ProtoFlux:EarsOverriden

Ears Overriden

\*

Local Output

The **Ears Overriden** node returns if the local user's audio output (hearing audio) is being overridden by something.

The [Camera](https://wiki.resonite.com/Camera "Camera") has the ability to override the user's audio output by using the `Audio from camera viewpoint` setting, and if true, will return that this is overriding the ears from the local user's head proxy.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if the local user's audio output (hearing audio) has been overridden by something (such as the [Camera](https://wiki.resonite.com/Camera "Camera")).
