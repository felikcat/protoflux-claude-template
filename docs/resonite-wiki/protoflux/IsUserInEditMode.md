Is User In Edit Mode
User
*
Status
The `Is User In Edit Mode` node returns if a user has the special permission of [Edit Mode](https://wiki.resonite.com/Edit_Mode "Edit Mode"). 
## Inputs
### User ([User](https://wiki.resonite.com/User "User"))
The user we are getting info from. 
## Outputs
### * ([bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))
The edit mode permission on this user.
