Get Active Locomotion Module
User
*
Locomotion
Get Active Locomotion Module is a ProtoFlux node that allows you to get the locomotion component that the User ([User](https://wiki.resonite.com/Type:User "Type:User")) is currently using to move. 
## Inputs
### User ([User](https://wiki.resonite.com/Type:User "Type:User"))
The user to get the current locomotion they are moving with. 
## Outputs
### * ([ILocomotionModule](https://wiki.resonite.com/Type:ILocomotionModule "Type:ILocomotionModule"))
The currently locomotion that User ([User](https://wiki.resonite.com/Type:User "Type:User")) is using to move.
