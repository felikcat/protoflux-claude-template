# ProtoFlux:CurrentCulture

> Source: https://wiki.resonite.com/ProtoFlux:CurrentCulture

Current Culture

\*

Localization

The **CurrentCulture** node outputs the local [user's](https://wiki.resonite.com/User "User") [culture](https://learn.microsoft.com/en-us/dotnet/api/system.globalization.cultureinfo?view=net-9.0).

## Outputs

### \\* ( [CultureInfo](https://wiki.resonite.com/Type:CultureInfo "Type:CultureInfo"))

The local user's current culture.
