# ProtoFlux:Constant

> Source: https://wiki.resonite.com/ProtoFlux:Constant

_due to wiki limitations, this node does not currently have an HTML rendering of the node._

Constant outputs the default value of a [Type](https://wiki.resonite.com/Type:Type "Type:Type") and is used as a base for classes that output constant values like e or Pi.

## Outputs

### \\* (Generic)

outputs the default value of the generic chosen.
