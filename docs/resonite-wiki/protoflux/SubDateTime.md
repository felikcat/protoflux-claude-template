The **Sub\_DateTime** node takes in two DateTime values and subtracts them, returning with a TimeSpan value corresponding to the duration between the two instants.

## Inputs

### A ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The first DateTime value.

### B ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The second DateTime value.

## Outputs

### \\* ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The new resulting TimeSpan value.
