# ProtoFlux:UserDownloadedBytes

> Source: https://wiki.resonite.com/ProtoFlux:UserDownloadedBytes

User Downloaded Bytes

User

\*

Info

The `User Downloaded Bytes` node returns a user's total downloaded bytes.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [ulong](https://wiki.resonite.com/Type:Ulong "Type:Ulong"))

The total downloaded bytes of the user.
