# ProtoFlux:Rad2Deg

> Source: https://wiki.resonite.com/ProtoFlux:Rad2Deg

Rad 2 Deg

Constants

The **RadToDeg** or **Rad 2 Deg** node outputs the value of 180 ÷ [π](https://en.wikipedia.org/wiki/Pi). Multiply by this to convert a value from [radians](https://en.wikipedia.org/wiki/Radian) to degrees.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Value of 180 ÷ π, approximately `57.29578`.
