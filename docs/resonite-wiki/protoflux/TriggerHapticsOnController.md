The **Trigger Haptics On Controller** node takes in a side and will send out haptic pulses to the controller on that side.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Calls an impulse to send haptic data to one side of the controllers.

### Side ( [Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality"))

The side that will trigger the haptics for the controllers.

### RelativeIntensity ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

How much this will notify the user using haptics.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Fires after sending the haptic pulse.
