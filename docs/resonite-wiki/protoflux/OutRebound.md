Ease-Out Rebound

Time

\*

ReboundAmplitude

Easing

The **Ease-Out Rebound** node takes in the length of time and amplitude, and returns the curve values.

## Inputs

### Time (Pseudo-Generic)

The length of time for the curve.

### ReboundAmplitude (Pseudo-Generic)

The height of the curve.

## Outputs

### \\* (Pseudo-Generic)

Returns the curve value.
