Half As UShort
Value
*
Binary
The `Half As UShort` node takes in a [half](https://wiki.resonite.com/Type:Half "Type:Half") and converts it into a [ushort](https://wiki.resonite.com/Type:Ushort "Type:Ushort"). 
## Inputs
### Value ([half](https://wiki.resonite.com/Type:Half "Type:Half"))
The half to convert. 
## Outputs
### * ([ushort](https://wiki.resonite.com/Type:Ushort "Type:Ushort"))
Returns a ushort value.
