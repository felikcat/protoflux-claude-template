The **Ray Plane Intersection** node takes in ray paremeters and plane parameters, and finds then returns an intersection at a point.

## Inputs

### RayOrigin ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The starting point of this ray.

### RayDirection ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The direction this ray will aim at.

### PlanePoint ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The plane position.

### PlaneNormal ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The plane normal.

## Outputs

### \\* ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the plane intersection value.
