# ProtoFlux:ActiveUserRootUser

> Source: https://wiki.resonite.com/ProtoFlux:ActiveUserRootUser

Active User Root User

UserRoot

\*

User Root

The **Active UserRoot User** node takes in a reference of a [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"), and if that UserRoot is active or being used, it returns a [User](https://wiki.resonite.com/Type:User "Type:User") reference.

## Inputs

### UserRoot ( [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"))

The user's user root.

## Outputs

### \\* ( [User](https://wiki.resonite.com/Type:User "Type:User"))

Returns the user reference.
