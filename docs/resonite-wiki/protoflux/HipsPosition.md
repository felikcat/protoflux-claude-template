# ProtoFlux:HipsPosition

> Source: https://wiki.resonite.com/ProtoFlux:HipsPosition

Hips Position

UserRoot

\*

User Root

The **Hips Position** node takes in a reference of a [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"), and if that UserRoot is active or being used, it returns the hip position value of the [User](https://wiki.resonite.com/Type:User "Type:User").

## Inputs

### UserRoot ( [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"))

The user's user root.

## Outputs

### \\* ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the hip position value.
