# ProtoFlux:UserControlMessages

> Source: https://wiki.resonite.com/ProtoFlux:UserControlMessages

User Control Messages

User

\*

Info

The `User Control Messages` node takes in a [user](https://wiki.resonite.com/Type:User "Type:User") and returns that user's control messages. If the user is [host](https://wiki.resonite.com/ProtoFlux:Host_User "ProtoFlux:Host User"), then that number is always zero (0).

## Inputs

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

The user that we want info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

Returns the control messages of that user.
