# ProtoFlux:ReplaceFirstSubstring

> Source: https://wiki.resonite.com/ProtoFlux:ReplaceFirstSubstring

Replace First Substring

Str

\*

SearchFor

ReplaceWith

StartIndex

Operators

Replace First Substring is a ProtoFlux node that replaces the first occurrence of SearchFor ( [String](https://wiki.resonite.com/Type:String "Type:String")) in Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) with ReplaceWith ( [String](https://wiki.resonite.com/Type:String "Type:String")). Will only start searching after the provided StartIndex ( [int](https://wiki.resonite.com/Type:Int "Type:Int")).

## Inputs

### Str ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to search for SearchFor ( [String](https://wiki.resonite.com/Type:String "Type:String")) in.

### SearchFor ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to be replacing in Str ( [String](https://wiki.resonite.com/Type:String "Type:String")).

### ReplaceWith ( [String](https://wiki.resonite.com/Type:String "Type:String"))

What to replace SearchFor ( [String](https://wiki.resonite.com/Type:String "Type:String")) with.

### StartIndex ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The point in the string to start searching from, rather than from the beginning.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

Returns Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) with the first SearchFor ( [String](https://wiki.resonite.com/Type:String "Type:String")) occurance replaced with ReplaceWith ( [String](https://wiki.resonite.com/Type:String "Type:String")). But only after the point StartIndex ( [int](https://wiki.resonite.com/Type:Int "Type:Int")) was searched after.
