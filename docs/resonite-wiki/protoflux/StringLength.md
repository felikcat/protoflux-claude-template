# ProtoFlux:StringLength

> Source: https://wiki.resonite.com/ProtoFlux:StringLength

String Length

A

\*

Strings

String Length is a ProtoFlux node that returns the length of A ( [String](https://wiki.resonite.com/Type:String "Type:String")) including any white spaces.

## Inputs

### A ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to check the length for.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

How long A ( [String](https://wiki.resonite.com/Type:String "Type:String")) is including any white spaces.
