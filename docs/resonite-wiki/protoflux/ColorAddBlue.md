# ProtoFlux:ColorAddBlue

> Source: https://wiki.resonite.com/ProtoFlux:ColorAddBlue

Add Blue

color

\*

Value

Channels

Add Blue is a ProtoFlux node that does a Add operation to a color on the Blue channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Add operation to the Blue channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Add operation to the Blue channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Add operation done to the Blue channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
