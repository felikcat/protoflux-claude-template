Get Parent Slot
Instance
*
Slots
Returns the parent of Instance ([Slot](https://wiki.resonite.com/Slot "Slot")). 
## Inputs
### Instance ([Slot](https://wiki.resonite.com/Slot "Slot"))
The slot to get the parent of. 
## Outputs
### * ([Slot](https://wiki.resonite.com/Slot "Slot"))
The parent of Instance ([Slot](https://wiki.resonite.com/Slot "Slot")).
