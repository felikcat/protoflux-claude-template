# ProtoFlux:InvertColor

> Source: https://wiki.resonite.com/ProtoFlux:InvertColor

Invert Color

Color

\*

Colors

The `Invert Color` node takes in a [color](https://wiki.resonite.com/Type:Color "Type:Color") and mathematically inverts it, returning a new color.

## Inputs

### Color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color we want to invert.

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the inverted color.
