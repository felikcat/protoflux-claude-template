Get Component Enabled is a ProtoFlux node Takes an [IComponent](https://wiki.resonite.com/Type:IComponent "Type:IComponent") and gets if it's enabled or not.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Impulse this to set Component ( [IComponent](https://wiki.resonite.com/Type:IComponent "Type:IComponent"))'s enabled field to State ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool")).

### Component ( [IComponent](https://wiki.resonite.com/Type:IComponent "Type:IComponent"))

The IComponent to set the enabled field for.

### State ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Whether to turn the Component ( [IComponent](https://wiki.resonite.com/Type:IComponent "Type:IComponent"))'s enabled field to true or false.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Fires after \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) is called and the Component ( [IComponent](https://wiki.resonite.com/Type:IComponent "Type:IComponent"))'s enabled field is set to State ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool")).
