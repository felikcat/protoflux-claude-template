The **Is Infinity** node takes in a value and returns if this value is `Infinity` or `-Infinity`.

## Inputs

### Value (Pseudo-Generic)

The value to check.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if the value is `Infinity` or `-Infinity`.
