Operators

The **Value Dec** node takes in a value and returns that value -1.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you need to decrement a value from an [impulse](https://wiki.resonite.com/Impulses "Impulses"), use the [Decrement](https://wiki.resonite.com/ProtoFlux:Decrement "ProtoFlux:Decrement") node instead.

This node is similar to the [Sub](https://wiki.resonite.com/ProtoFlux:Sub "ProtoFlux:Sub") node, but setting the sub node to only subtract 1.

## Inputs

### N (Pseudo-Generic)

The value to decrement by 1.

## Outputs

### \\* (Pseudo-Generic)

Returns the decremented value.
