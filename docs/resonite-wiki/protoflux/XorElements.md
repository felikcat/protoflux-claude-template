# ProtoFlux:XorElements

> Source: https://wiki.resonite.com/ProtoFlux:XorElements

Xor

V

\*

Bool Vectors

The **Xor Elements** node takes in a value ( [bool2](https://wiki.resonite.com/Type:Bool2 "Type:Bool2"), [bool3](https://wiki.resonite.com/Type:Bool3 "Type:Bool3"), or [bool4](https://wiki.resonite.com/Type:Bool4 "Type:Bool4")), and functions as a [logical xor](https://en.wikipedia.org/wiki/Exclusive_or), returning true if the inputs are set to true an odd number of times.

![](https://wiki.resonite.com/images/d/dd/InformationIcon.svg)

This node is similar to the [XOR](https://wiki.resonite.com/ProtoFlux:XOR "ProtoFlux:XOR") node or the [MultiXOR](https://wiki.resonite.com/ProtoFlux:MultiXOR "ProtoFlux:MultiXOR") node.


```
- If odd number of inputs are true, then the result will be true.
- If even number of inputs are true, then the result will be false.
```

## Inputs

### V (Pseudo-Generic)

The set of inputs to check.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns true if the inputs are set to true an odd number of times.
