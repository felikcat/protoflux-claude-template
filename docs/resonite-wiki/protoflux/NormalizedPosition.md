Normalized Position
Source
*
Media
Normalized Position is a ProtoFlux node that returns an [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable")'s playback head position from 0 to 1. See [Position ProtoFlux Node](https://wiki.resonite.com/ProtoFlux:Position "ProtoFlux:Position") for a node that returns the playback head position in seconds. 
## Inputs
### Source ([IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable"))
The clip to check the current position on. 
## Outputs
### * ([float](https://wiki.resonite.com/Type:Float "Type:Float"))
The current playback head position of the provided Source ([IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable")) from 0 to 1. See [Position ProtoFlux Node](https://wiki.resonite.com/ProtoFlux:Position "ProtoFlux:Position") for a node that returns the playback head position in seconds.
