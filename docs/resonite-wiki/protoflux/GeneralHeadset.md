The `General Headset` node takes in a [User](https://wiki.resonite.com/User "User") and returns headset information (if any) that this user has. Since it is generic, there is not much besides whether the headset is active or the battery info that can be returned from this node. 
## Inputs
### User ([User](https://wiki.resonite.com/User "User"))
The target user that is using a headset. 
## Outputs
### IsActive ([bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))
Is this headset being used right now. 
### BatteryLevel ([float](https://wiki.resonite.com/Type:Float "Type:Float"))
The current battery level of this headset. 
### IsBatteryCharging ([bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))
Is this headset's battery charging.
