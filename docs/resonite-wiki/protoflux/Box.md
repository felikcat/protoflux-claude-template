# ProtoFlux:Box

> Source: https://wiki.resonite.com/ProtoFlux:Box

Box

Box

\*

Core

The **Box<T>** node takes in a [value type](https://wiki.resonite.com/Value_type "Value type") and converts it to an [object](https://wiki.resonite.com/Type:Object "Type:Object").

## Inputs

### Box (Generic)

The value to box into an object.

## Outputs

### \\* ( [Object](https://wiki.resonite.com/Type:Object "Type:Object"))

The converted value as an object.
