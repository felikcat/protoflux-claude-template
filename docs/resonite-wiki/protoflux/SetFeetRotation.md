The **Set Feet Rotation** node takes in a user's [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot") and the rotation for the feet, then sets the rotation.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Calls an impulse to set the feet rotation.

### UserRoot ( [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"))

The user's user root.

### Rotation ( [floatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ"))

The rotation to set for the feet.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Continues flow execution from here.
