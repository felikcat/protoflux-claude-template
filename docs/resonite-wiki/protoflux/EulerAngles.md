Euler Angles
Q
*
Rotation
The **Euler Angles** node takes in a [Quaternion](https://en.wikipedia.org/wiki/Quaternion) value (a [floatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ") or [doubleQ](https://wiki.resonite.com/Type:DoubleQ "Type:DoubleQ")), and returns the converted value in [Euler Angles](https://en.wikipedia.org/wiki/Euler_angles). 
## Inputs
### Q (Pseudo-Generic)
The quaternion we want to convert. 
## Outputs
### * (Pseudo-Generic)
Returns the euler angle equivelent of this quaternion.
