# ProtoFlux:DateTimeMinute

> Source: https://wiki.resonite.com/ProtoFlux:DateTimeMinute

Minute

DateTime

\*

DateTime

The `DateTime Minute` node takes in a [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime") and returns the minute of that DateTime.

## Inputs

### DateTime ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The DateTime we are getting info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The minute portion of the DateTime.
