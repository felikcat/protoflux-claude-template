# ProtoFlux:UIntAsFloat

> Source: https://wiki.resonite.com/ProtoFlux:UIntAsFloat

UInt As Float

Value

\*

Binary

The `UInt As Float` node takes in a [uint](https://wiki.resonite.com/Type:Uint "Type:Uint") and converts it into a [float](https://wiki.resonite.com/Type:Float "Type:Float").

## Inputs

### Value ( [uint](https://wiki.resonite.com/Type:Uint "Type:Uint"))

The uint to convert.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Returns a float value.
