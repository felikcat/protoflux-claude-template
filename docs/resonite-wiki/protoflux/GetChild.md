GetChild
Instance
*
ChildIndex
Slots
Returns the children of a [ slot](https://wiki.resonite.com/Slot "Slot") from its index. 
## Inputs
### Instance ([Slot](https://wiki.resonite.com/Slot "Slot"))
The [ slot](https://wiki.resonite.com/Type:Slot "Type:Slot") you want to find the child of. 
### ChildIndex ([int](https://wiki.resonite.com/Type:Int "Type:Int"))
The index of the children you want as an [ int](https://wiki.resonite.com/Type:Int "Type:Int"). 
Starts at 0. 
## Outputs
### * ([Slot](https://wiki.resonite.com/Slot "Slot"))
The [ slot](https://wiki.resonite.com/Type:Slot "Type:Slot") found by the node from the index. If no slot is found, it will return null.
