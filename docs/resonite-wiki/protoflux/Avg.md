# ProtoFlux:Avg

> Source: https://wiki.resonite.com/ProtoFlux:Avg

Avg

A

\*

B

Math

The **Avg** node returns the [arithmetic mean](https://en.wikipedia.org/wiki/Arithmetic_mean) of two values.

## Inputs

### A (Pseudo-generic)

A value.

### B (Pseudo-generic)

A value. This value must match the type of `A`.

## Outputs

### \\* (Pseudo-generic)

The arithmetic mean of the two inputs.
