# ProtoFlux:WorldAwayKickMinutes

> Source: https://wiki.resonite.com/ProtoFlux:WorldAwayKickMinutes

World Away Kick Minutes

\*

Info

The `World Away Kick Minutes` node returns this world's "Max AFK Minutes" setting from the [Session](https://wiki.resonite.com/Session "Session") tab in the [Dash](https://wiki.resonite.com/Dash "Dash"), represented as a float value.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The world's Max AFK Minuets.
