# ProtoFlux:ColorXHue

> Source: https://wiki.resonite.com/ProtoFlux:ColorXHue

Hue

Hue

\*

TargetProfile

Colors

Color X Hue returns a color with a given hue value. This is equivalent to using [HSV To ColorX](https://wiki.resonite.com/ProtoFlux:HSV_To_ColorX "ProtoFlux:HSV To ColorX") with a value and saturation equal to 1.

## Inputs

### Hue ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The hue to get a color for. Values must be in the range \[0, 1\], values outside this range are wrapped to it. For example a value of 1.5 gives the same result as 0.5.

### TargetProfile ( [ColorProfile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile"))

The profile that will be assigned on the resulting [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX").

## Outputs

### \\* ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

The resulting color.
