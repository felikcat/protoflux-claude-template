# ProtoFlux:WorldDescription

> Source: https://wiki.resonite.com/ProtoFlux:WorldDescription

World Description

\*

Info

The `World Description` node returns this world's "Description" field from the [Session](https://wiki.resonite.com/Session "Session") tab in the [Dash](https://wiki.resonite.com/Dash "Dash").

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The world's description.
