The **On Grip Stay** node takes in a global reference of a [LocomotionGrip](https://wiki.resonite.com/index.php?title=Type:LocomotionGrip&action=edit&redlink=1 "Type:LocomotionGrip (page does not exist)"), and fires an event when the grip is happening per frame, as well as information of the locomotion module and the body node that is gripping.

## Outputs

### OnEvent ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Fires when the grip is being used by the user.

### Module ( [ILocomotionModule](https://wiki.resonite.com/Type:ILocomotionModule "Type:ILocomotionModule"))

Returns the module that is gripping.

### GrippingBodyNode ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))

Returns the body node that is gripping.

## Globals

### Grip ( [LocomotionGrip](https://wiki.resonite.com/index.php?title=Type:LocomotionGrip&action=edit&redlink=1 "Type:LocomotionGrip (page does not exist)"))

The global reference to the user's locomotion grip.
