The `Remap` node remaps a value between the defined In and Out.

The node can be either a [float](https://wiki.resonite.com/Type:Float "Type:Float") or [double](https://wiki.resonite.com/Type:Double "Type:Double") data type.

## Inputs

### InMin ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The minimum defined value for the input.

### InMax ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The maximum defined value for the input.

### OutMin ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The minimum defined value for the output.

### OutMax ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The maximum defined value for the output.

### Value ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The input value to be evaluated and remapped.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Outputs the remapped value.
