# ProtoFlux:ToolEquippingSlot

> Source: https://wiki.resonite.com/ProtoFlux:ToolEquippingSlot

Tool Equipping Slot

Tool

\*

Tools

The `Tool Equipping Side` node takes in a [Tools](https://wiki.resonite.com/Tools "Tools") reference and returns with the parent [Slot](https://wiki.resonite.com/Slot "Slot") that it is equipped on.

## Inputs

### Side ( [ITool](https://wiki.resonite.com/Type:ITool "Type:ITool"))

The tool reference.

## Outputs

### \\* ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The slot this tool is on that is equipped on the user.
