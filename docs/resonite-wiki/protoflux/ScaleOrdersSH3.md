Spherical Harmonics

The **Scale Orders SH3** node takes in a [SphericalHarmonicsL3](https://wiki.resonite.com/Type:SphericalHarmonicsL3 "Type:SphericalHarmonicsL3") along with a set of orders that refine/modify/scale it, then returns the new Spherical Harmonic of the same type.

## Inputs

### SH ( [SphericalHarmonicsL3](https://wiki.resonite.com/Type:SphericalHarmonicsL3 "Type:SphericalHarmonicsL3"))

The provided Spherical Harmonic.

### Order0 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 0).

### Order1 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 1).

### Order2 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 2).

### Order3 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 3).

## Outputs

### \\* ( [SphericalHarmonicsL3](https://wiki.resonite.com/Type:SphericalHarmonicsL3 "Type:SphericalHarmonicsL3"))

The new Spherical Harmonic.
