# ProtoFlux:WorldTimeFloat

> Source: https://wiki.resonite.com/ProtoFlux:WorldTimeFloat

T

\*

Time

The `World Time Float` node returns the total world time that this world has been open for.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

There are convenient world time nodes that make your coding life easier, preventing the need to make math nodes when these nodes would suffice:

- **[World Time 10 Float](https://wiki.resonite.com/ProtoFlux:World_Time_10_Float "ProtoFlux:World Time 10 Float")**: 10x The world time value.
- **[World Time 2 Float](https://wiki.resonite.com/ProtoFlux:World_Time_2_Float "ProtoFlux:World Time 2 Float")**: 2x The world time value.
- **[World Time Half Float](https://wiki.resonite.com/ProtoFlux:World_Time_Half_Float "ProtoFlux:World Time Half Float")**: 0.5x The world time value.
- **[World Time Tenth Float](https://wiki.resonite.com/ProtoFlux:World_Time_Tenth_Float "ProtoFlux:World Time Tenth Float")**: 0.1x The world time value.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Returns the total world time as a float.
