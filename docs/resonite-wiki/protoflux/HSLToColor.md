The `HSL To Color` node takes in a `Hue`, `Saturation`, and `Lightness`, and returns a color with those values. For clarification, here are explanations on [HSL and HSV](https://en.wikipedia.org/wiki/HSL_and_HSV). 
## Inputs
### H ([float](https://wiki.resonite.com/Type:Float "Type:Float"))
The Hue float value. 
### S ([float](https://wiki.resonite.com/Type:Float "Type:Float"))
The Saturation float value. 
### L ([float](https://wiki.resonite.com/Type:Float "Type:Float"))
The lightness float value. 
## Outputs
### * ([Color](https://wiki.resonite.com/Type:Color "Type:Color"))
Returns the color value with `Hue`, `Saturation`, and `Lightness`.
