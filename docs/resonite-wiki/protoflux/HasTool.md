The `Has Tool` node takes in a user and side of that user and returns if that user has a [tool](https://wiki.resonite.com/Tools "Tools").

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user to check.

### Side ( [Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality"))

The side to check on the user.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The result if the user has a tool on that side of them.
