# ProtoFlux:IsAppDashOpened

> Source: https://wiki.resonite.com/ProtoFlux:IsAppDashOpened

Is App Dash Opened

User

\*

Status

The `Is App Dash Opened` node returns if a user's [Dash](https://wiki.resonite.com/Dash "Dash") is opened.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The user's dash being open or not.
