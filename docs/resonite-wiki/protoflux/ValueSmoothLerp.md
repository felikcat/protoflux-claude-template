# ProtoFlux:ValueSmoothLerp

> Source: https://wiki.resonite.com/ProtoFlux:ValueSmoothLerp

Smooth Lerp

Input

\*

Speed

Interpolation

The **Value Smooth Lerp** node takes in an input value and the speed of which it lerps smoothly towards (starts fast and slows down over time), then returns the value over time.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you need the lerp to be linear, use the [Value Constant Lerp](https://wiki.resonite.com/ProtoFlux:Value_Constant_Lerp "ProtoFlux:Value Constant Lerp") node instead.


## Inputs

### Input (Pseudo-Generic)

The value we want to lerp towards.

### Speed ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

How fast the lerp should be.

## Outputs

### \\* (Pseudo-Generic)

Returns the value currently being lerped.
