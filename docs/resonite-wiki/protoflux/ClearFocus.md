# ProtoFlux:ClearFocus

Clear Focus

\*

Next

UI

The `Clear Focus` node clears the current [IFocusable](https://wiki.resonite.com/Type:IFocusable "Type:IFocusable") UI from the [user](https://wiki.resonite.com/User "User").

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Calls an impulse to clear the current focus of a UI from the user.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Fires after the focus has cleared.
