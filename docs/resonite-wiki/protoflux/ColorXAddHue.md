# ProtoFlux:ColorXAddHue

> Source: https://wiki.resonite.com/ProtoFlux:ColorXAddHue

ColorXAdd Hue

colorX

\*

Value

Channels

Add Hue is a ProtoFlux node that does a Add operation to a colorX on the Hue channel.

## Inputs

### colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

the colorX to do a Add operation to the Hue channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

the value to do a Add operation to the Hue channel on the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

returns the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")) with a Add operation done to the Hue channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
