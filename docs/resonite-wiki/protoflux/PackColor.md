Packing

The **Pack Color** node combines the specified red, green, blue and alpha channels into a [Color](https://wiki.resonite.com/Type:Color "Type:Color"). This is not to be confused with [Pack ColorX](https://wiki.resonite.com/ProtoFlux:Pack_ColorX "ProtoFlux:Pack ColorX"), which also uses color profiles, and results in a [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX") object.

## Inputs

### R ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Red Channel of resulting [Color](https://wiki.resonite.com/Type:Color "Type:Color")

### G ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Blue Channel of resulting [Color](https://wiki.resonite.com/Type:Color "Type:Color")

### B ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Green Channel of resulting [Color](https://wiki.resonite.com/Type:Color "Type:Color")

### A ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Alpha (Aka Transparency) Channel of resulting [Color](https://wiki.resonite.com/Type:Color "Type:Color")

## Outputs

### \\* ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

[Color](https://wiki.resonite.com/Type:Color "Type:Color") output which is a compilation of all this node's inputs.
