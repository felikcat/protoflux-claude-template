# ProtoFlux:BaseValue

> Source: https://wiki.resonite.com/ProtoFlux:BaseValue

Base Value <Type>

Value

\*

Quantity

The **Base Value** node takes in a value as a [quantity type](https://wiki.resonite.com/Quantity_Types "Quantity Types") and returns the corresponding base value as a [double](https://wiki.resonite.com/Type:Double "Type:Double").

To create this node, you need to specify a valid [quantity type](https://wiki.resonite.com/Quantity_Types "Quantity Types") for its generic parameter. Examples: `Mass`, `Time`, and `Voltage`.

## Inputs

### Value ( [Quantity](https://wiki.resonite.com/Quantity_Types "Quantity Types"))

The quantity value.

## Outputs

### \\* ( [double](https://wiki.resonite.com/Type:Double "Type:Double"))

The base value of the quantity, in the quantity type's base unit. (See [Quantity Types](https://wiki.resonite.com/Quantity_Types#Quantity_Types "Quantity Types") for a list of all quantity types and their corresponding base units)
