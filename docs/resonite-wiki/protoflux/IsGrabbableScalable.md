# ProtoFlux:IsGrabbableScalable

> Source: https://wiki.resonite.com/ProtoFlux:IsGrabbableScalable

Is Grabbable Scalable

Grabbable

\*

Grabbable

Is Grabbable Scalable is a ProtoFlux node that gets whether or not an [IGrabbable](https://wiki.resonite.com/Type:IGrabbable "Type:IGrabbable") is able to be scaled.

Scalable determines if the grabbable's slot can be scaled.

## Inputs

### Grabbable ( [IGrabbable](https://wiki.resonite.com/Type:IGrabbable "Type:IGrabbable"))

The grabbable to check the scalable capabilities for.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Whether Grabbable ( [IGrabbable](https://wiki.resonite.com/Type:IGrabbable "Type:IGrabbable")) is scalable or not.
