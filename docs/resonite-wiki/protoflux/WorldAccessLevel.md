# ProtoFlux:WorldAccessLevel

> Source: https://wiki.resonite.com/ProtoFlux:WorldAccessLevel

World Access Level

\*

Info

The `World Access Level` node returns this world's [Session Access Level](https://wiki.resonite.com/Type:SessionAccessLevel "Type:SessionAccessLevel") setting in the [Session](https://wiki.resonite.com/Session "Session") tab in the [Dash](https://wiki.resonite.com/Dash "Dash").

## Outputs

### \\* ( [SessionAccessLevel](https://wiki.resonite.com/Type:SessionAccessLevel "Type:SessionAccessLevel"))

The world's Session Access Level.
