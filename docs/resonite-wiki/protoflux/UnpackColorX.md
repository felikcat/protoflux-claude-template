Unpack ColorX is a ProtoFlux node that takes a [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX") and splits the color into it's RGBA channels and it's Color Profile.

## Inputs

### V ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

The color to split into it's RGBA channels and it's Color Profile.

## Outputs

### R ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Red Channel of the source V ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

### G ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Blue Channel of the source V ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

### B ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Green Channel of the source V ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

### A ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Alpha (Aka Transparency) Channel of the source V ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

### Profile ( [ColorProfile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile"))

The Color Profile of the source V ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))
