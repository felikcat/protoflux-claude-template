# ProtoFlux:Round

> Source: https://wiki.resonite.com/ProtoFlux:Round

Round

N

\*

Math

The **Round** node takes in a value and rounds it with the "Round-To-Even" rule (also known as [Banker's Rounding](https://learn.microsoft.com/en-us/openspecs/microsoft_general_purpose_programming_languages/ms-vbal/98152b5a-4d86-4acb-b875-66cb1f49433e)).

## Inputs

### N (Pseudo-Generic)

The number we want to check.

## Outputs

### \\* (Pseudo-Generic)

Returns the rounded out number.
