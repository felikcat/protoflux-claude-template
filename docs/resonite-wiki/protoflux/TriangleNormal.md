The **Triangle Normal** node takes in 3 points of a triangle and returns the normal direction value.

## Inputs

### Point0 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The first point.

### Point1 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The second point.

### Point2 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The third point.

## Outputs

### Normal ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the normal from the created triangle of points.

### IsValid ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if this is a valid triangle.
