# ProtoFlux:UserDeltaMessages

> Source: https://wiki.resonite.com/ProtoFlux:UserDeltaMessages

User Delta Messages

User

\*

Info

The `User Delta Messages` node takes in a [user](https://wiki.resonite.com/Type:User "Type:User") and returns that user's delta (or change of) time of the messages. If the user is [host](https://wiki.resonite.com/ProtoFlux:Host_User "ProtoFlux:Host User"), then that number is always zero (0).

## Inputs

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

The user that we want info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

Returns the delta messages of that user.
