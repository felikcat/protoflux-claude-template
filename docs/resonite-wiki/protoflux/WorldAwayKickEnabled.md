# ProtoFlux:WorldAwayKickEnabled

> Source: https://wiki.resonite.com/ProtoFlux:WorldAwayKickEnabled

World Away Kick Enabled

\*

Info

The `World Away Kick Enabled` node returns this world's "Auto Kick AFK users" setting in the [Session](https://wiki.resonite.com/Session "Session") tab of the [Dash](https://wiki.resonite.com/Dash "Dash").

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The world's away kick setting.
