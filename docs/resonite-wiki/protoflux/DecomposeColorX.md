The **Decompose ColorX** node takes in a [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"), then returns the [Color](https://wiki.resonite.com/Type:Color "Type:Color") and [ColorProfile](https://wiki.resonite.com/Type:Color_Profile "Type:Color Profile").

## Inputs

### Color ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

The ColorX.

## Outputs

### BaseColor ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the base color.

### Profile ( [ColorProfile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile"))

Returns the color space.
