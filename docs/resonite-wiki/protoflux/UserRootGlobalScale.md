# ProtoFlux:UserRootGlobalScale

> Source: https://wiki.resonite.com/ProtoFlux:UserRootGlobalScale

User Root Global Scale

UserRoot

\*

User Root

The **User Root Global Scale** node takes in a reference of a [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"), and if that UserRoot is active or being used, it returns the [user's](https://wiki.resonite.com/Type:User "Type:User") root scale value.

![](https://wiki.resonite.com/images/d/dd/InformationIcon.svg)

There is a set of features in [Resonite](https://wiki.resonite.com/Resonite "Resonite") that allows you to [scale and calculate scale](https://wiki.resonite.com/User_Scale "User Scale") in different ways.


## Inputs

### UserRoot ( [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"))

The user's user root.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Returns the user's root scale value.
