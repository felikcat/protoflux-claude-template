The **Avg Multi** node returns the [arithmetic mean](https://en.wikipedia.org/wiki/Arithmetic_mean) of multiple values as a more general form of the [Avg](https://wiki.resonite.com/ProtoFlux:Avg "ProtoFlux:Avg") node.

## Inputs

### Inputs (Pseudo-generic)

The values to be averaged.

## Outputs

### \\* (Pseudo-generic)

The arithmetic mean of the inputs.
