# ProtoFlux:WorldMaxUsers

> Source: https://wiki.resonite.com/ProtoFlux:WorldMaxUsers

World Max Users

\*

Info

The `World Max Users` node returns this world's "Maximum Users" setting from the [Session](https://wiki.resonite.com/Session "Session") tab in the [Dash](https://wiki.resonite.com/Dash "Dash").

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The world's max users for this world.
