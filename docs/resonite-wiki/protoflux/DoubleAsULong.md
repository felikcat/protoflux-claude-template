# ProtoFlux:DoubleAsULong

> Source: https://wiki.resonite.com/ProtoFlux:DoubleAsULong

Double As ULong

Value

\*

Binary

The `Double As ULong` node takes in a [double](https://wiki.resonite.com/Type:Double "Type:Double") and converts it into a [ulong](https://wiki.resonite.com/Type:Ulong "Type:Ulong").

## Inputs

### Value ( [double](https://wiki.resonite.com/Type:Double "Type:Double"))

The double to convert.

## Outputs

### \\* ( [ulong](https://wiki.resonite.com/Type:Ulong "Type:Ulong"))

Returns a ulong value.
