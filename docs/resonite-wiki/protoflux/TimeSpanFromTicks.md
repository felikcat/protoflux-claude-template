# ProtoFlux:TimeSpanFromTicks

> Source: https://wiki.resonite.com/ProtoFlux:TimeSpanFromTicks

Timespan

The `TimeSpan From Ticks` node takes in a [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan") and returns with the amount of ticks in that TimeSpan. There is Microsoft documentation about [Ticks](https://learn.microsoft.com/en-us/dotnet/api/system.timespan.ticks?view=net-8.0).

## Inputs

### TimeSpan ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The TimeSpan we are getting info from.

## Outputs

### \\* ( [long](https://wiki.resonite.com/Type:Long "Type:Long"))

Returns the amount of ticks in this TimeSpan.
