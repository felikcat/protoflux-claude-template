# ProtoFlux:EndsWith

> Source: https://wiki.resonite.com/ProtoFlux:EndsWith

Ends With

Str

\*

Substring

Strings

Ends With is a ProtoFlux node that checks if Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) ends with Substring ( [String](https://wiki.resonite.com/Type:String "Type:String")).

## Inputs

### Str ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to check.

### Substring ( [String](https://wiki.resonite.com/Type:String "Type:String"))

Used to check if Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) ends with this.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) ends with Substring ( [String](https://wiki.resonite.com/Type:String "Type:String")).
