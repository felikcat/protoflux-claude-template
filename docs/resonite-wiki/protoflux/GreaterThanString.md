Greater Than String is a ProtoFlux node that will say if A ([String](https://wiki.resonite.com/Type:String "Type:String")) is greater than B ([String](https://wiki.resonite.com/Type:String "Type:String")) using the same comparison method as the [Compare Strings](https://wiki.resonite.com/ProtoFlux:Compare_Strings "ProtoFlux:Compare Strings") Node. Which is based on [Comparing Strings Microsoft 4.6.2](https://learn.microsoft.com/en-us/dotnet/api/system.string.compare?view=netframework-4.6.2). 
## Inputs
### A ([String](https://wiki.resonite.com/Type:String "Type:String"))
The first string to check. 
### B ([String](https://wiki.resonite.com/Type:String "Type:String"))
The string to check is greater than A ([String](https://wiki.resonite.com/Type:String "Type:String")). 
### ComparisonType ([StringComparison](https://wiki.resonite.com/Type:StringComparison "Type:StringComparison"))
The string comparison to use. 
## Outputs
### * ([bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))
Whether A is greater than B according to [Comparing Strings Microsoft 4.6.2](https://learn.microsoft.com/en-us/dotnet/api/system.string.compare?view=netframework-4.6.2).
