The **Set Head Facing Direction** node takes in a user's [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot") and the facing direction for the head, then sets the facing direction.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Calls an impulse to set the head facing direction.

### UserRoot ( [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"))

The user's user root.

### Position ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The facing direction to set for the head.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Continues flow execution from here.
