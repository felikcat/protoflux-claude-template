Global Direction To Local
Instance
*
GlobalDirection
Direction
Global Direction To Local Is a protoflux node that allows you to convert a [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3") from Global space to Local space using the Instance ([Slot](https://wiki.resonite.com/Slot "Slot")) as a reference space. 
See [Coordinate Spaces](https://wiki.resonite.com/Coordinate_Spaces "Coordinate Spaces") For how this node can be used in bigger systems. 
## Inputs
### Instance ([Slot](https://wiki.resonite.com/Slot "Slot"))
The slot to use as reference for converting the provided Direction. 
### GlobalDirection ([float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))
The Direction to convert from Global space to Local space. 
## Outputs
### * ([float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))
The converted GlobalDirection ([float3](https://wiki.resonite.com/Type:Float3 "Type:Float3")).
