Color To HSL converts a [Color](https://wiki.resonite.com/Type:Color "Type:Color") to [HSV](https://en.wikipedia.org/wiki/HSL_and_HSV). The result is represented as 3 separate [Float](https://wiki.resonite.com/Type:Float "Type:Float") values for Hue, Saturation, and Value.

## Inputs

### Color ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to calculate the HSV equivalent of.

## Outputs

### H ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The hue of the input color.

### S ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The saturation of the input color.

### V ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value of the input color.
