# ProtoFlux:TransformRotation

Transform Rotation Is a protoflux node that allows you to convert a [FloatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ") from the provided FromSpace ( [Slot](https://wiki.resonite.com/Slot "Slot")) slot's space to ToSpace ( [Slot](https://wiki.resonite.com/Slot "Slot")) slot's space.

See [Coordinate Spaces](https://wiki.resonite.com/Coordinate_Spaces "Coordinate Spaces") For how this node can be used in bigger systems.

## Inputs

### FromSpace ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The slot that is assumed to be the space SourceRotation ( [FloatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ")) is in.

### ToSpace ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The slot to transform SourceRotation ( [FloatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ")) to.

### SourceRotation ( [FloatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ"))

The Rotation to convert from FromSpace ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s space to ToSpace ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s space.

## Outputs

### \\* ( [FloatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ"))

The converted Rotation of the source Rotation SourceRotation ( [FloatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ")).
