# ProtoFlux:WorldActiveUserCount

> Source: https://wiki.resonite.com/ProtoFlux:WorldActiveUserCount

World Active User Count

\*

Info

The `World Active User Count` node returns the amount of active users. Active users are users that are focused into this world currently.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The amount of actives users in this world.
