# ProtoFlux:IsLocalUser

> Source: https://wiki.resonite.com/ProtoFlux:IsLocalUser

Is Local User

User

\*

Status

The `Is Local User` node returns if the user is the local user.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The user being a local user or not.
