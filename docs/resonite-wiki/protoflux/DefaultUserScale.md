# ProtoFlux:DefaultUserScale

> Source: https://wiki.resonite.com/ProtoFlux:DefaultUserScale

DefaultUserScale

User

\*

Avatars

The **Default User Scale** node takes in a [User](https://wiki.resonite.com/Type:User "Type:User") or [Slot](https://wiki.resonite.com/Slot "Slot") that allows you to read a user's default [Avatar](https://wiki.resonite.com/Avatar "Avatar") scale.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you need to get the scale value to change a user's size, it is recommended to use the [User Root Global Scale](https://wiki.resonite.com/ProtoFlux:User_Root_Global_Scale "ProtoFlux:User Root Global Scale") and [Set User Scale](https://wiki.resonite.com/ProtoFlux:Set_User_Scale "ProtoFlux:Set User Scale") nodes instead.


![](https://wiki.resonite.com/images/d/dd/InformationIcon.svg)

There is a set of features in [Resonite](https://wiki.resonite.com/Resonite "Resonite") that allows you to [scale and calculate scale](https://wiki.resonite.com/User_Scale "User Scale") in different ways.


## Inputs

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

The user to check their default user scale for.

## Outputs

### \\* (Pseudo-generic)

The user's default [User Scale](https://wiki.resonite.com/User_Scale "User Scale") as a decimal number.

Examples: [Float](https://wiki.resonite.com/Type:Float "Type:Float"), [Double](https://wiki.resonite.com/Type:Double "Type:Double").
