Color To Hex Code converts a [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX") to a string [hex color code](https://en.wikipedia.org/wiki/Web_colors) such as "#FFAABC".

## Inputs

### Color ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to convert. Color components are clamped to the range 0-1.

### ShortForm ( [Bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

If true, convert to a short form such as "#FAC" instead of "#FFAACC".

### IncludeAlpha ( [Bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

If true, include the alpha channel as the last component.

### Prefix ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to prefix the color code with. Defaults to "#".

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The resulting hex code.
