# ProtoFlux:InvertedTau

> Source: https://wiki.resonite.com/ProtoFlux:InvertedTau

1/τ

Constants

The **InvertedTau** or **1/τ** node outputs the value of 1 ÷ τ which is equivalent to 1 ÷ [2π](https://en.wikipedia.org/wiki/Pi).

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value of 1 ÷ τ, approximately `0.1591549`.
