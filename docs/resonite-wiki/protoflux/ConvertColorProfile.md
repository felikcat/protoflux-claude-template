# ProtoFlux:ConvertColorProfile

> Source: https://wiki.resonite.com/ProtoFlux:ConvertColorProfile

Convert Color Profile

Color

\*

Profile

Colors

## Inputs

### Color ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

[ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX") value to change the profile for.

### Profile ( [ColorProfile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile"))

[ColorProfile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile") of resulting [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"). Can be either Linear, sRGB, sRGBA.

## Outputs

### \\* ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

Resulting changed [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX") value to the specified [ColorProfile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile").
