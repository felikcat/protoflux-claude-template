# ProtoFlux:TrimString

> Source: https://wiki.resonite.com/ProtoFlux:TrimString

Trim String

A

\*

Strings

Trim String is a ProtoFlux node that trims a string according to the microsoft implmentation [Trim String](https://learn.microsoft.com/en-us/dotnet/api/system.string.trim?view=netframework-4.6.2) which removes any white space at the beginning or end of the string including any extra special invisible control characters.

## Inputs

### A ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to trim.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

A ( [String](https://wiki.resonite.com/Type:String "Type:String")) that is trimmed according to the microsoft implmentation [Trim String](https://learn.microsoft.com/en-us/dotnet/api/system.string.trim?view=netframework-4.6.2).
