The **Allow Join** node takes in a [JoinRequestHandle](https://wiki.resonite.com/index.php?title=Type:JoinRequestHandle&action=edit&redlink=1 "Type:JoinRequestHandle (page does not exist)"), and if the [world](https://wiki.resonite.com/World "World") enabled a join verification system using the [Verify Join Request](https://wiki.resonite.com/ProtoFlux:Verify_Join_Request "ProtoFlux:Verify Join Request") node, this node will run and allow the [user](https://wiki.resonite.com/User "User") to join.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Calls an impulse to allow the user to join the world.

### Handle ( [JoinRequestHandle](https://wiki.resonite.com/index.php?title=Type:JoinRequestHandle&action=edit&redlink=1 "Type:JoinRequestHandle (page does not exist)"))

The handle for the join verification system.
