# ProtoFlux:ReplaceSubstring

> Source: https://wiki.resonite.com/ProtoFlux:ReplaceSubstring

Replace Substring

Str

\*

SearchFor

ReplaceWith

ComparisonMode

Strings

**Replace Substring** is a ProtoFlux node that replaces the all occurrences of SearchFor ( [String](https://wiki.resonite.com/Type:String "Type:String")) in Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) with ReplaceWith ( [String](https://wiki.resonite.com/Type:String "Type:String")).

## Inputs

### Str ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to search for SearchFor ( [String](https://wiki.resonite.com/Type:String "Type:String")) in.

### SearchFor ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to be replacing in Str ( [String](https://wiki.resonite.com/Type:String "Type:String")).

### ReplaceWith ( [String](https://wiki.resonite.com/Type:String "Type:String"))

What to replace SearchFor ( [String](https://wiki.resonite.com/Type:String "Type:String")) with.

### ComparasonMode ( [StringComparison](https://wiki.resonite.com/Type:StringComparison "Type:StringComparison"))

The mode used to compare the string before evaluation of the replace.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

Returns Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) with all occurrences of SearchFor ( [String](https://wiki.resonite.com/Type:String "Type:String")) replaced with ReplaceWith ( [String](https://wiki.resonite.com/Type:String "Type:String")).
