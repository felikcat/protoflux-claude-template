# ProtoFlux:IsEye

> Source: https://wiki.resonite.com/ProtoFlux:IsEye

Is Eye

Node

\*

Body Nodes

Is Eye is a ProtoFlux node that checks if the provided Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) is equal to "Left Eye" or "Right Eye".

## Inputs

### Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))

The node to check if it is an eye.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

If Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) is equal to "Left Eye" or "Right Eye".
