# ProtoFlux:ColorSetValue

> Source: https://wiki.resonite.com/ProtoFlux:ColorSetValue

Set Value

color

\*

Value

Channels

Set Value is a ProtoFlux node that does a Set operation to a color on the Value channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Set operation to the Value channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Set operation to the Value channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Set operation done to the Value channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
