# ProtoFlux:IsControl

> Source: https://wiki.resonite.com/ProtoFlux:IsControl

Is Control

Character

\*

Characters

The **Is Control** node takes in a character literal and returns if that character is a control character.

## Inputs

### Character ( [char](https://wiki.resonite.com/Type:Char "Type:Char"))

The character literal to check.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Outputs `true` if `Character` is a control character, `false` otherwise. Control characters recognized by this node are part of the [C0 and C1 sets of Unicode](https://en.wiktionary.org/wiki/Appendix:Control_characters).
