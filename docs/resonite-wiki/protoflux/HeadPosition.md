# ProtoFlux:HeadPosition

> Source: https://wiki.resonite.com/ProtoFlux:HeadPosition

Head Position

UserRoot

\*

User Root

The **Head Position** node takes in a reference of a [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"), and if that UserRoot is active or being used, it returns the head position value of the [User](https://wiki.resonite.com/Type:User "Type:User").

## Inputs

### UserRoot ( [UserRoot](https://wiki.resonite.com/Type:UserRoot "Type:UserRoot"))

The user's user root.

## Outputs

### \\* ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the head position value.
