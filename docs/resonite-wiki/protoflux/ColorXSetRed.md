# ProtoFlux:ColorXSetRed

> Source: https://wiki.resonite.com/ProtoFlux:ColorXSetRed

ColorXSet Red

colorX

\*

Value

Channels

Set Red is a ProtoFlux node that does a Set operation to a colorX on the Red channel.

## Inputs

### colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

the colorX to do a Set operation to the Red channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

the value to do a Set operation to the Red channel on the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

returns the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")) with a Set operation done to the Red channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
