Evaluate SH
SH
*
Direction
Spherical Harmonics
The **Evaluate SH** node takes in a [Spherical Harmonics](https://wiki.resonite.com/SphericalHarmonics "SphericalHarmonics") type of any LV and a direction, then returns the evaluated type value. 
There is only 4 Spherical Harmonics types for this node: 
  * [SphericalHarmonicsL1](https://wiki.resonite.com/Type:SphericalHarmonicsL1 "Type:SphericalHarmonicsL1")
  * [SphericalHarmonicsL2](https://wiki.resonite.com/Type:SphericalHarmonicsL2 "Type:SphericalHarmonicsL2")
  * [SphericalHarmonicsL3](https://wiki.resonite.com/Type:SphericalHarmonicsL3 "Type:SphericalHarmonicsL3")
  * [SphericalHarmonicsL4](https://wiki.resonite.com/Type:SphericalHarmonicsL4 "Type:SphericalHarmonicsL4")


There is currently only one known type that can spawn this node (but please fill in this list if there are more): 
  * [float](https://wiki.resonite.com/Type:Float "Type:Float")


## Inputs
### SH (Pseudo-Generic)
The provided Spherical Harmonic. 
### Direction ([float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))
The direction of the Spherical Harmonic. 
## Outputs
### * (Generic)
The returning evaluated type of this Spherical Harmonic.
