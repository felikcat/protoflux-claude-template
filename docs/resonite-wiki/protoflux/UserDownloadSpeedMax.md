# ProtoFlux:UserDownloadSpeedMax

> Source: https://wiki.resonite.com/ProtoFlux:UserDownloadSpeedMax

User Download Speed Max

User

\*

Info

The `User Download Speed Max` node returns a user's maximum download speed in this session.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The max download speed of the user in bytes.
