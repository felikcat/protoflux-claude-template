This page represents a meme/joke, and contains jokes from the community or the [Team](https://wiki.resonite.com/Team "Team").


**OwO** is a node which outputs the string "What's This?"

This node is an easter egg and a reference to a common phrase used by the [furry fandom](https://en.wikipedia.org/wiki/Furry_fandom "wikipedia:Furry fandom")[\[1\]](https://wiki.resonite.com/ProtoFlux:OwO#cite_note-1). It has no practical use in [ProtoFlux](https://wiki.resonite.com/ProtoFlux "ProtoFlux") projects.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

"What's This?"
