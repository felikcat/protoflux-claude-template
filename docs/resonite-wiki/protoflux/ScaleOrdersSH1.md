Spherical Harmonics

The **Scale Orders SH1** node takes in a [SphericalHarmonicsL1](https://wiki.resonite.com/Type:SphericalHarmonicsL1 "Type:SphericalHarmonicsL1") along with a set of orders that refine/modify/scale it, then returns the new Spherical Harmonic of the same type.

## Inputs

### SH ( [SphericalHarmonicsL1](https://wiki.resonite.com/Type:SphericalHarmonicsL1 "Type:SphericalHarmonicsL1"))

The provided Spherical Harmonic.

### Order0 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 0).

### Order1 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 1).

## Outputs

### \\* ( [SphericalHarmonicsL1](https://wiki.resonite.com/Type:SphericalHarmonicsL1 "Type:SphericalHarmonicsL1"))

The new Spherical Harmonic.
