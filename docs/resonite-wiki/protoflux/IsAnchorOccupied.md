# ProtoFlux:IsAnchorOccupied

> Source: https://wiki.resonite.com/ProtoFlux:IsAnchorOccupied

Is Anchor Occupied

Anchor

\*

Anchors

Is Anchor Occupied Is a ProtoFlux node that returns whether an anchor has a user in it or not.

## Inputs

### Anchor ( [IAvatarAnchor](https://wiki.resonite.com/Type:IAvatarAnchor "Type:IAvatarAnchor"))

The anchor to check for if a user is sitting in it.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Whether Anchor ( [IAvatarAnchor](https://wiki.resonite.com/Type:IAvatarAnchor "Type:IAvatarAnchor")) has a user in it or not.
