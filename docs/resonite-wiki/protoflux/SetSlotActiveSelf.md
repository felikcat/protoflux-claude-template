# ProtoFlux:SetSlotActiveSelf

> Source: https://wiki.resonite.com/ProtoFlux:SetSlotActiveSelf

Set Slot Active Self

\*

Next

Instance

Active

Info

Set's the active field of the provided Instance [slot](https://wiki.resonite.com/Slot "Slot") upon calling \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")). The Active ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool")) value must be provided for this node to work.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Call this to set the Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s active field to Active ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool")).

### Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))

[Slot](https://wiki.resonite.com/Slot "Slot") to set the property for.

### Active ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The value to set Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s active field to when \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) is called.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Sends an impulse after \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) is called and Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s active field was set to Active ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool")).
