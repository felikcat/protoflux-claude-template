# ProtoFlux:UserFingerPoseSource

> Source: https://wiki.resonite.com/ProtoFlux:UserFingerPoseSource

User Finger Pose Source

User

\*

Avatars

User Finger Pose Source is a ProtoFlux node that allows you to get an [IFingerPoseSource](https://wiki.resonite.com/Type:IFingerPoseSource "Type:IFingerPoseSource") for use in [Finger Pose](https://wiki.resonite.com/ProtoFlux:Finger_Pose "ProtoFlux:Finger Pose") Node. This is a pose source that represents the pose sources of both hands in one.

## Inputs

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

The user to get \* ( [IFingerPoseSource](https://wiki.resonite.com/Type:IFingerPoseSource "Type:IFingerPoseSource")) from.

## Outputs

### \\* ( [IFingerPoseSource](https://wiki.resonite.com/Type:IFingerPoseSource "Type:IFingerPoseSource"))

The IFingerPoseSource that User ( [User](https://wiki.resonite.com/Type:User "Type:User")) has.
