# ProtoFlux:Contains

> Source: https://wiki.resonite.com/ProtoFlux:Contains

Contains

Str

\*

Substring

Strings

Contains is a ProtoFlux node that returns if Substring ( [String](https://wiki.resonite.com/Type:String "Type:String")) occurs anywhere in Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) as a [bool](https://wiki.resonite.com/Type:Bool "Type:Bool").

## Inputs

### Str ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to search for Substring ( [String](https://wiki.resonite.com/Type:String "Type:String")) in.

### Substring ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to be searching for.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Whether or not Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) contains at least one occurrence of Substring ( [String](https://wiki.resonite.com/Type:String "Type:String")).
