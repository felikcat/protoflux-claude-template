The **Greater Or Equal** node takes in 2 values and returns if the first value is larger than or equal to the second value. 
![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)
For other comparison nodes: 
  * If you need to check if a value is greater than something else, use the [Greater Than](https://wiki.resonite.com/ProtoFlux:Greater_Than "ProtoFlux:Greater Than") node instead.
  * If you need to check if a value is less than something else, use the [Less Than](https://wiki.resonite.com/ProtoFlux:Less_Than "ProtoFlux:Less Than") node instead.
  * If you need to check if a value is less than or equal to something else, use the [Less Or Equal](https://wiki.resonite.com/ProtoFlux:Less_Or_Equal "ProtoFlux:Less Or Equal") node instead.


## Inputs
### A (Pseudo-Generic)
The first value. 
### B (Pseudo-Generic)
The second value. 
## Outputs
### * ([bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))
Returns if the first value is larger than or equal to the second value.
