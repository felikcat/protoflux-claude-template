Format Locale String
Locale
*
Key
Localization
The `Format Locale String` node takes in a locale asset (or repository) and a key and it returns the translated string for the [user's](https://wiki.resonite.com/User "User") [culture](https://wiki.resonite.com/Type:CultureInfo "Type:CultureInfo"). 
## Inputs
### Locale ([LocaleResource](https://wiki.resonite.com/index.php?title=Type:LocaleResource&action=edit&redlink=1 "Type:LocaleResource \(page does not exist\)"))
The list of translatable strings (the repository or collection of strings). 
### Key ([String](https://wiki.resonite.com/String "String"))
The key to search for in the locale resource. 
## Outputs
### * ([String](https://wiki.resonite.com/String "String"))
The translated result.
