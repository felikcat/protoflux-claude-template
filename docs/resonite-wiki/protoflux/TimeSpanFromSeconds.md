# ProtoFlux:TimeSpanFromSeconds

> Source: https://wiki.resonite.com/ProtoFlux:TimeSpanFromSeconds

Time Span From Seconds

Value

\*

Timespan

The `TimeSpan From Seconds` node takes in the amount of seconds and converts it into a [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan") that shows the span of time in seconds.

## Inputs

### Value ( [double](https://wiki.resonite.com/Type:Double "Type:Double"))

The amount of seconds.

## Outputs

### \\* ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

Returns the TimeSpan with the amount of seconds.
