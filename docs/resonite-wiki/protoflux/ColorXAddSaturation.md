# ProtoFlux:ColorXAddSaturation

> Source: https://wiki.resonite.com/ProtoFlux:ColorXAddSaturation

ColorXAdd Saturation

colorX

\*

Value

Channels

Add Saturation is a ProtoFlux node that does a Add operation to a colorX on the Saturation channel.

## Inputs

### colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

the colorX to do a Add operation to the Saturation channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

the value to do a Add operation to the Saturation channel on the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

returns the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")) with a Add operation done to the Saturation channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
