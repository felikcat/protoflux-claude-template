# ProtoFlux:UserReferenceAsVariable

> Source: https://wiki.resonite.com/ProtoFlux:UserReferenceAsVariable

User Ref As Variable

UserRef

\*

Core

The **User Reference As Variable** node takes in a [UserRef](https://wiki.resonite.com/Type:UserRef "Type:UserRef") and returns an [IVariable](https://wiki.resonite.com/Type:IVariable "Type:IVariable") of that [user](https://wiki.resonite.com/User "User").

## Inputs

### UserRef ( [UserRef](https://wiki.resonite.com/Type:UserRef "Type:UserRef"))

The UserRef of a user.

## Outputs

### \\* ( [IVariable](https://wiki.resonite.com/Type:IVariable "Type:IVariable"))

Returns the variable of the user.
