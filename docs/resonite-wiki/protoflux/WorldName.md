# ProtoFlux:WorldName

> Source: https://wiki.resonite.com/ProtoFlux:WorldName

World Name

\*

Info

The `World Name` node returns this world's "World Name" field from the [Session](https://wiki.resonite.com/Session "Session") tab in the [Dash](https://wiki.resonite.com/Dash "Dash").

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The world's name.
