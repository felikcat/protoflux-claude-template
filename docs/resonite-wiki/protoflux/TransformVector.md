# ProtoFlux:TransformVector

Direction

Transform Vector Is a protoflux node that allows you to convert a [Float3](https://wiki.resonite.com/Type:Float3 "Type:Float3") from the provided FromSpace ( [Slot](https://wiki.resonite.com/Slot "Slot")) slot's space to ToSpace ( [Slot](https://wiki.resonite.com/Slot "Slot")) slot's space.

See [Coordinate Spaces](https://wiki.resonite.com/Coordinate_Spaces "Coordinate Spaces") For how this node can be used in bigger systems.

## Inputs

### FromSpace ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The slot that is assumed to be the space SourceVector ( [Float3](https://wiki.resonite.com/Type:Float3 "Type:Float3")) is in.

### ToSpace ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The slot to transform SourceVector ( [Float3](https://wiki.resonite.com/Type:Float3 "Type:Float3")) to.

### SourceVector ( [Float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The Vector to convert from FromSpace ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s space to ToSpace ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s space.

## Outputs

### \\* ( [Float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The converted Vector of the source Vector SourceVector ( [Float3](https://wiki.resonite.com/Type:Float3 "Type:Float3")).
