# ProtoFlux:OnPackageImported

> Source: https://wiki.resonite.com/ProtoFlux:OnPackageImported

On Package Imported

Trigger

Events

**On Package Imported** is a ProtoFlux node that sends an impulse out of Trigger when it detects that it has been imported as a part of a [ResonitePackage](https://wiki.resonite.com/ResonitePackage "ResonitePackage").

## Outputs

### Trigger ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Sends an Impulse when the node's hierarchy has been imported from a [ResonitePackage](https://wiki.resonite.com/ResonitePackage "ResonitePackage")
