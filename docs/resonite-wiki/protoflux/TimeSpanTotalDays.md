# ProtoFlux:TimeSpanTotalDays

> Source: https://wiki.resonite.com/ProtoFlux:TimeSpanTotalDays

Days

TimeSpan

\*

Timespan

The `TimeSpan Total Days` node takes in a [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan") and returns with the amount of days in that TimeSpan, including the fractional portion of a day.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you only want the days from a TimeSpan, use [TimeSpan Days](https://wiki.resonite.com/ProtoFlux:TimeSpan_Days "ProtoFlux:TimeSpan Days") instead.


## Inputs

### TimeSpan ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The TimeSpan we are getting info from.

## Outputs

### \\* ( [double](https://wiki.resonite.com/Type:Double "Type:Double"))

Returns the amount of days in this TimeSpan as an double.
