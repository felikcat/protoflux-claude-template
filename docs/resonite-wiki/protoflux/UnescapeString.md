# ProtoFlux:UnescapeString

> Source: https://wiki.resonite.com/ProtoFlux:UnescapeString

Unescape String

String

\*

Strings

Unescape String gets rid of the '\\' generated by an [Escape String](https://wiki.resonite.com/ProtoFlux:Escape_String "ProtoFlux:Escape String") Node, which escapes strings according to the Microsoft implementation [4.6 escape string](https://learn.microsoft.com/en-us/dotnet/api/system.text.regularexpressions.regex.escape?view=netframework-4.6)

### String ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to unescape.

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The unescaped string which was escaped with '\\' according to the Microsoft implementation [4.6 escape string](https://learn.microsoft.com/en-us/dotnet/api/system.text.regularexpressions.regex.escape?view=netframework-4.6).
