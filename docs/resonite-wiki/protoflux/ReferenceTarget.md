# ProtoFlux:ReferenceTarget

> Source: https://wiki.resonite.com/ProtoFlux:ReferenceTarget

Reference Target

Reference

\*

References

The `Reference Target` node takes in an [SyncRef](https://wiki.resonite.com/Type:SyncRef%601 "Type:SyncRef`1") and returns with the [Reference Type](https://wiki.resonite.com/Reference_Type "Reference Type") that was asked for.

## Inputs

### Reference ( [SyncRef](https://wiki.resonite.com/Type:SyncRef%601 "Type:SyncRef`1"))

The reference of something.

## Outputs

### \\* (Generic)

Returns the [reference type](https://wiki.resonite.com/Reference_Type "Reference Type") associated with that [SyncRef](https://wiki.resonite.com/Type:SyncRef%601 "Type:SyncRef`1").
