# ProtoFlux:WorldRecordURL

> Source: https://wiki.resonite.com/ProtoFlux:WorldRecordURL

World Record URL

\*

Info

The `World Record URL` node returns a resrec link of the current [world](https://wiki.resonite.com/World "World"). This is very useful for [Headless](https://wiki.resonite.com/Headless "Headless") clients to run a world that they own, using the resrec from this node.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The world's record.
