# ProtoFlux:IsPlaying

> Source: https://wiki.resonite.com/ProtoFlux:IsPlaying

Is Playing

Source

\*

Media

Is Playing is a ProtoFlux node that returns if an [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable") is currently playing.

## Inputs

### Source ( [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable"))

The clip to check if it is playing

## Outputs

### \\* ( [boolean](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Whether the provided Source ( [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable")) is playing or not.
