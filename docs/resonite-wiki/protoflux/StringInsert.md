# ProtoFlux:StringInsert

> Source: https://wiki.resonite.com/ProtoFlux:StringInsert

String Insert

Str

\*

Value

StartIndex

Strings

String Insert is a ProtoFlux node that allows for inserting Value ( [String](https://wiki.resonite.com/Type:String "Type:String")) into Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) at position StartIndex ( [int](https://wiki.resonite.com/Type:Int "Type:Int")).

## Inputs

### Str ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to be inserted into.

### Value ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to insert into Str ( [String](https://wiki.resonite.com/Type:String "Type:String")).

### StartIndex ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The place to insert Value ( [String](https://wiki.resonite.com/Type:String "Type:String")) into Str ( [String](https://wiki.resonite.com/Type:String "Type:String")).

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

Returns Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) with Value ( [String](https://wiki.resonite.com/Type:String "Type:String")) inserted at position StartIndex ( [int](https://wiki.resonite.com/Type:Int "Type:Int")).
