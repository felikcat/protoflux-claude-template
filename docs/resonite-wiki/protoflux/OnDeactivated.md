# ProtoFlux:OnDeactivated

> Source: https://wiki.resonite.com/ProtoFlux:OnDeactivated

On Deactivated

OnlyHost

Trigger

Events

On Deactivated is a ProtoFlux node that sends an impulse out of Trigger whenever the node's hierarchy is disabled.

## Inputs

### OnlyHost ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Checks the event on the host rather than on everyone.

## Outputs

### Trigger ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Sends an Impulse when the node's hierarchy is disabled.
