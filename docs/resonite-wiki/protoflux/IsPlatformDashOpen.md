# ProtoFlux:IsPlatformDashOpen

> Source: https://wiki.resonite.com/ProtoFlux:IsPlatformDashOpen

Is Platform Dash Opened

User

\*

Status

The `Is Platform Dash Opened` node returns if a user's Steam overlay or Oculus overlay is opened.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The user's platform dash being open or not.
