# ProtoFlux:ComposeColorX

> Source: https://wiki.resonite.com/ProtoFlux:ComposeColorX

Compose ColorX

BaseColor

\*

Profile

Colors

The **Compose ColorX** node takes in a [Color](https://wiki.resonite.com/Type:Color "Type:Color") and [ColorProfile](https://wiki.resonite.com/Type:Color_Profile "Type:Color Profile"), then returns the new [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX").

## Inputs

### Color ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

The base color.

### Profile ( [ColorProfile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile"))

The color space.

## Outputs

### \\* ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

Returns the new ColorX.
