# ProtoFlux:WorldTimeDouble

> Source: https://wiki.resonite.com/ProtoFlux:WorldTimeDouble

T (ᵈᵒᵘᵇˡᵉ)

\*

Time

The `World Time Double` node returns the total world time that this world has been open for.

## Outputs

### \\* ( [double](https://wiki.resonite.com/Type:Double "Type:Double"))

Returns the total world time as a double.
