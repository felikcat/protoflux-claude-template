Get Tag
Instance
*
Info
The **Get Tag** node gets the [tag](https://wiki.resonite.com/Tag "Tag") value of the provided [slot](https://wiki.resonite.com/Slot "Slot"). 
## Inputs
### Instance ([Slot](https://wiki.resonite.com/Slot "Slot"))
[Slot](https://wiki.resonite.com/Slot "Slot") to check. 
## Outputs
### * ([String](https://wiki.resonite.com/Type:String "Type:String"))
Instance ([Slot](https://wiki.resonite.com/Slot "Slot"))'s Tag field.
