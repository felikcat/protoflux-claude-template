# ProtoFlux:ShiftPosition

> Source: https://wiki.resonite.com/ProtoFlux:ShiftPosition

Shift Position

\*

Next

Target

Delta

Media

The Shift Position node tells a IPlayable to move the current playback position upon impulse.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

### Target ( [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable"))

The [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable") to set the position on.

### Delta ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

Set how much the position should be moved. A positive value moves the position forward, while a negative one moves it backwards.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Sends an impulse after the initial call.
