# ProtoFlux:UserStreamMessages

> Source: https://wiki.resonite.com/ProtoFlux:UserStreamMessages

User Stream Messages

User

\*

Info

The `User Stream Messages` node takes in a [user](https://wiki.resonite.com/Type:User "Type:User") and returns that user's streamed messaged (is this upstream, downstream, or bi-directional?).

## Inputs

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

The user that we want info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

Returns the streamed messages of that user.
