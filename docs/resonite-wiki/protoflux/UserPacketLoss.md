# ProtoFlux:UserPacketLoss

> Source: https://wiki.resonite.com/ProtoFlux:UserPacketLoss

User Packet Loss

User

\*

Info

The `User Packet Loss` node takes in a [user](https://wiki.resonite.com/Type:User "Type:User") and returns that user's lost packets during this network communication, session, or protocol. If the user is [host](https://wiki.resonite.com/ProtoFlux:Host_User "ProtoFlux:Host User"), then that number is always zero (0).

## Inputs

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

The user that we want info from.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Returns the lost packets of that user.
