# ProtoFlux:ColorHue

> Source: https://wiki.resonite.com/ProtoFlux:ColorHue

Hue

Hue

\*

Colors

Color Hue returns a color with a given hue value. This is equivalent to using [HSV To Color](https://wiki.resonite.com/ProtoFlux:HSV_To_Color "ProtoFlux:HSV To Color") with a value and saturation equal to 1.

## Inputs

### Hue ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The hue to get a color for. Values must be in the range \[0, 1\], values outside this range are wrapped to it. For example a value of 1.5 gives the same result as 0.5.

## Outputs

### \\* ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

The resulting color.
