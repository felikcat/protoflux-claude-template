NOT Null
Instance
*
Operators
The **Not Null** node takes in a [reference type](https://wiki.resonite.com/Reference_Type "Reference Type") and returns if the provided reference is not null. 
![](https://wiki.resonite.com/images/d/dd/InformationIcon.svg)
[Value types](https://wiki.resonite.com/Value_Type "Value Type") cannot be null, unless using the nullable version of those types. Using value types that are [nullables](https://wiki.resonite.com/Type:Nullable%601 "Type:Nullable`1") will not work with this node however. 
![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)
If you want to check if something is null, use the [Is Null](https://wiki.resonite.com/ProtoFlux:Is_Null "ProtoFlux:Is Null") node instead. 
## Inputs
### Instance (Pseudo-Generic)
The reference to check. 
## Outputs
### * ([bool](https://wiki.resonite.com/Types:Bool "Types:Bool"))
Outputs if instance is not null.
