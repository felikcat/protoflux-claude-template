# ProtoFlux:SymmetricPowShape

> Source: https://wiki.resonite.com/ProtoFlux:SymmetricPowShape

Power

Shaping

The **Symmetric Pow Shape** node takes in a value and a power to raise this curve by, then returns the curve shape value (a curve that is customizable based on the value and power given, the higher the power, the more squished the resulting curve is).

## Inputs

### Value (Pseudo-Generic)

The value for the curve.

### Power (Pseudo-Generic)

The power to raise this by, and will control how squished this curve is.

## Outputs

### \\* (Pseudo-Generic)

Returns the curve value.
