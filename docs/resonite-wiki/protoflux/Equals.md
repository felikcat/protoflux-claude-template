# ProtoFlux:Equals

> Source: https://wiki.resonite.com/ProtoFlux:Equals

==

A

\*

B

Operators

The **Equals** node takes in 2 inputs and returns if the values match.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you need to check if values are not equal to each other, use the [Not Equals](https://wiki.resonite.com/ProtoFlux:Not_Equals "ProtoFlux:Not Equals") node.


## Inputs

### A (Pseudo-Generic)

The first value to check.

### B (Pseudo-Generic)

The second value to check.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if the 2 input values are equal.

## Bugs & Issues

- This node does not correctly return the correct value when using and comparing `NaN` values against each other: [#1046](https://github.com/Yellow-Dog-Man/Resonite-Issues/issues/1046) & [#1244](https://github.com/Yellow-Dog-Man/Resonite-Issues/issues/1244).
