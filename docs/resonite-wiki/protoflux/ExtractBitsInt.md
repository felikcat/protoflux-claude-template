The `Extract Bits Int` node takes in an [int](https://wiki.resonite.com/Type:Int "Type:Int") value and returns 32 bits of [boolean](https://wiki.resonite.com/Type:Bool "Type:Bool") data. Used for binary math and data transmission. 
## Inputs
### Integer ([int](https://wiki.resonite.com/Type:Int "Type:Int"))
The int we want to split into boolean values. 
## Outputs
### Bit# ([bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))
The bit from this int value.
