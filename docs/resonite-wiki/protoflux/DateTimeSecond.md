# ProtoFlux:DateTimeSecond

> Source: https://wiki.resonite.com/ProtoFlux:DateTimeSecond

Second

DateTime

\*

DateTime

The `DateTime Second` node takes in a [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime") and returns the second of that DateTime.

## Inputs

### DateTime ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The DateTime we are getting info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The second portion of the DateTime.
