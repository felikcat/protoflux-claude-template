Tool Equipping Side

Tool

\*

Tools

The `Tool Equipping Side` node takes in a [Tools](https://wiki.resonite.com/Tools "Tools") reference and returns with the side it is equipped on.

## Inputs

### Side ( [ITool](https://wiki.resonite.com/Type:ITool "Type:ITool"))

The tool reference.

## Outputs

### \\* ( [Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality"))

The side this tool is on that is equipped on the user.
