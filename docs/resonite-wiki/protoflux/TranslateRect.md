The **Translate Rect** node takes in a [rect](https://wiki.resonite.com/Type:Rect "Type:Rect") [value](https://wiki.resonite.com/Value_Type "Value Type") and an offset ( [float2](https://wiki.resonite.com/Type:Float2 "Type:Float2")) which corresponds with the `X` and `Y` position of the entire rect, and returns a rect that has been translated to its new location.

## Inputs

### Rect ( [Rect](https://wiki.resonite.com/Type:Rect "Type:Rect"))

The rect to offset.

### Offset ( [float2](https://wiki.resonite.com/Type:Float2 "Type:Float2"))

The `X` and `Y` position to shift the rect in a direction.

## Outputs

### \\* ( [Rect](https://wiki.resonite.com/Type:Rect "Type:Rect"))

Returns the translated rect.
