# ProtoFlux:SmoothStep

> Source: https://wiki.resonite.com/ProtoFlux:SmoothStep

Smooth Step

Value

\*

Shaping

The **Smooth Step** node takes in a value, then returns the smoothed shape value (a curve that starts and ends fast and slows down in the middle).

## Inputs

### Value (Pseudo-Generic)

The value for the curve.

## Outputs

### \\* (Pseudo-Generic)

Returns the curve value.
