# ProtoFlux:EmptyString

> Source: https://wiki.resonite.com/ProtoFlux:EmptyString

Empty String

\*

Strings

Empty String is a ProtoFlux node that returns a constant that is a String with no content but is not null.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

Gives a String with no content but is not null.
