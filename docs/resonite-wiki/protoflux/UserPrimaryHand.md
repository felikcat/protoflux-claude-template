# ProtoFlux:UserPrimaryHand

> Source: https://wiki.resonite.com/ProtoFlux:UserPrimaryHand

User Primary Hand

User

\*

Info

The `User Primary Hand` node returns the user's primary/preferred hand as a [Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality") value. This gets set when the user makes an account for the first time and gets asked which hand is their primary hand.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality"))

The primary hand of the user.
