# ProtoFlux:ComposeRotation

> Source: https://wiki.resonite.com/ProtoFlux:ComposeRotation

ComposeRotation

Rotation

\*

Matrix

The **Compose Rotation** node takes in either a [floatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ") or a [doubleQ](https://wiki.resonite.com/Type:DoubleQ "Type:DoubleQ") and composes it into a matrix table value.

## Inputs

### Rotation (Pseudo-Generic)

The rotation value.

## Outputs

### \\* (Pseudo-Generic)

Returns the composed matrix.
