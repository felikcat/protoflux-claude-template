# ProtoFlux:IsLetter

> Source: https://wiki.resonite.com/ProtoFlux:IsLetter

Is Letter

Character

\*

Characters

The **Is Letter** node takes in a character literal and returns if that character is a letter.

## Inputs

### Character ( [char](https://wiki.resonite.com/Type:Char "Type:Char"))

The character literal to check.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns `true` if `Character` is a letter, `false` otherwise. Letters recognized by this node are part of the Unicode delegations [`Lu`](https://www.compart.com/en/unicode/category/Lu), [`Ll`](https://www.compart.com/en/unicode/category/Ll), [`Lt`](https://www.compart.com/en/unicode/category/Lt), [`Lm`](https://www.compart.com/en/unicode/category/Lm), or [`Lo`](https://www.compart.com/en/unicode/category/Lo).
