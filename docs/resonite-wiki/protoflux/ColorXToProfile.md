# ProtoFlux:ColorXToProfile

> Source: https://wiki.resonite.com/ProtoFlux:ColorXToProfile

\*

Profile

Colors

The **ColorX To Profile** node takes in a [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX") and the target [ColorProfile](https://wiki.resonite.com/Type:Color_Profile "Type:Color Profile"), then returns the new profiled [Color](https://wiki.resonite.com/Type:Color "Type:Color").

## Inputs

### Color ( [ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

The ColorX to profile.

### Profile ( [ColorProfile](https://wiki.resonite.com/Type:ColorProfile "Type:ColorProfile"))

The target color space used to profile a new color.

## Outputs

### \\* ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the new profiled color.
