# ProtoFlux:ColorAddRed

> Source: https://wiki.resonite.com/ProtoFlux:ColorAddRed

Add Red

color

\*

Value

Channels

Add Red is a ProtoFlux node that does a Add operation to a color on the Red channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Add operation to the Red channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Add operation to the Red channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Add operation done to the Red channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
