Spherical Harmonics

The **Scale Orders SH4** node takes in a [SphericalHarmonicsL4](https://wiki.resonite.com/Type:SphericalHarmonicsL4 "Type:SphericalHarmonicsL4") along with a set of orders that refine/modify/scale it, then returns the new Spherical Harmonic of the same type.

## Inputs

### SH ( [SphericalHarmonicsL4](https://wiki.resonite.com/Type:SphericalHarmonicsL4 "Type:SphericalHarmonicsL4"))

The provided Spherical Harmonic.

### Order0 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 0).

### Order1 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 1).

### Order2 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 2).

### Order3 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 3).

### Order4 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 4).

## Outputs

### \\* ( [SphericalHarmonicsL4](https://wiki.resonite.com/Type:SphericalHarmonicsL4 "Type:SphericalHarmonicsL4"))

The new Spherical Harmonic.
