# ProtoFlux:PackNullable

> Source: https://wiki.resonite.com/ProtoFlux:PackNullable

Pack Nullable

Value

\*

HasValue

Packing

The **Pack Nullable** node takes in a [value](https://wiki.resonite.com/Value_Type "Value Type") and an input if this value is considered as having a value. This returns a [nullable](https://wiki.resonite.com/Type:Nullable%601 "Type:Nullable`1") version of this value.

## Inputs

### Value (Pseudo-Generic)

The value itself.

### HasValue ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Sets the state of having a value.

## Outputs

### \\* (Pseudo-Generic)

Returns the nullable version of this value.
