# ProtoFlux:Sqrt

> Source: https://wiki.resonite.com/ProtoFlux:Sqrt

Sqrt

N

\*

Math

The **Sqrt** node returns the [positive](https://simple.wikipedia.org/wiki/Positive_number), [real](https://en.wikipedia.org/wiki/Real_number) [square root](https://en.wikipedia.org/wiki/Square_root) of the input.

## Inputs

### N (Pseudo-generic)

The number to square root.

## Outputs

### \\* (Pseudo-generic)

The positive, real square root of `N`. If `N` is negative or `NaN`, `NaN` is returned. If `N` is `Infinity`, `Infinity` is returned. For types with multiple fields, each individual field is square rooted.
