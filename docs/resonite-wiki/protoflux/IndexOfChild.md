# ProtoFlux:IndexOfChild

> Source: https://wiki.resonite.com/ProtoFlux:IndexOfChild

Index Of Child

Instance

\*

Slots

Returns the offset of the provided Instance ( [Slot](https://wiki.resonite.com/Slot "Slot")) in the inspector compared to others that have the same immediate parent.

## Inputs

### Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The slot to read the child index of.

## Outputs

### \\* ( [Int](https://wiki.resonite.com/Types:Int "Types:Int"))

The child index of the provided Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))
