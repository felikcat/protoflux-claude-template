# ProtoFlux:EarsScale

> Source: https://wiki.resonite.com/ProtoFlux:EarsScale

Ears Scale

\*

Local Output

The **Ears Scale** node returns the local user's audio output (hearing audio) scale.

The [Camera](https://wiki.resonite.com/Camera "Camera") has the ability to override this scale by using the `Audio from camera viewpoint` setting, and if true, will return this new scale value, otherwise it will just use the user's head proxy's scale.

## Outputs

### \\* ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the local user's audio output (hearing audio) scale.
