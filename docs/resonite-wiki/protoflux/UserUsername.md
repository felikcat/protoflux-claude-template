This node returns a user's username from a [user object](https://wiki.resonite.com/Type:User "Type:User").

## Inputs

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

User you want the username of.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The username of the user.
