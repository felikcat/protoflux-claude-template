# ProtoFlux:IsLooped

> Source: https://wiki.resonite.com/ProtoFlux:IsLooped

Is Looped

Source

\*

Media

Is Looped is a ProtoFlux node that returns if an [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable") is currently set to automatic looping.

## Inputs

### Source ( [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable"))

The clip to check if it is set to automatic looping

## Outputs

### \\* ( [boolean](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Whether the provided Source ( [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable")) is set to automatic looping.
