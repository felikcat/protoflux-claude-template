# ProtoFlux:ToUTF16

> Source: https://wiki.resonite.com/ProtoFlux:ToUTF16

To UTF16

Character

\*

Characters

The **To UTF16** node returns the integer codepoint of the given UTF-16 character.

## Inputs

### Character ( [char](https://wiki.resonite.com/Type:Char "Type:Char"))

The character to retrieve the codepoint of.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The integer codepoint of the given character.
