# ProtoFlux:DateTimeHour

> Source: https://wiki.resonite.com/ProtoFlux:DateTimeHour

Hour

DateTime

\*

DateTime

The `DateTime Hour` node takes in a [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime") and returns the hour of that DateTime. This returns the hour as if it were a 24-hour time (if the hour is 3pm, it returns 15).

## Inputs

### DateTime ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The DateTime we are getting info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The hour portion of the DateTime.
