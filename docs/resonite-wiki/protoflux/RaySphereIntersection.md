The **Ray Sphere Intersection** node takes in ray paremeters and sphere parameters, and finds then returns an intersection at a point as well as if it intersected at all.

## Inputs

### Center ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The center of the sphere.

### Radius ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The radius of the sphere.

### RayOrigin ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The starting point of this ray.

### RayDirection ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The direction this ray will aim at.

## Outputs

### Point ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the sphere intersection value.

### IsIntersecting ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if this ray is intersecting this sphere.
