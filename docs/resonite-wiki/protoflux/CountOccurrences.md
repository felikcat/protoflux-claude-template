# ProtoFlux:CountOccurrences

> Source: https://wiki.resonite.com/ProtoFlux:CountOccurrences

Count Occurrences

Str

\*

Search

ComparisonMode

Strings

Contains is a ProtoFlux node that returns the amount of times Search ( [String](https://wiki.resonite.com/Type:String "Type:String")) occurs in Str ( [String](https://wiki.resonite.com/Type:String "Type:String")) as a [int](https://wiki.resonite.com/Type:Int "Type:Int").

## Inputs

### Str ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to search for Search ( [String](https://wiki.resonite.com/Type:String "Type:String")) in.

### Search ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The string to be searching for.

### ComparisonMode ( [StringComparison](https://wiki.resonite.com/Type:StringComparison "Type:StringComparison"))

The search method to use when searching.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The amount of times Search ( [String](https://wiki.resonite.com/Type:String "Type:String")) occurs in Str ( [String](https://wiki.resonite.com/Type:String "Type:String")).
