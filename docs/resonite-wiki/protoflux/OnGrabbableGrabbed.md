On Grabbable Grabbed is a ProtoFlux node that monitors an [IGrabbable](https://wiki.resonite.com/Type:IGrabbable "Type:IGrabbable") for when someone grabs it and sends an [Impulse](https://wiki.resonite.com/Impulses "Impulses") that runs on the grabbing user's client.

## Outputs

### OnGrabbed ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Sends an impulse when someone starts grabbing Grabbable ( [IGrabbable](https://wiki.resonite.com/Type:IGrabbable "Type:IGrabbable")).

## Globals

### Grabbable ( [IGrabbable](https://wiki.resonite.com/Type:IGrabbable "Type:IGrabbable"))

The [IGrabbable](https://wiki.resonite.com/Type:IGrabbable "Type:IGrabbable") to monitor for when someone starts grabbing it.
