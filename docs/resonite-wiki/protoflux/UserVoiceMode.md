# ProtoFlux:UserVoiceMode

> Source: https://wiki.resonite.com/ProtoFlux:UserVoiceMode

User Voice Mode

User

\*

Info

The `User Voice Mode` node returns a user's current voice mode.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [VoiceMode](https://wiki.resonite.com/Type:VoiceMode "Type:VoiceMode"))

The Voice Mode of the user.
