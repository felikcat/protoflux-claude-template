# ProtoFlux:InvertedHalfPi

> Source: https://wiki.resonite.com/ProtoFlux:InvertedHalfPi

1/(π/2)

Constants

The **HalfPi** or **1/(π/2)** node outputs the value of 1 ÷ ( [π](https://en.wikipedia.org/wiki/Pi) ÷ 2).

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value of 1 ÷ (π ÷ 2), approximately `0.6366197`.
