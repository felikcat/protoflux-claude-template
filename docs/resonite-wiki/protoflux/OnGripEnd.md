The **On Grip End** node takes in a global reference of a [LocomotionGrip](https://wiki.resonite.com/index.php?title=Type:LocomotionGrip&action=edit&redlink=1 "Type:LocomotionGrip (page does not exist)"), and fires an event when the grip has been released, as well as information of the locomotion module and the body node that was gripping.

## Outputs

### OnEvent ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Fires when the grip is released by the user.

### Module ( [ILocomotionModule](https://wiki.resonite.com/Type:ILocomotionModule "Type:ILocomotionModule"))

Returns the module that was gripping.

### GrippingBodyNode ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))

Returns the body node that was gripping.

## Globals

### Grip ( [LocomotionGrip](https://wiki.resonite.com/index.php?title=Type:LocomotionGrip&action=edit&redlink=1 "Type:LocomotionGrip (page does not exist)"))

The global reference to the user's locomotion grip.
