# ProtoFlux:UserRootSlot

> Source: https://wiki.resonite.com/ProtoFlux:UserRootSlot

User Root Slot

User

\*

Users

The **User Root Slot** node returns a [user's](https://wiki.resonite.com/User "User") root [slot](https://wiki.resonite.com/Slot "Slot"). Slots found by this node will have the [UserRoot](https://wiki.resonite.com/Component:UserRoot "Component:UserRoot") component.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting the root slot from.

## Outputs

### \\* ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The root slot of the user.
