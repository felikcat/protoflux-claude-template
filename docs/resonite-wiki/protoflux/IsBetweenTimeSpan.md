# ProtoFlux:IsBetweenTimeSpan

The `Is Between TimeSpan` node takes in a TimeSpan value, along with a minimum and maximum TimeSpan values, then returns a result if the first TimeSpan provided is actually between the min and max.

## Inputs

### Value ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The TimeSpan we want to check.

### Min ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The minimum this TimeSpan can be and still be true (inclusive).

### Max ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The maximum this TimeSpan can be and still be true (inclusive).

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The result if the provided TimeSpan value is in between the min and max values.
