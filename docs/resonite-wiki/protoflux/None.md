None
V
*
Bool Vectors
The **None** node takes in a value ([bool2](https://wiki.resonite.com/Type:Bool2 "Type:Bool2"), [bool3](https://wiki.resonite.com/Type:Bool3 "Type:Bool3"), or [bool4](https://wiki.resonite.com/Type:Bool4 "Type:Bool4")), and returns true if none of the inputs are true. 
## Inputs
### V (Pseudo-Generic)
The set of inputs to check. 
## Outputs
### * ([bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))
Returns true if none of the inputs are true.
