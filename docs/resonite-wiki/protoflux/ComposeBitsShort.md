The `Compose Bits Short` node takes in 16 bits of [boolean](https://wiki.resonite.com/Type:Bool "Type:Bool") data and returns a [short](https://wiki.resonite.com/Type:Short "Type:Short") result. Used for binary math and data transmission.

## Inputs

### Bit\# ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The bit for this short value.

## Outputs

### \\* ( [short](https://wiki.resonite.com/Type:Short "Type:Short"))

Returns the combined bits of this short.
