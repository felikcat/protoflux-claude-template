# ProtoFlux:ColorMulAlpha

> Source: https://wiki.resonite.com/ProtoFlux:ColorMulAlpha

Mul Alpha

color

\*

Value

Channels

Mul Alpha is a ProtoFlux node that does a Mul operation to a color on the Alpha channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Mul operation to the Alpha channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Mul operation to the Alpha channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Mul operation done to the Alpha channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
