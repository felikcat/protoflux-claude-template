The `Defocus Focusable` node clears the target [IFocusable](https://wiki.resonite.com/Type:IFocusable "Type:IFocusable") UI from the [user](https://wiki.resonite.com/User "User").

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Calls an impulse to clear the target focus of a UI from the user.

### Target ( [IFocusable](https://wiki.resonite.com/Type:IFocusable "Type:IFocusable"))

The target IFocusable reference to defocus from.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Fires after the focus has been defocused.
