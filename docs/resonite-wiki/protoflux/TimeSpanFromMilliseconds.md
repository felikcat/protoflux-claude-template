# ProtoFlux:TimeSpanFromMilliseconds

> Source: https://wiki.resonite.com/ProtoFlux:TimeSpanFromMilliseconds

Time Span From Milliseconds

Value

\*

Timespan

The `TimeSpan From Milliseconds` node takes in the amount of milliseconds and converts it into a [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan") that shows the span of time in milliseconds. When using a [display](https://wiki.resonite.com/ProtoFlux:Value_Display "ProtoFlux:Value Display") node, the TimeSpan will show the extra information.

## Inputs

### Value ( [double](https://wiki.resonite.com/Type:Double "Type:Double"))

The amount of milliseconds.

## Outputs

### \\* ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

Returns the TimeSpan with the amount of milliseconds.
