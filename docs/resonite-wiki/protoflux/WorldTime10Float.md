# ProtoFlux:WorldTime10Float

> Source: https://wiki.resonite.com/ProtoFlux:WorldTime10Float

T\*10

\*

Time

The `World Time 10 Float` node returns the total world time that this world has been open for multiplied by 10.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you want to use the normal world time value, use [World Time Float](https://wiki.resonite.com/ProtoFlux:World_Time_Float "ProtoFlux:World Time Float") instead.


## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Returns the total world time as a float multiplied by 10.
