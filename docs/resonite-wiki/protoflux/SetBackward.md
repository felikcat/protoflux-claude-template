Set Backward Is a protoflux node that allows you to set the Backward direction of the provided Instance ( [Slot](https://wiki.resonite.com/Slot "Slot")) upon calling \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")).

See [Coordinate Spaces](https://wiki.resonite.com/Coordinate_Spaces "Coordinate Spaces") For how this node can be used in bigger systems.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Tell this node to set the Backward of the provided Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))

### Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))

The slot to set the Backward of.

### Backward ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The value to set the provided Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s Backward to.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Sends an impulse after \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) was called and the Backward of the provided Instance ( [Slot](https://wiki.resonite.com/Slot "Slot")) was set to Backward ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3")).
