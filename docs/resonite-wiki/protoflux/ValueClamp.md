The **ValueClamp** node will force a value to stay within the defined Min and Max.

![](https://wiki.resonite.com/images/f/fe/WarningIcon.svg)

There is currently an [issue (#1530)](https://github.com/Yellow-Dog-Man/Resonite-Issues/issues/1530) with this node (specifically inclusive vs exclusive).


## Inputs

### Value (Pseudo-generic)

The value to clamp.

### Min (Pseudo-generic)

The forced minimum value for the input.

### Max (Pseudo-generic)

The forced maximum value for the input.

## Outputs

### \\* (Pseudo-generic)

Returns the result.
