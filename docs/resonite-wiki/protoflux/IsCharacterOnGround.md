# ProtoFlux:IsCharacterOnGround

> Source: https://wiki.resonite.com/ProtoFlux:IsCharacterOnGround

Is Character On Ground

Character

\*

Physics

The `Is Character On Ground` node takes in a [character controller](https://wiki.resonite.com/Component:CharacterController "Component:CharacterController") reference and returns if that character controller is touching the ground and has a simulating user.

## Inputs

### Character ( [Character Controller](https://wiki.resonite.com/Component:CharacterController "Component:CharacterController"))

The character controller reference.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if this character controller is touching the ground or not.
