The **Conditional** node changes its output depending on the input `Condition`.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you want something similar to this but with [impulses](https://wiki.resonite.com/Impulses "Impulses"), use the [If](https://wiki.resonite.com/ProtoFlux:If "ProtoFlux:If") node instead.

If you want to mask different parts of a [value](https://wiki.resonite.com/Value_Type "Value Type"), use the [Mask](https://wiki.resonite.com/ProtoFlux:Mask "ProtoFlux:Mask") node instead.

## Inputs

### OnTrue (Pseudo-Generic)

The value to output when the Condition is true.

### OnFalse (Pseudo-Generic)

The value to output when the Condition is false.

### Condition ( [Bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

A boolean to decide which input value to output.

## Outputs

### \\* (Pseudo-Generic)

Returns the result.
