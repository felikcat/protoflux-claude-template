# ProtoFlux:ShiftEnum

> Source: https://wiki.resonite.com/ProtoFlux:ShiftEnum

Shift Enum < T >

Value

\*

Delta

Enums

See [Enums](https://wiki.resonite.com/Category:Enums "Category:Enums") for a more in depth explaination on what Enums are.

Shift Enum is a [ProtoFlux](https://wiki.resonite.com/ProtoFlux "ProtoFlux") node that when provided an [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") will give an enum with the provided Delta ( [Int](https://wiki.resonite.com/Types:Int "Types:Int")) offset from said enum numerically according to the [Enums](https://wiki.resonite.com/Category:Enums "Category:Enums") page. "\*" and "Value" will never be different enum types.

## Inputs

### Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic)

The Enum to start at.

### Delta ( [Int](https://wiki.resonite.com/Types:Int "Types:Int"))

The offset from Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic) to use for the returned \* ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic).

## Outputs

### \\* ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic)

The Enum that is Delta ( [Int](https://wiki.resonite.com/Types:Int "Types:Int")) offset from Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic) numerically.
This node will wrap around forwards and backwards for the output.
