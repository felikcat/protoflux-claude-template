The `Compose Bits Uint` node takes in 32 bits of [boolean](https://wiki.resonite.com/Type:Bool "Type:Bool") data and returns a [uint](https://wiki.resonite.com/Type:Uint "Type:Uint") result. Used for binary math and data transmission.

## Inputs

### Bit\# ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The bit for this uint value.

## Outputs

### \\* ( [uint](https://wiki.resonite.com/Type:Uint "Type:Uint"))

Returns the combined bits of this uint.
