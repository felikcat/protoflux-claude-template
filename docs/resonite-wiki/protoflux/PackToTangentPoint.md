# ProtoFlux:PackToTangentPoint

> Source: https://wiki.resonite.com/ProtoFlux:PackToTangentPoint

Pack Tangent Point

Value

\*

Tangent

Trigonometry

The **Pack To Tangent Point** node takes in a value and a tangent and returns a tangent point of that value type (making a unique type that will be used for the [Bezier Curve](https://wiki.resonite.com/ProtoFlux:Bezier_Curve "ProtoFlux:Bezier Curve") and the [Cubic Lerp](https://wiki.resonite.com/ProtoFlux:Cubic_Lerp "ProtoFlux:Cubic Lerp") nodes).

## Inputs

### Value (Pseudo-Generic)

The value for this tangent point.

### Tangent (Pseudo-Generic)

The tangent itself.

## Outputs

### \\* (Pseudo-Generic)

Returns a tangent point to be used by other ProtoFlux nodes.
