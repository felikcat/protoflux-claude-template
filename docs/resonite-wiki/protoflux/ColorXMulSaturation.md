# ProtoFlux:ColorXMulSaturation

ColorXMul Saturation

colorX

Channels

Mul Saturation is a ProtoFlux node that does a Mul operation to a colorX on the Saturation channel.

## Inputs

### colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

the colorX to do a Mul operation to the Saturation channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

the value to do a Mul operation to the Saturation channel on the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

## Outputs

### \\* ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX"))

returns the provided colorX ( [colorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")) with a Mul operation done to the Saturation channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
