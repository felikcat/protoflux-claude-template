The `Extract Bits Long` node takes in a [long](https://wiki.resonite.com/Type:Long "Type:Long") value and returns 64 bits of [boolean](https://wiki.resonite.com/Type:Bool "Type:Bool") data. Used for binary math and data transmission. 
![](https://wiki.resonite.com/images/d/dd/InformationIcon.svg)
This node is commonly used with reading from a [Cloud Variable](https://wiki.resonite.com/Cloud_Variables "Cloud Variables") storing a lot of data. 
## Inputs
### Integer ([long](https://wiki.resonite.com/Type:Long "Type:Long"))
The long we want to split into boolean values. 
## Outputs
### Bit# ([bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))
The bit from this long value.
