# ProtoFlux:Cos

> Source: https://wiki.resonite.com/ProtoFlux:Cos

Cos

N

\*

Trigonometry

The **Cos** node returns the [Cosine](https://en.wikipedia.org/wiki/Sine_and_cosine%7C) of the input value.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

Input values are treated as [radians](https://en.wikipedia.org/wiki/Radian). If you want to use values in degrees, first multiply by the [Deg 2 Rad](https://wiki.resonite.com/ProtoFlux:Deg_2_Rad "ProtoFlux:Deg 2 Rad") node.


This node has the following variants:

## Inputs

### N (Pseudo-Generic)

Input value.

## Outputs

### \\* (Pseudo-Generic)

Cosine of N.
