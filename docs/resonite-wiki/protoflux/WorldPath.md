# ProtoFlux:WorldPath

> Source: https://wiki.resonite.com/ProtoFlux:WorldPath

World Path

\*

Info

The `World Path` node returns a [String](https://wiki.resonite.com/Type:String "Type:String") that is the [world path](https://wiki.resonite.com/World_Path "World Path") for this world.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The world's path.
