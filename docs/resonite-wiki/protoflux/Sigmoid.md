# ProtoFlux:Sigmoid

> Source: https://wiki.resonite.com/ProtoFlux:Sigmoid

Sigmoid

X

\*

E

Math

The `Sigmoid` node remaps a value between 0 and 1, based on the [sigmoid curve](https://en.wikipedia.org/wiki/Sigmoid_function).

## Inputs

### X ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to evaluate.

### E ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Euler's number

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Returns the result.
