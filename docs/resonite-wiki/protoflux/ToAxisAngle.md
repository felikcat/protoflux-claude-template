# ProtoFlux:ToAxisAngle

> Source: https://wiki.resonite.com/ProtoFlux:ToAxisAngle

To Axis Angle

Q

Axis

Angle

Rotation

The **To Axis Angle** node takes in a [Quaternion](https://en.wikipedia.org/wiki/Quaternion) and returns an axis being rotating and the angle of how much it is rotating by.

## Inputs

### Q (Pseudo-Generic)

The quaternion value.

## Outputs

### Axis (Pseudo-Generic)

Returns the axis.

### Angle (Pseudo-Generic)

Returns the the amount rotated.
