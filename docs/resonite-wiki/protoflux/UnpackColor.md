Unpack Color is a ProtoFlux node that takes a [Color](https://wiki.resonite.com/Type:Color "Type:Color") and splits the color into it's RGBA channels.

## Inputs

### V ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to split into it's RGBA channels.

## Outputs

### R ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Red Channel of the source V ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

### G ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Blue Channel of the source V ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

### B ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Green Channel of the source V ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))

### A ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Alpha (Aka Transparency) Channel of the source V ( [Color](https://wiki.resonite.com/Type:Color "Type:Color"))
