# ProtoFlux:DecomposedPosition

> Source: https://wiki.resonite.com/ProtoFlux:DecomposedPosition

DecomposedPosition

A

\*

Matrix

The **Decomposed Position** node takes in a matrix value and decomposes it into a position value.

## Inputs

### A (Pseudo-Generic)

The matrix value.

## Outputs

### \\* (Pseudo-Generic)

Returns the decomposed position value.
