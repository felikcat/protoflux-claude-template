Normalize
A
*
Vectors
The **Normalized** node takes in a vector value and returns that same vector but to the size of 1. 
## Inputs
### A (Pseudo-Generic)
The vector to normalize. 
## Outputs
### * (Pseudo-Generic)
Returns the normalized vector.
