The **Seconds Timer** node sends a pulse from the `UpdatingUser` at the specified interval of seconds.

## Inputs

### Interval ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The interval in seconds to send pulses at.

## Outputs

### OnUpdate ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Sends a pulse at every `Interval` seconds from the `UpdatingUser`.

This input can be [driven](https://wiki.resonite.com/Drives "Drives") by a [Local User](https://wiki.resonite.com/ProtoFlux:Local_User "ProtoFlux:Local User") node to generate impulses for every user.

## Globals

### UpdatingUser ( [User](https://wiki.resonite.com/Type:User "Type:User"))

The user that the `OnUpdate` pulse will be generated from and owned by.

### SkipIfNull ( [Bool?](https://wiki.resonite.com/Type:Nullable%601 "Type:Nullable`1") -\> [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

If `True`, no pulses will be generated if `UpdatingUser` is null. Otherwise, impulses will be generated by the [host user](https://wiki.resonite.com/ProtoFlux:Host_User "ProtoFlux:Host User") if `UpdatingUser` is null.
