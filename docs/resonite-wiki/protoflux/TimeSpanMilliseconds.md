# ProtoFlux:TimeSpanMilliseconds

> Source: https://wiki.resonite.com/ProtoFlux:TimeSpanMilliseconds

Milliseconds

TimeSpan

\*

Timespan

The `TimeSpan Milliseconds` node takes in a [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan") and returns with only the amount of millisecondsin that TimeSpan. Anything other than the milliseconds will be ignored.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you want the accumulated milliseconds that includes the fractional portion of a millisecond in one value, use [TimeSpan Total Milliseconds](https://wiki.resonite.com/ProtoFlux:TimeSpan_Total_Milliseconds "ProtoFlux:TimeSpan Total Milliseconds") instead.


## Inputs

### TimeSpan ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The TimeSpan we are getting info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

Returns the amount of millisecondsin this TimeSpan as an int.
