Websockets

This node will allow you do detect when a [WebSocket client](https://wiki.resonite.com/WebsocketClient_(Component) "WebsocketClient (Component)") is connected or disconnected.

## Outputs

### OnConnected ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

An [Call](https://wiki.resonite.com/Impulses "Impulses") triggered on a connection to the WebSocket server.

### OnDisconnected ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

An [Call](https://wiki.resonite.com/Impulses "Impulses") triggered when the connection to the WebSocket server is cut.

## Globals

### Client ( [WebsocketClient](https://wiki.resonite.com/WebsocketClient_(Component) "WebsocketClient (Component)"))

A [WebSocketClient](https://wiki.resonite.com/WebsocketClient_(Component) "WebsocketClient (Component)") component.
