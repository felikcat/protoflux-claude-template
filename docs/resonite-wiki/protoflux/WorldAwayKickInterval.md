# ProtoFlux:WorldAwayKickInterval

> Source: https://wiki.resonite.com/ProtoFlux:WorldAwayKickInterval

World Away Kick Interval

\*

Info

The `World Away Kick Interval` node returns this world's "Max AFK Minutes" setting from the [Session](https://wiki.resonite.com/Session "Session") tab in the [Dash](https://wiki.resonite.com/Dash "Dash"), represented as a TimeSpan value.

## Outputs

### \\* ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The world's Max AFK Minuets.
