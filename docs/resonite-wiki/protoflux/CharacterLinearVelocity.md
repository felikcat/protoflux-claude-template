# ProtoFlux:CharacterLinearVelocity

> Source: https://wiki.resonite.com/ProtoFlux:CharacterLinearVelocity

Character Linear Velocity

Character

\*

Physics

The `Character Linear Velocity` node takes in a [character controller](https://wiki.resonite.com/Component:CharacterController "Component:CharacterController") reference and returns the current velocity that it is currently going at over time.

## Inputs

### Character ( [Character Controller](https://wiki.resonite.com/Component:CharacterController "Component:CharacterController"))

The character controller reference.

## Outputs

### \\* ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The rate this character controller is moving over time.
