# ProtoFlux:AreAppFacetsOpened

> Source: https://wiki.resonite.com/ProtoFlux:AreAppFacetsOpened

Are App Facets Opened

User

\*

Status

The **Are App Facets Opened** node takes in a [user](https://wiki.resonite.com/Type:User "Type:User") and returns if that user's [facet](https://wiki.resonite.com/Facets "Facets") anchors are opened currently.

## Inputs

### \\* ( [User](https://wiki.resonite.com/Type:User "Type:User"))

The user to check.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if this user's facet anchors are opened currently.
