# ProtoFlux:BodyNodeChirality

Body Node Chirality

Node

\*

Body Nodes

Body Node Chirality is a ProtoFlux node that takes a Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) and returns whether it's on the left or right side of the human body, or -1 if it's a "None" body node or in the center.

## Inputs

### Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))

The body node to check if it is on the left or right side of the body.

## Outputs

### \\* ( [Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality"))

Returns if Node ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode")) is on the left or right side of a human body. Returns null if the node is in the center of the body or "None".
