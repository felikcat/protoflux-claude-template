# ProtoFlux:NullString

> Source: https://wiki.resonite.com/ProtoFlux:NullString

Null String

\*

Operators

Null String is a ProtoFlux node that returns a string with no content even white space, so the object is null, but still a [String](https://wiki.resonite.com/Type:String "Type:String") object.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Types:String "Types:String"))

The string A ( [String](https://wiki.resonite.com/Type:String "Type:String")) repeated B ( [int](https://wiki.resonite.com/Types:Int "Types:Int")) times.
