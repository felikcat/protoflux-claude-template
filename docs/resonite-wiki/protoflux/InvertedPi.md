# ProtoFlux:InvertedPi

> Source: https://wiki.resonite.com/ProtoFlux:InvertedPi

1/π

Constants

The **InvertedPi** or **1/π** node outputs the value of 1 ÷ [π](https://en.wikipedia.org/wiki/Pi).

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value of 1 ÷ π ÷ 2, approximately `0.3183099`.
