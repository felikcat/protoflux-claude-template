UI
The `Focus Focusable` node sets the target [IFocusable](https://wiki.resonite.com/Type:IFocusable "Type:IFocusable") UI for the [user](https://wiki.resonite.com/User "User"). When the UI is focused (if it was not already focused beforehand), it places your caret at the end of any text input. 
## Inputs
### * ([Call](https://wiki.resonite.com/Impulses "Impulses"))
Calls an impulse to set the target focus of a UI for the user. 
### Target ([IFocusable](https://wiki.resonite.com/Type:IFocusable "Type:IFocusable"))
The target IFocusable reference to focus. 
## Outputs
### Next ([Continuation](https://wiki.resonite.com/Impulses "Impulses"))
Fires after the focus has been focused.
