# ProtoFlux:UserUploadSpeedMax

> Source: https://wiki.resonite.com/ProtoFlux:UserUploadSpeedMax

User Upload Speed Max

User

\*

Info

The `User Upload Speed Max` node returns a user's maximum upload speed in this session.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The max upload speed of the user in bytes.
