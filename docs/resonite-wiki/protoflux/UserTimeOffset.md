# ProtoFlux:UserTimeOffset

> Source: https://wiki.resonite.com/ProtoFlux:UserTimeOffset

User Time Offset

User

\*

Info

The `User Time Offset` node returns a user's time zone offset. More information about UTC here: [\[1\]](https://en.wikipedia.org/wiki/UTC_offset)

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [TimeSpan](https://wiki.resonite.com/Type:TimeSpan "Type:TimeSpan"))

The time offset of the user.
