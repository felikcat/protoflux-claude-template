# ProtoFlux:UserTime

> Source: https://wiki.resonite.com/ProtoFlux:UserTime

User Time

User

\*

Info

The `User Time` node returns a user's device system time.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [DateTime](https://wiki.resonite.com/Type:DateTime "Type:DateTime"))

The system time of the user.
