# ProtoFlux:UnescapeUriDataString

> Source: https://wiki.resonite.com/ProtoFlux:UnescapeUriDataString

Unescape Uri Data String

String

\*

Network

The `Unescape Uri Data String` node takes in an encoded string literal that uses [Percent Encoding](https://en.wikipedia.org/wiki/Percent-encoding) and converts that string into another string that is human readable. This is used mostly for applications that can read the characters and glyphs that are complex.

## Inputs

### \\* ( [string](https://wiki.resonite.com/Type:String "Type:String"))

The string literal that we want to decode.

## Outputs

### \\* ( [string](https://wiki.resonite.com/Type:String "Type:String"))

The decoded string literal.
