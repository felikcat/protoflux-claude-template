# ProtoFlux:UserVRActive

> Source: https://wiki.resonite.com/ProtoFlux:UserVRActive

User VR Active

User

\*

Info

The `User VR Active` node returns if a user is in VR or not.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The user being in VR mode or not.
