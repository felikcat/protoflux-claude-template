# ProtoFlux:DecoposedScale

> Source: https://wiki.resonite.com/ProtoFlux:DecoposedScale

DecomposeScale

A

\*

Matrix

The **Decomposed Scale** node takes in a matrix value and decomposes it into a scale value.

## Inputs

### A (Pseudo-Generic)

The matrix value.

## Outputs

### \\* (Pseudo-Generic)

Returns the decomposed scale value.
