# ProtoFlux:UserViewVoiceActive

> Source: https://wiki.resonite.com/ProtoFlux:UserViewVoiceActive

User View Voice Active

User

\*

Info

The `User View Voice Active` node returns a user's view voice. This is used with the component [View Reference Controller](https://wiki.resonite.com/Component:ViewReferenceController "Component:ViewReferenceController").

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The view voice of the user.
