# ProtoFlux:Deg2Rad

> Source: https://wiki.resonite.com/ProtoFlux:Deg2Rad

Deg 2 Rad

Constants

The **DegToRad** or **Deg 2 Rad** node outputs the value of [π](https://en.wikipedia.org/wiki/Pi) ÷ 180\. Multiply by this to convert a value from degrees to [radians](https://en.wikipedia.org/wiki/Radian).

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Value of π ÷ 180 value, approximately `0.01745329`.
