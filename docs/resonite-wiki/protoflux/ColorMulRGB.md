# ProtoFlux:ColorMulRGB

Mul RGB

color

\*

Channels

Mul RGB is a ProtoFlux node that does a Mul operation to a color on the RGB channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Mul operation to the RGB channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Mul operation to the RGB channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Mul operation done to the RGB channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
