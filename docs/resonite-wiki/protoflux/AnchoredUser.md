# ProtoFlux:AnchoredUser

> Source: https://wiki.resonite.com/ProtoFlux:AnchoredUser

Anchored User

Anchor

User

Anchors

Anchored User is a ProtoFlux node that returns the [user](https://wiki.resonite.com/Type:User "Type:User") that is anchored to the provided Anchor ( [IAvatarAnchor](https://wiki.resonite.com/Type:IAvatarAnchor "Type:IAvatarAnchor")). This node will return a null user if there is no user anchored to the anchor.

## Inputs

### Anchor ( [IAvatarAnchor](https://wiki.resonite.com/Type:IAvatarAnchor "Type:IAvatarAnchor"))

The anchor to check for an anchored user.

## Outputs

### User ( [User](https://wiki.resonite.com/Type:User "Type:User"))

The user that is currently anchored, or null.
