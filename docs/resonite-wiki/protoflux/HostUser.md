# ProtoFlux:HostUser

> Source: https://wiki.resonite.com/ProtoFlux:HostUser

Host User

\*

Users

The `Host User` node gets the host user in this world's session.

![](https://wiki.resonite.com/images/b/b7/DangerIcon.svg)

If you are running a headless session, the headless will be host and code using this node will return the headless, which may be unintentional behavior in your code if not accounted for!


## Outputs

### \\* ( [User](https://wiki.resonite.com/User "User"))

The host user in this world's session.
