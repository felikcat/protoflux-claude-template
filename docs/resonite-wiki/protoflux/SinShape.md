# ProtoFlux:SinShape

> Source: https://wiki.resonite.com/ProtoFlux:SinShape

Sin Shape

Value

\*

Shaping

The **Sin Shape** node takes in a value between `0` and `1`, then returns the sine shape value (a curve that starts fast and slows down at the end).

## Inputs

### Value (Pseudo-Generic)

The value for the sine calculation.

## Outputs

### \\* (Pseudo-Generic)

Returns the sine shape value.
