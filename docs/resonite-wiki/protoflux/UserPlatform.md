# ProtoFlux:UserPlatform

> Source: https://wiki.resonite.com/ProtoFlux:UserPlatform

User Platform

User

\*

Info

The `User Platform` node returns the user's current platform/operating system they are playing Resonite on.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [Platform](https://wiki.resonite.com/Type:Platform "Type:Platform"))

The platform the user is currently on.
