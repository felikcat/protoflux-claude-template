Websockets

This node allows you to connect to a [websocket](https://wiki.resonite.com/WebSocket "WebSocket") server via the [WebSocketClient](https://wiki.resonite.com/WebsocketClient_(Component) "WebsocketClient (Component)") component.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

An incoming [Call](https://wiki.resonite.com/Impulses "Impulses") that will trigger the node.

### Client ( [WebsocketClient](https://wiki.resonite.com/WebsocketClient_(Component) "WebsocketClient (Component)"))

A [WebsocketClient](https://wiki.resonite.com/WebsocketClient_(Component) "WebsocketClient (Component)") component.

### URL ( [URI](https://wiki.resonite.com/Type:Uri "Type:Uri"))

An [URI](https://wiki.resonite.com/Type:Uri "Type:Uri") to the WebSocket server.

### HandlingUser ( [User](https://wiki.resonite.com/Type:User "Type:User"))

An [user](https://wiki.resonite.com/Type:User "Type:User") that will handle the WebSocket requests.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

A continuation impulse to trigger the rest of the code.
