User Head Output Device

User

Info

The `User Head Output Device` node shows the given user's display platform or output from [Resonite](https://wiki.resonite.com/Resonite "Resonite").

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [HeadOutputDevice](https://wiki.resonite.com/Type:HeadOutputDevice "Type:HeadOutputDevice"))

The Head Output Device of the user.
