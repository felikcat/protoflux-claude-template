# ProtoFlux:EnumToNumber

> Source: https://wiki.resonite.com/ProtoFlux:EnumToNumber

EnumToNumber

Value

\*

Enums

See [Enums](https://wiki.resonite.com/Category:Enums "Category:Enums") for a more in depth explaination on what Enums are.
This node can be useful for keeping enums as numbers, in cases like [Cloud Variables](https://wiki.resonite.com/Cloud_Variables "Cloud Variables") that can't store Enums.

For what to input for variants other than `EnumToInt`, please see [Complex Types in Components](https://wiki.resonite.com/Complex_Types_in_Components "Complex Types in Components")

This node has the following variants, with their respective outputs:

| Variant | Node Type | Output |
| --- | --- | --- |
| Enum To Number ( [Byte](https://wiki.resonite.com/Types:Byte "Types:Byte")) | `EnumToByte` | provided Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) as a Byte |
| Enum To Number ( [Int](https://wiki.resonite.com/Types:Int "Types:Int")) | `EnumToInt` | provided Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) as an Int |
| Enum To Number ( [Long](https://wiki.resonite.com/Types:Long "Types:Long")) | `EnumToLong` | provided Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) as a Long |
| Enum To Number ( [Sbyte](https://wiki.resonite.com/Types:Sbyte "Types:Sbyte")) | `EnumToSbyte` | provided Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) as a Sbyte |
| Enum To Number ( [Short](https://wiki.resonite.com/Types:Short "Types:Short")) | `EnumToShort` | provided Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) as a Short |
| Enum To Number ( [Uint](https://wiki.resonite.com/Types:Uint "Types:Uint")) | `EnumToUint` | provided Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) as an Uint |
| Enum To Number ( [Ulong](https://wiki.resonite.com/Types:Ulong "Types:Ulong")) | `EnumToUlong` | provided Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) as an Ulong |
| Enum To Number ( [Ushort](https://wiki.resonite.com/Types:Ushort "Types:Ushort")) | `EnumToUshort` | provided Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums")) as a Ushort |

## Inputs

### Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic)

The Enum to convert to a whole number with \* (Pseudo-generic (Whole Number))'s type according to [Enums Page](https://wiki.resonite.com/Category:Enums "Category:Enums");

## Outputs

### \\* (Variant)

The Value ( [Enum](https://wiki.resonite.com/Category:Enums "Category:Enums") Generic)'s Whole Number equivalent.
