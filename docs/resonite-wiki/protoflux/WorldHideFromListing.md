# ProtoFlux:WorldHideFromListing

> Source: https://wiki.resonite.com/ProtoFlux:WorldHideFromListing

World Hide From Listing

\*

Info

The `World Hide From Listing` node returns this world's "Don't show in session lists" setting from the [Session](https://wiki.resonite.com/Session "Session") tab in the [Dash](https://wiki.resonite.com/Dash "Dash").

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The world's "Don't show in session lists" setting.
