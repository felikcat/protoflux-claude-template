# ProtoFlux:ViewOverriden

> Source: https://wiki.resonite.com/ProtoFlux:ViewOverriden

View Overriden

\*

Local Output

The **View Overriden** node returns if the local user's view output is being overridden by something.

When in [Desktop Mode](https://wiki.resonite.com/Desktop_Mode "Desktop Mode") or, starting [Resonite](https://wiki.resonite.com/Resonite "Resonite") in VR mode and then hot-swapping to Desktop Mode, this node will return as true.

When changing your view by using a different head proxy, templated head proxy, or something else that moves your view point (such as the [AvatarUserRootOverrideAssigner](https://wiki.resonite.com/Component:AvatarUserRootOverrideAssigner "Component:AvatarUserRootOverrideAssigner") or [AvatarObjectSlot](https://wiki.resonite.com/Component:AvatarObjectSlot "Component:AvatarObjectSlot") components), this node will return as true.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

Returns if the local user's view output has been overridden by something.
