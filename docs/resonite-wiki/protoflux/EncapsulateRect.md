# ProtoFlux:EncapsulateRect

> Source: https://wiki.resonite.com/ProtoFlux:EncapsulateRect

Encapsulate Rect

A

\*

B

Rects

The **Encapsulate Rect** node takes in 2 [rect](https://wiki.resonite.com/Type:Rect "Type:Rect") [values](https://wiki.resonite.com/Value_Type "Value Type"), and returns a rect that encapsulates both of them into one rect.

![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you need to check for masking/cliping of 2 rects, use the [Clip Rect](https://wiki.resonite.com/ProtoFlux:Clip_Rect "ProtoFlux:Clip Rect") node instead.


## Inputs

### A ( [Rect](https://wiki.resonite.com/Type:Rect "Type:Rect"))

The first rect.

### B ( [Rect](https://wiki.resonite.com/Type:Rect "Type:Rect"))

The second rect.

## Outputs

### \\* ( [Rect](https://wiki.resonite.com/Type:Rect "Type:Rect"))

Returns the encapsulated rect from the 2 provided rects.
