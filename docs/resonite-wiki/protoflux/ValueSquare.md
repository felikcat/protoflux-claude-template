# ProtoFlux:ValueSquare

> Source: https://wiki.resonite.com/ProtoFlux:ValueSquare

n2

N

Operators

The **Value Square** node takes in a value and squares it, basically multiplying itself by 2 and returning that value.

![](https://wiki.resonite.com/images/d/dd/InformationIcon.svg)

This is similar to using the [Pow](https://wiki.resonite.com/ProtoFlux:Pow "ProtoFlux:Pow") node, and using the value that you want to square and setting the power to 2.


![](https://wiki.resonite.com/images/c/c1/SuggestionIcon.svg)

If you want to cube a value, use the [Value Cube](https://wiki.resonite.com/ProtoFlux:Value_Cube "ProtoFlux:Value Cube") node instead.


## Inputs

### N (Pseudo-Generic)

The number we want to square.

## Outputs

### \\* (Pseudo-Generic)

Returns the squared value.
