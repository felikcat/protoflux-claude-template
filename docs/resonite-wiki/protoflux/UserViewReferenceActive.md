# ProtoFlux:UserViewReferenceActive

> Source: https://wiki.resonite.com/ProtoFlux:UserViewReferenceActive

User View Reference Active

User

\*

Info

The `User View Reference Active` node returns a user's view reference. This is used with the component [View Reference Controller](https://wiki.resonite.com/Component:ViewReferenceController "Component:ViewReferenceController").

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The view reference of the user.
