# ProtoFlux:CloseContextMenu

> Source: https://wiki.resonite.com/ProtoFlux:CloseContextMenu

Close Context Menu

\*

Next

Summoner

Interaction

The `Close Context Menu` node takes in a summoner (the thing that opened the [context menu](https://wiki.resonite.com/Context_menu "Context menu")), and tells it to close the menu for the user.

## Inputs

### \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Calls an impulse to close the [context menu](https://wiki.resonite.com/Context_menu "Context menu").

### Summoner ( [IWorldElement](https://wiki.resonite.com/Type:IWorldElement "Type:IWorldElement"))

The thing that opened the [context menu](https://wiki.resonite.com/Context_menu "Context menu").

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Continues execution from this node.
