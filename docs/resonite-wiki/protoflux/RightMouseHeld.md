# ProtoFlux:RightMouseHeld

> Source: https://wiki.resonite.com/ProtoFlux:RightMouseHeld

Right Mouse Held

\*

Mouse

The `Right Mouse Held` node returns the local [user's](https://wiki.resonite.com/User "User") right mouse button being held down.

## Outputs

### \\* ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The result from the user's right mouse button being held down currently.
