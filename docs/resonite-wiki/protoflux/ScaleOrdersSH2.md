Spherical Harmonics

The **Scale Orders SH2** node takes in a [SphericalHarmonicsL2](https://wiki.resonite.com/Type:SphericalHarmonicsL2 "Type:SphericalHarmonicsL2") along with a set of orders that refine/modify/scale it, then returns the new Spherical Harmonic of the same type.

## Inputs

### SH ( [SphericalHarmonicsL2](https://wiki.resonite.com/Type:SphericalHarmonicsL2 "Type:SphericalHarmonicsL2"))

The provided Spherical Harmonic.

### Order0 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 0).

### Order1 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 1).

### Order2 (Pseudo-Generic)

The refinement for the Spherical Harmonic (Order 2).

## Outputs

### \\* ( [SphericalHarmonicsL2](https://wiki.resonite.com/Type:SphericalHarmonicsL2 "Type:SphericalHarmonicsL2"))

The new Spherical Harmonic.
