# ProtoFlux:UserFromID

> Source: https://wiki.resonite.com/ProtoFlux:UserFromID

User From ID

UserId

\*

MachineId

Users

The `User From ID` node takes in a user ID and/or machine ID and returns with a user. This only works for users in the same session.

## Inputs

### UserId ( [String](https://wiki.resonite.com/String "String"))

The user ID to target user.

### MachineId ( [String](https://wiki.resonite.com/String "String"))

The machine ID to target user.

## Outputs

### \\* ( [User](https://wiki.resonite.com/User "User"))

The user that is found.
