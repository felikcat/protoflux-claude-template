# ProtoFlux:ColorAddSaturation

> Source: https://wiki.resonite.com/ProtoFlux:ColorAddSaturation

Add Saturation

color

\*

Value

Channels

Add Saturation is a ProtoFlux node that does a Add operation to a color on the Saturation channel.

## Inputs

### color ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

The color to do a Add operation to the Saturation channel on.

### Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float"))

The value to do a Add operation to the Saturation channel on the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")).

## Outputs

### \\* ( [color](https://wiki.resonite.com/Type:Color "Type:Color"))

Returns the provided color ( [color](https://wiki.resonite.com/Type:Color "Type:Color")) with a Add operation done to the Saturation channel using Value ( [Float](https://wiki.resonite.com/Type:Float "Type:Float")).
