# ProtoFlux:SetSlotPersistentSelf

> Source: https://wiki.resonite.com/ProtoFlux:SetSlotPersistentSelf

Set Slot Persistent Self

\*

Next

Instance

Persistent

Info

Set's the persistent field of the provided Instance [slot](https://wiki.resonite.com/Slot "Slot") upon calling \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")).

## Inputs

## \\* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) =

Call this to set the Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s persistent field to Persistent ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool")).

### Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))

[Slot](https://wiki.resonite.com/Slot "Slot") to set the property for.

### Persistent ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool"))

The value to set Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s persistent field to.

## Outputs

### Next ( [Continuation](https://wiki.resonite.com/Impulses "Impulses"))

Sends an impulse when Instance ( [Slot](https://wiki.resonite.com/Slot "Slot"))'s persistent field is set to Persistent ( [bool](https://wiki.resonite.com/Type:Bool "Type:Bool")) when \* ( [Call](https://wiki.resonite.com/Impulses "Impulses")) is called.
