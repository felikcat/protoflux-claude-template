# ProtoFlux:ClosestPointOnLine

> Source: https://wiki.resonite.com/ProtoFlux:ClosestPointOnLine

Closest Point On Line

LinePoint0

\*

LinePoint1

Point

Geometry 3D

The **Closest Point On Line** node takes in the line parameters (2 points in 3D space), and a provided point to check. Then it returns the point that it is closest to on the line, based on the defined parameters.

## Inputs

### LinePoint0 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The first point.

### LinePoint1 ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The second point.

### Point ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

The point to check.

## Outputs

### \\* ( [float3](https://wiki.resonite.com/Type:Float3 "Type:Float3"))

Returns the point on the line compared with the point given.
