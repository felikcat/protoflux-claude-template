The **Value Mul Multi** node takes in a list of values and multiplies from the first given value onwards. The first value is always the starting point. Then the second value will multiply from the first, giving the result for the third input to multiply from, so on and so fourth. More information can be found on the [Mul](https://wiki.resonite.com/ProtoFlux:Mul "ProtoFlux:Mul") node page.

## Inputs

### Inputs (Pseudo-Generic)

The numbers to multiply with, using the first value as a starting point.

## Outputs

### \\* (Pseudo-Generic)

The final result of all the multiplication operations.
