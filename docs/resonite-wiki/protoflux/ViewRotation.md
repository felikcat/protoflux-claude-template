# ProtoFlux:ViewRotation

> Source: https://wiki.resonite.com/ProtoFlux:ViewRotation

View Rotation

\*

Local Output

The **View Rotation** node returns the local user's view output rotation.

When in [Desktop Mode](https://wiki.resonite.com/Desktop_Mode "Desktop Mode") or [VR Mode](https://wiki.resonite.com/VR_Mode "VR Mode"), this node will return the value of where your head proxy is rotated. When an item or object that can override your view is used (such as using a different head proxy, templated head proxy, or something else that moves your view point (such as the [AvatarUserRootOverrideAssigner](https://wiki.resonite.com/Component:AvatarUserRootOverrideAssigner "Component:AvatarUserRootOverrideAssigner") or [AvatarObjectSlot](https://wiki.resonite.com/Component:AvatarObjectSlot "Component:AvatarObjectSlot") components)), this node will return a new rotation value.

## Outputs

### \\* ( [floatQ](https://wiki.resonite.com/Type:FloatQ "Type:FloatQ"))

Returns the local user's view output rotation.
