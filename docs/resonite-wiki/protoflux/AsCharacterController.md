# ProtoFlux:AsCharacterController

> Source: https://wiki.resonite.com/ProtoFlux:AsCharacterController

As Character Controller

Collider

\*

Physics

The `As Character Controller` node is used to get the [character controller](https://wiki.resonite.com/Component:CharacterController "Component:CharacterController") reference using the [ICollider](https://wiki.resonite.com/Type:ICollider "Type:ICollider") from a [Slot](https://wiki.resonite.com/Slot "Slot") that associated with that collider, not necessarily what it collides with.

## Inputs

### Collider ( [ICollider](https://wiki.resonite.com/Type:ICollider "Type:ICollider"))

The specific collider reference from a slot.

## Outputs

### \\* ( [CharacterController](https://wiki.resonite.com/Component:CharacterController "Component:CharacterController"))

The character controller reference we receive from the provided ICollider.
