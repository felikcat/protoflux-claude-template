This node provides Twitch follow event information from a [TwitchInterface](https://wiki.resonite.com/Component:TwitchInterface "Component:TwitchInterface").

Note that all outputs are generated on a new Twitch follow, and will not display any information when connected to a display node.

## Outputs

### OnFollow ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

Fires on receiving a new Twitch channel follow.

### UserId ( [String](https://wiki.resonite.com/Type:String "Type:String"))

Outputs the Twitch Channel ID of the follower.

### DisplayName ( [String](https://wiki.resonite.com/Type:String "Type:String"))

Outputs the Twitch username of the follower.
