# ProtoFlux:UserDownloadSpeed

> Source: https://wiki.resonite.com/ProtoFlux:UserDownloadSpeed

User Download Speed

User

\*

Info

The `User Download Speed` node returns a user's download speed.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The download speed of the user in bytes.
