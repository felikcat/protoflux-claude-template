# ProtoFlux:ComposeFinger

Body Nodes

Compose Finger is a ProtoFlux node that allows you to get a [body node enum](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode") from a [FingerType](https://wiki.resonite.com/Type:FingerType "Type:FingerType"), [FingerSegmentType](https://wiki.resonite.com/Type:FingerSegmentType "Type:FingerSegmentType") and a [Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality") also known as a side.

## Inputs

### Finger ( [FingerType](https://wiki.resonite.com/Type:FingerType "Type:FingerType"))

The finger you want to get a body node enum for.

### Segment ( [FingerSegmentType](https://wiki.resonite.com/Type:FingerSegmentType "Type:FingerSegmentType"))

The segment of the Finger ( [FingerType](https://wiki.resonite.com/Type:FingerType "Type:FingerType")) you want to get a body node enum for.

### Chirality ( [Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality"))

Which hand, left or right, you want to get a body node for using Finger ( [FingerType](https://wiki.resonite.com/Type:FingerType "Type:FingerType")) and Segment ( [FingerSegmentType](https://wiki.resonite.com/Type:FingerSegmentType "Type:FingerSegmentType")).

## Outputs

### \\* ( [BodyNode](https://wiki.resonite.com/Type:BodyNode "Type:BodyNode"))

The body node enum that is on the finger: Finger ( [FingerType](https://wiki.resonite.com/Type:FingerType "Type:FingerType")), on the segment: Segment ( [FingerSegmentType](https://wiki.resonite.com/Type:FingerSegmentType "Type:FingerSegmentType")) on that finger, on the Chirality ( [Chirality](https://wiki.resonite.com/Type:Chirality "Type:Chirality")) side of the body.
