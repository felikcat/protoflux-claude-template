# ProtoFlux:WorldSessionURL

> Source: https://wiki.resonite.com/ProtoFlux:WorldSessionURL

World Session URL

\*

Info

The `World Session URL` node returns this world's session URL. This can be useful when you need to focus back into a world using this URL.

## Outputs

### \\* ( [String](https://wiki.resonite.com/Type:String "Type:String"))

The world's session URL.
