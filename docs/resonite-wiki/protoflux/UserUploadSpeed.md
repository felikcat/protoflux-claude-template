# ProtoFlux:UserUploadSpeed

> Source: https://wiki.resonite.com/ProtoFlux:UserUploadSpeed

User Upload Speed

User

\*

Info

The `User Upload Speed` node returns a user's upload speed in this session.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

The upload speed of the user in bytes.
