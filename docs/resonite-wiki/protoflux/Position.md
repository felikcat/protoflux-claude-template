# ProtoFlux:Position

> Source: https://wiki.resonite.com/ProtoFlux:Position

Position

Source

\*

Media

The position node tells a IPlayable's current position in seconds.

## Inputs

### Source ( [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable"))

The [IPlayable](https://wiki.resonite.com/Type:IPlayable "Type:IPlayable") to set the position on.

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Returns the position of the playback.
