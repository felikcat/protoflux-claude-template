The **Websocket Text Message Receiver** node allows you to receive text messages from a [WebsocketClient](https://wiki.resonite.com/WebsocketClient_(Component) "WebsocketClient (Component)").

## Outputs

### OnReceived ( [Call](https://wiki.resonite.com/Impulses "Impulses"))

A [Call](https://wiki.resonite.com/Impulses "Impulses") triggered when a text message is received.

### Data ( [String](https://wiki.resonite.com/Types:String "Types:String"))

The actual message sent by the server as a [string](https://wiki.resonite.com/Types:String "Types:String").

## Globals

### Client ( [WebsocketClient](https://wiki.resonite.com/WebsocketClient_(Component) "WebsocketClient (Component)"))

A [WebSocketClient](https://wiki.resonite.com/WebsocketClient_(Component) "WebsocketClient (Component)") component.
