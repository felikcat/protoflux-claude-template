Find Character Controller From User
Source
*
Physics
The `Find Character Controller From User` node takes in a [user](https://wiki.resonite.com/User "User") reference and tries to find the [character controller](https://wiki.resonite.com/Component:CharacterController "Component:CharacterController") from that user, then returning one when one is found. 
## Inputs
### Source ([User](https://wiki.resonite.com/User "User"))
The source that we want to try to find the character controller in. 
## Outputs
### * ([Character Controller](https://wiki.resonite.com/Component:CharacterController "Component:CharacterController"))
The character controller that we found.
