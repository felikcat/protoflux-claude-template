# ProtoFlux:DesktopFOV

> Source: https://wiki.resonite.com/ProtoFlux:DesktopFOV

Desktop FOV

\*

Local Output

The **Desktop FOV** node returns the local user's desktop field of view. This value can change depending on what is set in the [Settings](https://wiki.resonite.com/Settings "Settings") tab in the [Dash](https://wiki.resonite.com/Dash "Dash").

## Outputs

### \\* ( [float](https://wiki.resonite.com/Type:Float "Type:Float"))

Returns the local user's desktop field of view.
