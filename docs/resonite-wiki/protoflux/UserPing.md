# ProtoFlux:UserPing

> Source: https://wiki.resonite.com/ProtoFlux:UserPing

User Ping

User

\*

Info

The `User Ping` node returns the user's ping or latency.

## Inputs

### User ( [User](https://wiki.resonite.com/User "User"))

The user we are getting info from.

## Outputs

### \\* ( [int](https://wiki.resonite.com/Type:Int "Type:Int"))

The latency of the user.
