# Component:UI CircleSegment

> Source: https://wiki.resonite.com/Component:UI_CircleSegment

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/3/34/UI_CircleSegmentComponent.png/510px-UI_CircleSegmentComponent.png)](https://wiki.resonite.com/File:UI_CircleSegmentComponent.png) **UI Circle Segment** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **UI\_CircleSegment** material is used to render a curved circle onto a quad. This component only functions properly when rendered on a [Component:CircleSegmentShaderMesh](https://wiki.resonite.com/Component:CircleSegmentShaderMesh "Component:CircleSegmentShaderMesh") as it relies on the UV data generated by that mesh to create it's visuals.
