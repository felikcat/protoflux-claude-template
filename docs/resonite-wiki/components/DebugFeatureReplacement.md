# Component:DebugFeatureReplacement

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/6/63/DebugFeatureReplacementComponent.png/510px-DebugFeatureReplacementComponent.png)](https://wiki.resonite.com/File:DebugFeatureReplacementComponent.png) **Debug Feature Replacement** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **DebugFeatureReplacement** component is used to test upgrading features which is an Elements.Core.dll C# tag function.

![](https://wiki.resonite.com/images/f/fe/WarningIcon.svg)

This game element is a debug item. Elements like this may be removed at any time without warning, and creations should not rely on it.
