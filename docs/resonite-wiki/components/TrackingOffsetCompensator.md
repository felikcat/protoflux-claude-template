# Component:TrackingOffsetCompensator

> Source: https://wiki.resonite.com/Component:TrackingOffsetCompensator

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/b/bc/TrackingOffsetCompensatorComponent.png/510px-TrackingOffsetCompensatorComponent.png)](https://wiki.resonite.com/File:TrackingOffsetCompensatorComponent.png) **Tracking Offset Compensator** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **TrackingOffsetCompensator** counteracts changes to tracking offset to all trackers at once, so this one doesn't move when all others do. usually due to a tracking offset component or seated mode.
