# Component:Glue

> Source: https://wiki.resonite.com/Component:Glue

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/7/7c/GlueComponent.png/510px-GlueComponent.png)](https://wiki.resonite.com/File:GlueComponent.png) **Glue** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **Glue** component is generated by the [GlueTool](https://wiki.resonite.com/Component:GlueTool "Component:GlueTool") when the user press and drags primary with the tool equipped. The mode on that tool determines what this component's `GlueMode` will be.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `GlueMode` | **[Glue.Mode](https://wiki.resonite.com/Component:Glue#Mode)** | How to Glue the objects in the area or effect. |
| `Active` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether this component is ready to glue. |
| `DryTime` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | How long it takes for this Glue to dry. |
| `Expire` | **[Double](https://wiki.resonite.com/Type:Double "Type:Double")** | When this component should delete itself in world time. |
| `GluingUser` | **[User](https://wiki.resonite.com/Type:User "Type:User")** | The user that is gluing this glue. |
| `_forceDry` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether to force dry this glue. |

## Mode

| Name | Value | Description |
| --- | --- | --- |
| `ParentObjects` | 0 | Parent objects will parent all objects within the glue bubble to a single parent. |
| `BakeMeshes` | 1 | Bake meshes will bake all meshes within the glue bubble to a single asset. |
| `BakeSkinnedMeshes` | 2 | Bake skinned meshes will bake multiple skinned meshes in a single asset. |
