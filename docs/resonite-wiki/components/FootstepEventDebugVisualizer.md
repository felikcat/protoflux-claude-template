# Component:FootstepEventDebugVisualizer

> Source: https://wiki.resonite.com/Component:FootstepEventDebugVisualizer

Collapse **Component image**

[File:FootstepEventDebugVisualizerComponent.png](https://wiki.resonite.com/index.php?title=Special:Upload&wpDestFile=FootstepEventDebugVisualizerComponent.png "File:FootstepEventDebugVisualizerComponent.png") **Footstep Event Debug Visualizer** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **FootStepEventDebugVisualizer** shows footstep collisions with a collider on the same Slot as the one it's attached on.

![](https://wiki.resonite.com/images/f/fe/WarningIcon.svg)

This game element is a debug item. Elements like this may be removed at any time without warning, and creations should not rely on it.
