# Component:MigrationListUI

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/2/25/MigrationListUIComponent.png/510px-MigrationListUIComponent.png)](https://wiki.resonite.com/File:MigrationListUIComponent.png) **Migration List UI** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **MigrationListUI** component is used to list previous and current migration tasks.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `SelectedTaskId` | **[String](https://wiki.resonite.com/Type:String "Type:String")** | The task that the user has selected. |
| `_listRoot` | **[Slot](https://wiki.resonite.com/Type:Slot "Type:Slot")** | The place to list migrations. |
