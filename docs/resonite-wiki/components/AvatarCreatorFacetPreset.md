# Component:AvatarCreatorFacetPreset

> Source: https://wiki.resonite.com/Component:AvatarCreatorFacetPreset

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/8/87/AvatarCreatorFacetPresetComponent.png/510px-AvatarCreatorFacetPresetComponent.png)](https://wiki.resonite.com/File:AvatarCreatorFacetPresetComponent.png) **Avatar Creator Facet Preset** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **AvatarCreatorFacetPreset** component loads the Avatar Creator facet when it's not `_fullyLoaded`. It does this by loading the facet from the cloud, and using that loaded data to make the preset
