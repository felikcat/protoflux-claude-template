# Component:WorldListFacetPreset

> Source: https://wiki.resonite.com/Component:WorldListFacetPreset

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/a/aa/WorldListFacetPresetComponent.png/510px-WorldListFacetPresetComponent.png)](https://wiki.resonite.com/File:WorldListFacetPresetComponent.png) **World List Facet Preset** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **WorldListFacetPreset** component loads the World List facet when it's not `_fullyLoaded`. It does this by loading the facet from the cloud, and using that loaded data to make the preset

used in the main dash. no need to be used by the user unless they are missing the World List facet.
