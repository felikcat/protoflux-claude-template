# Component:VR ActiveSetttingSwitchSource

> Source: https://wiki.resonite.com/Component:VR_ActiveSetttingSwitchSource

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/e/e1/VR_ActiveSetttingSwitchSourceComponent.png/510px-VR_ActiveSetttingSwitchSourceComponent.png)](https://wiki.resonite.com/File:VR_ActiveSetttingSwitchSourceComponent.png) **VR Active Settting Switch Source** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **VR\_ActiveSetttingSwitchSource** component works in tandem with the [VR Active Setting Switch component](https://wiki.resonite.com/Component:VR_ActiveSettingSwitch "Component:VR ActiveSettingSwitch"). this component specifically acts as the source of enforcing a setting set switch from VR settings to desktop settings.
