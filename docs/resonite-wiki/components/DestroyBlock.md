[![](https://wiki.resonite.com/images/thumb/d/d8/DestroyBlockComponent.png/510px-DestroyBlockComponent.png)](https://wiki.resonite.com/File:DestroyBlockComponent.png) **Destroy Block** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **Destroy Block** component blocks destroying a [Slot](https://wiki.resonite.com/Slot "Slot") [tagged](https://wiki.resonite.com/Tag "Tag") by this component via the [Context Menu](https://wiki.resonite.com/Context_menu "Context menu"). Does not block destroying via [Scene Inspector](https://wiki.resonite.com/Scene_Inspector_Dialog "Scene Inspector Dialog") or [ProtoFlux](https://wiki.resonite.com/ProtoFlux "ProtoFlux").
