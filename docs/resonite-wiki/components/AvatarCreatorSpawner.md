Avatar Creator Spawner is a component that works in either [userspace](https://wiki.resonite.com/Userspace "Userspace") or real space. This component recieves [button events](https://wiki.resonite.com/Button_Events "Button Events"), which means it works when on the same slot of any [button](https://wiki.resonite.com/Type:IButton "Type:IButton"). When it recieves a button press event, this component spawns a fully set up [Avatar Creator](https://wiki.resonite.com/Component:AvatarCreator "Component:AvatarCreator") in the world.
