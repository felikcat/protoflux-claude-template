# Component:UserPermissions

> Source: https://wiki.resonite.com/Component:UserPermissions

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/7/76/UserPermissionsComponent.png/510px-UserPermissionsComponent.png)](https://wiki.resonite.com/File:UserPermissionsComponent.png) **User Permissions** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **UserPermissions** component is used to control the permissions of users in a world for the selected roles.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `AllowEnableEditMode` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether the selected roles can enable their [Edit Mode](https://wiki.resonite.com/Edit_Mode "Edit Mode"). |
| `AllowRespawn` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether the selected roles can respawn other users. |
| `AllowSilence` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether the selected roles can silence other users. |
| `AllowKick` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether the selected roles can kick other users from the session. |
| `AllowBan` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether the selected roles can ban other users from this session. |
| `AllowAssignRoles` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether the selected roles can assign roles for other users that have a role less than them. |
| `AllowSeeOtherRoles` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether the selected roles can see what roles other users have. |
