The **GraphicChunkRoot** component is an internal component that is used for graphics, marking them as the root of a graphical image or process on [UIX](https://wiki.resonite.com/UIX "UIX") elements. [Users](https://wiki.resonite.com/User "User") don't need to worry as it does not have a clear use at this time (until one is dicovered).
