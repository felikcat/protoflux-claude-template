# Component:MigrationDetailFacetPreset

> Source: https://wiki.resonite.com/Component:MigrationDetailFacetPreset

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/a/ac/MigrationDetailFacetPresetComponent.png/510px-MigrationDetailFacetPresetComponent.png)](https://wiki.resonite.com/File:MigrationDetailFacetPresetComponent.png) **Migration Detail Facet Preset** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **MigrationDetailFacetPreset** component loads the Migration Detail facet when it's not `_fullyLoaded`. It does this by loading the facet from the cloud, and using that loaded data to make the preset

used in the main dash. no need to be used by the user unless they are missing the Migration Detail facet.
