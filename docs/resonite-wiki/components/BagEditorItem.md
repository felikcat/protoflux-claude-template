The **BagEditorItem** component is used in conjunction with a [Component:BagEditor](https://wiki.resonite.com/Component:BagEditor "Component:BagEditor"). This is a reference to an item in the bag. For more information, please see the page on [Component:BagEditor](https://wiki.resonite.com/Component:BagEditor "Component:BagEditor").

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `Item` | _direct_ **[SyncRef](https://wiki.resonite.com/Type:SyncRef "Type:SyncRef")** | The bag item this is referencing. |
