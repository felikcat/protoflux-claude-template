# Component:EnumFeed

> Source: https://wiki.resonite.com/Component:EnumFeed

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/4/4d/EnumFeed%601Component.png/510px-EnumFeed%601Component.png)](https://wiki.resonite.com/File:EnumFeed%601Component.png) **Enum Feed\`1** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **EnumFeed** component acts like a kind of [data feed](https://wiki.resonite.com/Data_Feeds "Data Feeds") that provides a list of enums that can be further used by feeds with filtering and searching.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `DisplayOrder` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether to feed the enums in order. |
| `Distinct` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether the enums should be distinct when gathering. |
| `IncludeObsolete` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether the enums should be gathered despite being tagged as obsolete. |
