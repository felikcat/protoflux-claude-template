# Component:PagingControlFacetPreset

> Source: https://wiki.resonite.com/Component:PagingControlFacetPreset

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/c/c3/PagingControlFacetPresetComponent.png/510px-PagingControlFacetPresetComponent.png)](https://wiki.resonite.com/File:PagingControlFacetPresetComponent.png) **Paging Control Facet Preset** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **PagingControlFacetPreset** component makes a container that stores a [PagingControl](https://wiki.resonite.com/Component:PagingControl "Component:PagingControl") component on the dash and tells that component to build a UIX for itself inside of this container.

## Fields

| Name | Type | Description |

| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `_pagingControl` | **[PagingControl](https://wiki.resonite.com/Component:PagingControl "Component:PagingControl")** | The paging control component for this facet |
