# Component:DashSettings

> Source: https://wiki.resonite.com/Component:DashSettings

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/7/7b/DashSettingsComponent.png/510px-DashSettingsComponent.png)](https://wiki.resonite.com/File:DashSettingsComponent.png) **Dash Settings** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **DashSettings** component is part of the [Settings](https://wiki.resonite.com/Settings "Settings") menu and is used to change the settings of how the [Dash](https://wiki.resonite.com/Dash "Dash") works.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `DashCurvature` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | How much to curve the dash in VR. |
| `OpenCloseSpeed` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | How fast the animation should play when opening and closing the dash. |
| `AllowReplacingSettings` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Allows you to remove panels from the settings menu, for usage as individual facets. |
