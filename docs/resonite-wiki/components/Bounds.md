# Component:Bounds

> Source: https://wiki.resonite.com/Component:Bounds

This page contains [changes](https://wiki.resonite.com/index.php?title=Component:Bounds&diff=96481) which are not marked for translation.

Bounds is a component that is used by the [Legacy Radio Component](https://wiki.resonite.com/Component:LegacyRadio "Component:LegacyRadio") When being generated by a UIBuilder, such as when being a legacy radio is created by the [DevTip](https://wiki.resonite.com/DevTip "DevTip"). This component implements [Type:IBounded](https://wiki.resonite.com/Type:IBounded "Type:IBounded") so it can be used with anything that accepts a bounds source such as [Component:BoundingBoxDriver](https://wiki.resonite.com/Component:BoundingBoxDriver "Component:BoundingBoxDriver").
