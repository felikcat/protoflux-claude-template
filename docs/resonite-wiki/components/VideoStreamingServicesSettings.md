# Component:VideoStreamingServicesSettings

> Source: https://wiki.resonite.com/Component:VideoStreamingServicesSettings

Collapse **Component image**

[File:VideoStreamingServicesSettingsComponent.png](https://wiki.resonite.com/index.php?title=Special:Upload&wpDestFile=VideoStreamingServicesSettingsComponent.png "File:VideoStreamingServicesSettingsComponent.png") **Video Streaming Services Settings** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **VideoStreamingServicesSettings** component handles settings for streaming service content downloading like allowing usage of browser cookies to get around the issue of "login to confirm you're not a bot" issue.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `UseCookiesFromBrowser` | **[CookiesBrowser](https://wiki.resonite.com/Type:CookiesBrowser "Type:CookiesBrowser")** | allows usage of browser cookies to get around the issue of "login to confirm you're not a bot" issue. |
