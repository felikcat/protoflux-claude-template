# Component:RibbonFollowerColorRangeInitializer

> Source: https://wiki.resonite.com/Component:RibbonFollowerColorRangeInitializer

Collapse **Component image**

[File:RibbonFollowerColorRangeInitializerComponent.png](https://wiki.resonite.com/index.php?title=Special:Upload&wpDestFile=RibbonFollowerColorRangeInitializerComponent.png "File:RibbonFollowerColorRangeInitializerComponent.png") **Ribbon Follower Color Range Initializer** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **RibbonFollowerColorRangeInitializer** component gives ribbon followers in a particle system a random color in a range upon start.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `MinValue` | **[ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")** | The minimum color a ribbon follower can have. |
| `MaxValue` | **[ColorX](https://wiki.resonite.com/Type:ColorX "Type:ColorX")** | The maximum color a ribbon follower can have. |

## Usage

Attach to a slot, add to the list of modules in a [Component:ParticleSystem](https://wiki.resonite.com/Component:ParticleSystem "Component:ParticleSystem"), and adjust the values to make the desired effect from this component.
