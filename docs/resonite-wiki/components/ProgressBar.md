[![](https://wiki.resonite.com/images/thumb/4/41/ProgressBarComponent.png/510px-ProgressBarComponent.png)](https://wiki.resonite.com/File:ProgressBarComponent.png) **ProgressBar** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **ProgressBar** component is used to show the [user](https://wiki.resonite.com/User "User") the normalized input of a float value on the [UIX](https://wiki.resonite.com/UIX "UIX") element.

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `Progress` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | The normalized progression the of value |
| `AnchorMinOffset` | **[Float2](https://wiki.resonite.com/Type:Float2 "Type:Float2")** | [UIX](https://wiki.resonite.com/UIX "UIX") Anchor minimum offset |
| `AnchorMaxOffset` | **[Float2](https://wiki.resonite.com/Type:Float2 "Type:Float2")** | [UIX](https://wiki.resonite.com/UIX "UIX") Anchor maximum offset |
| `AnchorMin` | _[field drive](https://wiki.resonite.com/Type:FieldDrive%601 "Type:FieldDrive`1")_ of **[Float2](https://wiki.resonite.com/Type:Float2 "Type:Float2")** | [UIX](https://wiki.resonite.com/UIX "UIX") Anchor minimum offset reference input |
| `AnchorMax` | _[field drive](https://wiki.resonite.com/Type:FieldDrive%601 "Type:FieldDrive`1")_ of **[Float2](https://wiki.resonite.com/Type:Float2 "Type:Float2")** | [UIX](https://wiki.resonite.com/UIX "UIX") Anchor maximum offset reference input |
| `Power` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | Modifies the Power of the progress bar |
