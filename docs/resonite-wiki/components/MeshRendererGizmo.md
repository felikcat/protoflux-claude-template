# Component:MeshRendererGizmo

> Source: https://wiki.resonite.com/Component:MeshRendererGizmo

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/5/59/MeshRendererGizmoComponent.png/510px-MeshRendererGizmoComponent.png)](https://wiki.resonite.com/File:MeshRendererGizmoComponent.png) **Mesh Renderer Gizmo** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **MeshRenderer** Gizmo is used to change the Bounding properties to a mesh object using the [dev tool](https://wiki.resonite.com/Dev_tool "Dev tool").
