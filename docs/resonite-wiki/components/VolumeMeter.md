# Component:VolumeMeter

> Source: https://wiki.resonite.com/Component:VolumeMeter

Collapse **Component image**

[File:VolumeMeterComponent.png](https://wiki.resonite.com/index.php?title=Special:Upload&wpDestFile=VolumeMeterComponent.png "File:VolumeMeterComponent.png") **Volume Meter** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **VolumeMeter** component can be used to get the volume of an inputted audio source.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `Smoothing` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | Smooth lerp the `Volume` field this much. |
| `Power` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | Raise `Volume` to this exponent. 1 will leave `Volume` unchanged. |
| `Method` | **[VolumeMeterMethod](https://wiki.resonite.com/Type:VolumeMeterMethod "Type:VolumeMeterMethod")** | Whether the outputted value should be the current volume, or the change in volume over time (delta) |
| `Source` | **[IAudioSource](https://wiki.resonite.com/Type:IAudioSource "Type:IAudioSource")** | The audio source to read the volume of (as long as it's streaming or playing). |
| `Volume` | _[raw output](https://wiki.resonite.com/Type:RawOutput%601 "Type:RawOutput`1")_ of **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | The current volume of `Source` raised to the exponent `Power`, after smoothing factor of `Smoothing`. |

Fields

## Usage

Attach to a slot and provide any [IAudioSource](https://wiki.resonite.com/Type:IAudioSource "Type:IAudioSource") to `Source`. When audio data is being generated by `Source` the `Volume` field will start changing values.

## Examples

This component can be used to add visual reaction to avatars when talking, or in audio/music visualizers.
