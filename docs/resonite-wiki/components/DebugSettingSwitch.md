# Component:DebugSettingSwitch

> Source: https://wiki.resonite.com/Component:DebugSettingSwitch

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/3/3f/DebugSettingSwitchComponent.png/510px-DebugSettingSwitchComponent.png)](https://wiki.resonite.com/File:DebugSettingSwitchComponent.png) **Debug Setting Switch** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **DebugSettingSwitch** component is used to Switch profiles as a way of debugging. Same thing switching between VR and desktop does. Only works in user space.

![](https://wiki.resonite.com/images/f/fe/WarningIcon.svg)

This game element is a debug item. Elements like this may be removed at any time without warning, and creations should not rely on it.
