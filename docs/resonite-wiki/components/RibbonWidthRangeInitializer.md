# Component:RibbonWidthRangeInitializer

> Source: https://wiki.resonite.com/Component:RibbonWidthRangeInitializer

Collapse **Component image**

[File:RibbonWidthRangeInitializerComponent.png](https://wiki.resonite.com/index.php?title=Special:Upload&wpDestFile=RibbonWidthRangeInitializerComponent.png "File:RibbonWidthRangeInitializerComponent.png") **Ribbon Width Range Initializer** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **RibbonWidthRangeInitializer** component gives ribbons in a particle system a random width in a range.

This component is part of the [Photon Dust](https://wiki.resonite.com/Photon_Dust "Photon Dust") system made by [Frooxius](https://wiki.resonite.com/User:Frooxius "User:Frooxius").

## Usage

Attach to a slot, add to the list of modules in a [Component:ParticleSystem](https://wiki.resonite.com/Component:ParticleSystem "Component:ParticleSystem"), and adjust the values to make the desired effect from this component.
