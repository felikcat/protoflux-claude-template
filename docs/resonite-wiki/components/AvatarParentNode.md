[File:AvatarParentNodeComponent.png](https://wiki.resonite.com/index.php?title=Special:Upload&wpDestFile=AvatarParentNodeComponent.png "File:AvatarParentNodeComponent.png") **Avatar Parent Node** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

This component is used to parent and pose the avatar in a certain way. When using this component now, it will attach then remove itself from the slot.

![](https://wiki.resonite.com/images/d/dd/InformationIcon.svg)

This component is no longer used and in the code reads the following: "AvatarParentNode is obsolete, please use [AvatarPoseNode](https://wiki.resonite.com/Component:AvatarPoseNode "Component:AvatarPoseNode") instead".
