# Component:InteractiveCameraSpawner

> Source: https://wiki.resonite.com/Component:InteractiveCameraSpawner

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/1/1c/InteractiveCameraSpawnerComponent.png/510px-InteractiveCameraSpawnerComponent.png)](https://wiki.resonite.com/File:InteractiveCameraSpawnerComponent.png) **Interactive Camera Spawner** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **InteractiveCameraSpawner** can receive [Button Events](https://wiki.resonite.com/Button_Events "Button Events"), including ones from a button on the same slot. When receiving a pressed event, this component will spawn an [InteractiveCamera](https://wiki.resonite.com/Component:InteractiveCamera "Component:InteractiveCamera") in the currently focused world.
