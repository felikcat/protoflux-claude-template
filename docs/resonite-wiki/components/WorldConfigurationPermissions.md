# Component:WorldConfigurationPermissions

> Source: https://wiki.resonite.com/Component:WorldConfigurationPermissions

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/3/3f/WorldConfigurationPermissionsComponent.png/510px-WorldConfigurationPermissionsComponent.png)](https://wiki.resonite.com/File:WorldConfigurationPermissionsComponent.png) **World Configuration Permissions** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **WorldConfigurationPermissions** component is used to handle the [Permissions](https://wiki.resonite.com/index.php?title=Permissions&action=edit&redlink=1 "Permissions (page does not exist)") of roles to change world configuration values like max user count and world name.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `AllowChanges` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether changes are allowed for the selected roles. |

Fields
Collapse

## Usage

click on the desired roles to affect with the `AllowChanges` value in the bottom of the component.
