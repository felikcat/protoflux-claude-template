# Component:FirstPersonTargettingController

> Source: https://wiki.resonite.com/Component:FirstPersonTargettingController

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/b/b4/FirstPersonTargettingControllerComponent.png/510px-FirstPersonTargettingControllerComponent.png)](https://wiki.resonite.com/File:FirstPersonTargettingControllerComponent.png) **First Person Targetting Controller** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **FirstPersonTargettingController** component is used to control the head movement of a desktop user when they look around.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `MinVerticalAngle` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | The furthest down angle from 0 being forward the user can look in desktop. |
| `MaxVerticalAngle` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | The furthest up angle from 0 being forward the user can look in desktop. |

Fields
Collapse

Not used directly by the user. Appears on a user root slot.
