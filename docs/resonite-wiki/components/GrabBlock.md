# Component:GrabBlock

> Source: https://wiki.resonite.com/Component:GrabBlock

This page contains [changes](https://wiki.resonite.com/index.php?title=Component:GrabBlock&diff=91417) which are not marked for translation.

Used to prevent grabbing a [Slot](https://wiki.resonite.com/Slot "Slot") from a certain [Collider](https://wiki.resonite.com/Collider "Collider") or set of colliders. Place between a collider and a [Grabbable](https://wiki.resonite.com/Component:Grabbable "Component:Grabbable") component and the slot will no longer be able to be grabbed via that collider.
