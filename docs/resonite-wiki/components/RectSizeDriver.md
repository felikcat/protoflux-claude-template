[![](https://wiki.resonite.com/images/thumb/b/bc/RectSizeDriverComponent.png/510px-RectSizeDriverComponent.png)](https://wiki.resonite.com/File:RectSizeDriverComponent.png) **RectSizeDriver** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **RectSizeDriver** component is used to change the size of a [RectTransform](https://wiki.resonite.com/Component:RectTransform "Component:RectTransform") (or any other [float2](https://wiki.resonite.com/Type:Float2 "Type:Float2") value).

![](https://wiki.resonite.com/images/f/fe/WarningIcon.svg)

Using this component may have a chance of breaking your [UIX](https://wiki.resonite.com/UIX "UIX"), and will need to be recalculated when the component gets removed.


| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `TargetSize` | _[field drive](https://wiki.resonite.com/Type:FieldDrive%601 "Type:FieldDrive`1")_ of **[Float2](https://wiki.resonite.com/Type:Float2 "Type:Float2")** | The target to scale using this component. |
| `Scale` | **[Float2](https://wiki.resonite.com/Type:Float2 "Type:Float2")** | The scale to adjust. |
