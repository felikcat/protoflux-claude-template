# Component:ObjectOrb

> Source: https://wiki.resonite.com/Component:ObjectOrb

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/e/e8/ObjectOrbComponent.png/510px-ObjectOrbComponent.png)](https://wiki.resonite.com/File:ObjectOrbComponent.png) **Object Orb** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

According to [Frooxius](https://wiki.resonite.com/User:Frooxius "User:Frooxius") this was originally made during the infancy stages of [FrooxEngine](https://wiki.resonite.com/FrooxEngine "FrooxEngine") to serve as an item the user would stick into an "Object Placement Tool" where users could easily place objects into the world in the form of duplicates and make it easier to create worlds. However this never came to fruition and the concept lives on as this remnant in the game code and Component set. [\[1\]](https://wiki.resonite.com/Component:ObjectOrb#cite_note-1)
