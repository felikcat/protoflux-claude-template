# Component:ProtoFluxNodeDebugInfo

> Source: https://wiki.resonite.com/Component:ProtoFluxNodeDebugInfo

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/f/f4/ProtoFluxNodeDebugInfoComponent.png/510px-ProtoFluxNodeDebugInfoComponent.png)](https://wiki.resonite.com/File:ProtoFluxNodeDebugInfoComponent.png) **Proto Flux Node Debug Info** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **ProtoFluxNodeDebugInfo** component is used to gather Debug Info on a particular [ProtoFlux](https://wiki.resonite.com/ProtoFlux "ProtoFlux") node and it's group.

![](https://wiki.resonite.com/images/f/fe/WarningIcon.svg)

This game element is a debug item. Elements like this may be removed at any time without warning, and creations should not rely on it.
