# Component:SupporterCreditList

> Source: https://wiki.resonite.com/Component:SupporterCreditList

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/8/81/SupporterCreditListComponent.png/510px-SupporterCreditListComponent.png)](https://wiki.resonite.com/File:SupporterCreditListComponent.png) **Supporter Credit List** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **SupporterCreditList** component is used to get a list of supporters currently supporting the game financially.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `CreditType` | **[CreditType](https://wiki.resonite.com/index.php?title=Type:CreditType&action=edit&redlink=1 "Type:CreditType (page does not exist)")** | What kind of supporter level/category to get names from. |
| `Separator` | **[String](https://wiki.resonite.com/Type:String "Type:String")** | How to separate the list of names. |
| `Prefix` | **[String](https://wiki.resonite.com/Type:String "Type:String")** | What to put at the beginning of each name (Usually a color tag) |
| `Suffix` | **[String](https://wiki.resonite.com/Type:String "Type:String")** | What to put at the end of each name (Usually a color tag ender. |
| `FormattedList` | **[String](https://wiki.resonite.com/Type:String "Type:String")** | The resulting list. |
