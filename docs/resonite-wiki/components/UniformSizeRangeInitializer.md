# Component:UniformSizeRangeInitializer

> Source: https://wiki.resonite.com/Component:UniformSizeRangeInitializer

Collapse **Component image**

[File:UniformSizeRangeInitializerComponent.png](https://wiki.resonite.com/index.php?title=Special:Upload&wpDestFile=UniformSizeRangeInitializerComponent.png "File:UniformSizeRangeInitializerComponent.png") **Uniform Size Range Initializer** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **UniformSizeRangeInitializer** component makes particles in a particle system have a random size in a range.

This component is part of the [Photon Dust](https://wiki.resonite.com/Photon_Dust "Photon Dust") system made by [Frooxius](https://wiki.resonite.com/User:Frooxius "User:Frooxius").

## Usage

Attach to a slot, add to the list of modules in a [Component:ParticleSystem](https://wiki.resonite.com/Component:ParticleSystem "Component:ParticleSystem"), and adjust the values to make the desired effect from this component.
