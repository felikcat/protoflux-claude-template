# Component:ColorConstantInitializer

> Source: https://wiki.resonite.com/Component:ColorConstantInitializer

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/3/34/ColorConstantInitializerComponent.png/510px-ColorConstantInitializerComponent.png)](https://wiki.resonite.com/File:ColorConstantInitializerComponent.png) **Color Constant Initializer** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **ColorConstantInitializer** component makes all particles start with the same color when they are born.

This component is part of the [Photon Dust](https://wiki.resonite.com/Photon_Dust "Photon Dust") system made by [Frooxius](https://wiki.resonite.com/User:Frooxius "User:Frooxius").

## Usage

Attach to a slot, add to the list of modules in a [Component:ParticleSystem](https://wiki.resonite.com/Component:ParticleSystem "Component:ParticleSystem"), and adjust the values to make the desired effect from this component.
