# Component:LegacyColorAsLinearAdapter

> Source: https://wiki.resonite.com/Component:LegacyColorAsLinearAdapter

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/a/a8/LegacyColorAsLinearAdapterComponent.png/510px-LegacyColorAsLinearAdapterComponent.png)](https://wiki.resonite.com/File:LegacyColorAsLinearAdapterComponent.png) **Legacy Color As Linear Adapter** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **LegacyColorAsLinearAdapter** component is used to convert old colors from migrated content into the linear color space for use in modern content.
