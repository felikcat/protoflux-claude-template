# Component:AngularVelocity3DRangeInitializer

> Source: https://wiki.resonite.com/Component:AngularVelocity3DRangeInitializer

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/9/94/AngularVelocity3DRangeInitializerComponent.png/510px-AngularVelocity3DRangeInitializerComponent.png)](https://wiki.resonite.com/File:AngularVelocity3DRangeInitializerComponent.png) **Angular Velocity 3D Range Initializer** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **AngularVelocity3DRangeInitializer** component makes particles have a random starting rotation angular speed along the X, Y, and Z axies.

This component is part of the [Photon Dust](https://wiki.resonite.com/Photon_Dust "Photon Dust") system made by [Frooxius](https://wiki.resonite.com/User:Frooxius "User:Frooxius").

## Usage

Attach to a slot, add to the list of modules in a [Component:ParticleSystem](https://wiki.resonite.com/Component:ParticleSystem "Component:ParticleSystem"), and adjust the values to make the desired effect from this component.
