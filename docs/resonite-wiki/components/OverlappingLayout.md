# Component:OverlappingLayout

> Source: https://wiki.resonite.com/Component:OverlappingLayout

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/a/a4/OverlappingLayoutComponent.png/510px-OverlappingLayoutComponent.png)](https://wiki.resonite.com/File:OverlappingLayoutComponent.png) **Overlapping Layout** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **OverlappingLayout** is best explained on the [UIX](https://wiki.resonite.com/UIX "UIX") page. The description for such is as follows:

A \[Overlapping layout\] layout overlaps or places on top of another layout in the same space.
