# Component:DebugSettings

> Source: https://wiki.resonite.com/Component:DebugSettings

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/7/7d/DebugSettingsComponent.png/510px-DebugSettingsComponent.png)](https://wiki.resonite.com/File:DebugSettingsComponent.png) **DebugSettings** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **DebugSettings** component is a component updated during Resonite updates to contain bleeding edge [settings](https://wiki.resonite.com/Settings "Settings") for the game.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `DebugInputBindings` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether to display the user's controller inputs on their hands in [userspace](https://wiki.resonite.com/Userspace "Userspace"). |
