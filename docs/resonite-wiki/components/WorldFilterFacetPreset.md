# Component:WorldFilterFacetPreset

> Source: https://wiki.resonite.com/Component:WorldFilterFacetPreset

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/5/56/WorldFilterFacetPresetComponent.png/510px-WorldFilterFacetPresetComponent.png)](https://wiki.resonite.com/File:WorldFilterFacetPresetComponent.png) **World Filter Facet Preset** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **WorldFilterFacetPreset** component loads the World Filter facet when it's not `_fullyLoaded`. It does this by loading the facet from the cloud, and using that loaded data to make the preset

used in the main dash. no need to be used by the user unless they are missing the World Filter facet.
