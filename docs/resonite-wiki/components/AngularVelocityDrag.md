# Component:AngularVelocityDrag

[![](https://wiki.resonite.com/images/thumb/c/c3/AngularVelocityDragComponent.png/510px-AngularVelocityDragComponent.png)](https://wiki.resonite.com/File:AngularVelocityDragComponent.png) **Angular Velocity Drag** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **Angular Velocity Drag** component reduces the angular momentum of particles in a particle system over time.
