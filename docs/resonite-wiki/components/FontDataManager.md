# Component:FontDataManager

> Source: https://wiki.resonite.com/Component:FontDataManager

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/9/9c/FontDataManagerComponent.png/510px-FontDataManagerComponent.png)](https://wiki.resonite.com/File:FontDataManagerComponent.png) **Font Data Manager** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **FontDataManager** component handles the mapping of Font data and parameters to font based materials during rendering.

![](https://wiki.resonite.com/images/f/fe/WarningIcon.svg)

If an existing version of this component is found in a world. Do not delete it. This can cause unforseen consequences.
