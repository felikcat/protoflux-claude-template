# Component:FreeformTargettingController

> Source: https://wiki.resonite.com/Component:FreeformTargettingController

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/b/bc/FreeformTargettingControllerComponent.png/510px-FreeformTargettingControllerComponent.png)](https://wiki.resonite.com/File:FreeformTargettingControllerComponent.png) **Freeform Targetting Controller** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **FreeformTargettingController** component is used to handle the 3rd person targeting onto [UIX](https://wiki.resonite.com/UIX "UIX") in desktop mode.
