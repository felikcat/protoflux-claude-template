# Component:FullBodyCalibratorSpawner

> Source: https://wiki.resonite.com/Component:FullBodyCalibratorSpawner

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/3/3e/FullBodyCalibratorSpawnerComponent.png/510px-FullBodyCalibratorSpawnerComponent.png)](https://wiki.resonite.com/File:FullBodyCalibratorSpawnerComponent.png) **Full Body Calibrator Spawner** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **FullBodyCalibratorSpawner** when on the same Slot as a [button](https://wiki.resonite.com/Type:IButton "Type:IButton"), spawns a [full body calibrator UI](https://wiki.resonite.com/Component:FullBodyCalibrator "Component:FullBodyCalibrator") in front of the user that pressed the button.
