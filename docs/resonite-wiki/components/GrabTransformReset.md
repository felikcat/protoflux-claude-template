# Component:GrabTransformReset

> Source: https://wiki.resonite.com/Component:GrabTransformReset

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/c/c8/GrabTransformResetComponent.png/510px-GrabTransformResetComponent.png)](https://wiki.resonite.com/File:GrabTransformResetComponent.png) **Grab Transform Reset** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **GrabTransformReset** component resets transforms of a grabbable on the same slot upon grab and/or release.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `ResetPositionOnGrab` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether to set the grabbable position to 0 on grab. |
| `ResetRotationOnGrab` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether to set the grabbable rotation to 0 on grab. |
| `ResetScaleOnGrab` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether to set the grabbable scale to 1 on grab. |
| `ResetPositionOnRelease` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether to set the grabbable position to 0 on release. |
| `ResetRotationOnRelease` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether to set the grabbable rotation to 0 on release. |
