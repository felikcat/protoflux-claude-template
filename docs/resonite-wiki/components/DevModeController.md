# Component:DevModeController

> Source: https://wiki.resonite.com/Component:DevModeController

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/e/ee/DevModeControllerComponent.png/510px-DevModeControllerComponent.png)](https://wiki.resonite.com/File:DevModeControllerComponent.png) **Dev Mode Controller** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **DevModeController** component only works in [userspace](https://wiki.resonite.com/Userspace "Userspace") and is used to control the user's current [edit mode](https://wiki.resonite.com/Edit_mode "Edit mode") state.

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `DevModeEnabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether [Edit Mode](https://wiki.resonite.com/Edit_Mode "Edit Mode") is enabled. |

Used in settings. Not usually used by the user.
