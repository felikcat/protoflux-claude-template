## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `MotionBlurIntensity` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | Controls the strength of motion blur, disabled entirely if set to 0%. |
| `BloomIntensity` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | The Bloom effect causes a glow to appear around bright objects in the scene, disabled entirely if set to 0%. |
| `AmbientOcclusionIntensity` | **[Float](https://wiki.resonite.com/Type:Float "Type:Float")** | Adds shading in corners and area that'd receive less light. Controls the strength of [Ambient Occlusion](https://en.wikipedia.org/wiki/Ambient_occlusion), disabled entirely if set to 0%. |
| `ScreenSpaceReflections` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | When enabled, any parts of the scene currently visible to the camera will reflect off shiny surfaces in realtime. |
| `Antialiasing` | **[AntiAliasingMethod](https://wiki.resonite.com/Type:AntiAliasingMethod "Type:AntiAliasingMethod")** | Smooths out jagged edges on objects. Some may not work or be suitable for VR. |
