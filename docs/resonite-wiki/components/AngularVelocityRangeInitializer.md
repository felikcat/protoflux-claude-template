# Component:AngularVelocityRangeInitializer

> Source: https://wiki.resonite.com/Component:AngularVelocityRangeInitializer

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/f/f1/AngularVelocityRangeInitializerComponent.png/510px-AngularVelocityRangeInitializerComponent.png)](https://wiki.resonite.com/File:AngularVelocityRangeInitializerComponent.png) **Angular Velocity Range Initializer** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **AngularVelocityRangeInitializer** component makes particles have a random rotation speed within a range along their orientation axis when born.

This component is part of the [Photon Dust](https://wiki.resonite.com/Photon_Dust "Photon Dust") system made by [Frooxius](https://wiki.resonite.com/User:Frooxius "User:Frooxius").

## Usage

Attach to a slot, add to the list of modules in a [Component:ParticleSystem](https://wiki.resonite.com/Component:ParticleSystem "Component:ParticleSystem"), and adjust the values to make the desired effect from this component.
