# Component:UndoListRoot

> Source: https://wiki.resonite.com/Component:UndoListRoot

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/b/b4/UndoListRootComponent.png/510px-UndoListRootComponent.png)](https://wiki.resonite.com/File:UndoListRootComponent.png) **Undo List Root** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **UndoListRoot** component acts as a marker for a particular user's undo actions list. For more information on undos, see [Undo](https://wiki.resonite.com/Undo "Undo").

## Fields

| Name | Type | Description |

| `OwnerUser` | _direct_ **[UserRef](https://wiki.resonite.com/Type:UserRef "Type:UserRef")** | the owner of this undo list. |
