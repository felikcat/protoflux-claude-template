# Component:SettingsDialog

> Source: https://wiki.resonite.com/Component:SettingsDialog

Collapse **Component image**

[File:SettingsDialogComponent.png](https://wiki.resonite.com/index.php?title=Special:Upload&wpDestFile=SettingsDialogComponent.png "File:SettingsDialogComponent.png") **Settings Dialog** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **SettingsDialog** component is shown during account setup to allow the user to specify their height during the intro.

| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |

| `_useImperial` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether the inputted height should be imperial. |
