# Component:SimpleVirtualKeyboard

[![](https://wiki.resonite.com/images/thumb/8/81/SimpleVirtualKeyboardComponent.png/510px-SimpleVirtualKeyboardComponent.png)](https://wiki.resonite.com/File:SimpleVirtualKeyboardComponent.png) **Simple Virtual Keyboard** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **SimpleVirtualKeyboard** component makes a simple keyboard UI that can be used.

## Usage

On apply this component populates a [VirtualKeyboard](https://wiki.resonite.com/Component:VirtualKeyboard "Component:VirtualKeyboard") as well as a basic UIX layout.
