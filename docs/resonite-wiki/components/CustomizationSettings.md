# Component:CustomizationSettings

> Source: https://wiki.resonite.com/Component:CustomizationSettings

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/c/c7/CustomizationSettingsComponent.png/510px-CustomizationSettingsComponent.png)](https://wiki.resonite.com/File:CustomizationSettingsComponent.png) **Customization Settings** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **CustomizationSettings** component adds an option to the [Settings](https://wiki.resonite.com/Settings "Settings") menu to enable and disable [UI Edit Mode](https://wiki.resonite.com/UI_Edit_Mode "UI Edit Mode").

## Fields

| Name | Type | Description |
| --- | --- | --- |
| `persistent` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Determines whether or not this item will be saved to the server. |
| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `UserInterfaceEditMode` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Whether [UI Edit Mode](https://wiki.resonite.com/UI_Edit_Mode "UI Edit Mode") is enabled or not. |

Fields

## Usage

Not used by the user directly.

Used in the settings menu.
