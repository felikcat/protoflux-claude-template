# Component:RotationGizmo

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/2/2b/RotationGizmoComponent.png/510px-RotationGizmoComponent.png)](https://wiki.resonite.com/File:RotationGizmoComponent.png) **Rotation Gizmo** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **RotationGizmo** component is used by dev tools to allow rotation of slots.

See [Dev tool](https://wiki.resonite.com/Dev_tool "Dev tool").
