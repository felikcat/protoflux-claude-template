# Component:DebugHands

> Source: https://wiki.resonite.com/Component:DebugHands

Collapse **Component image**

[![](https://wiki.resonite.com/images/thumb/2/28/DebugHandsComponent.png/510px-DebugHandsComponent.png)](https://wiki.resonite.com/File:DebugHandsComponent.png) **Debug Hands** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

The **DebugHands** component shows the vector axes and finger axes for a particular hand on a particular user.

![](https://wiki.resonite.com/images/f/fe/WarningIcon.svg)

This game element is a debug item. Elements like this may be removed at any time without warning, and creations should not rely on it.
