[![](https://wiki.resonite.com/images/thumb/e/e6/BatchActionComponent.png/510px-BatchActionComponent.png)](https://wiki.resonite.com/File:BatchActionComponent.png) **Batch Action** component as seen in the [Scene Inspector](https://wiki.resonite.com/Scene_Inspector "Scene Inspector")

Batch action is a component that is used internally by the [Undo](https://wiki.resonite.com/Undo "Undo") Manager to handle keeping data that the user has done so that same user can Undo it later via the control menu(s).

## Fields

| Name | Type | Description |

| `UpdateOrder` | **[Int](https://wiki.resonite.com/Type:Int "Type:Int")** | Controls the order in which this component is updated. |
| `Enabled` | **[Bool](https://wiki.resonite.com/Type:Bool "Type:Bool")** | Controls whether or not this component is enabled. Some components stop their functionality when this field is disabled, but some don't. |
| `_description` | **[String](https://wiki.resonite.com/Type:String "Type:String")** | a short description of what this group of items is undoing. Can be filled automatically with custom content when generated by a begin and end Undo Batch protoflux node |

Fields

## Usage

Used for [User](https://wiki.resonite.com/User "User") group Undo Steps

## Examples

This component can be generated in the Undo Manager when doing certain actions, or when a [End Undo Batch ProtoFlux node](https://wiki.resonite.com/ProtoFlux:End_Undo_Batch "ProtoFlux:End Undo Batch") is used successfully for a user.
