# Tools

In Resonite, many actions require using some form of tool. A tool can be any object you use to achieve some goal.

## Common Tools

Most official tools are found in: **Resonite Essentials > Tools**

| Tool | Description |
|------|-------------|
| **Dev Tool** | Inspect objects and interact with gizmos |
| **ProtoFlux Tool** | Create and edit ProtoFlux scripting |
| **Material Tool** | Extract, edit, and apply materials to meshes |
| **Glue Tool** | Glue items together or bake multiple meshes |
| **Grabbable Setter Tool** | Set and manipulate grabbable properties |
| **Character Collider Setter Tool** | Set and manipulate collider properties |
| **Color Tool** | Change color of objects |
| **Component Clone Tool** | Store components to copy onto new slots |
| **Camera Exclusion Tool** | Hide specific objects from cameras |
| **Mesh Visibility Tool** | Change visibility of meshes |
| **Slicer Tool** | Create slicers to cut objects |
| **Splat Editor Tool** | Edit gaussian splats |
| **Mesh Transfer Tool** | Transfer meshes between armatures |
| **Mesh Tool** | Copy and edit meshes |
| **Light Tool** | Place light sources into the world |
| **Microphone** | Record voice to create audio clips |
| **Shape Tool** | Create primitive shapes (cubes, spheres) |
| **Brush Tool** | Draw strokes in 3D space |
| **Measurement Tool** | Measure distances |
| **Labeler Tool** | Apply labels to objects |
| **Controller Debug Tool** | Show diagnostic information for controllers |
| **Dynamic Bone Poke Tool** | Push around dynamic bones |
| **Module Snapping Tool** | Snap together modular components |

## Equipping

To equip a tool:
1. Grab it
2. Open context menu → Select fist icon "Equip"
   - **OR** grab with laser, hold joystick back until tool reaches hand

Tool remains mounted even when releasing grab.

To dequip: Context menu → hand releasing object icon "Dequip"

## Effects

While equipped:
- Context menu changes to tool-specific actions
- Primary and secondary buttons gain new functions based on tool

## Tool Shelf

Located on both wrists of most avatars - a shiny rectangular plate for quick tool access.

### Placing Tools on Tool Shelf

**Method 1:** Grab unequipped tool, position against shelf, release

**Method 2:** Point laser at shelf while tool equipped (beam turns bluish), click primary

### Accessing Tools from Shelf

**Method 1:** Grab and equip

**Method 2:** Point laser at tool (beam turns green), click primary (auto-equips)

**Tool Swap:** With tool equipped, point at another tool on shelf to swap instantly

**Warning:** Tools on your tool shelf can be taken by anyone with grab permissions.

## MultiTools

Special tools combining features of multiple tools. An official MultiTool with all common tools is in:
**Resonite Essentials > Tools > Misc**

Community-created MultiTools also exist but may be error-prone.
